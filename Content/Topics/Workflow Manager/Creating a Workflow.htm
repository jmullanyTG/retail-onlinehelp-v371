Creating a Workflow

  1. From the Maintain tab, click Workflow Manager.

  2. From the Workflow Manager screen, click New in the Workflow toolbar.

  3. Enter the following details in the Workflow dialog box and click OK:

    Field

    Description
    Name The workflow name appears in the Workflows list.
    Solution

    The JustEnough solution the workflow will govern.
    Active Whether or not to the workflow is "live" in the system. Consider making workflows inactive while creating or editing them to avoid unintended behavior.
    Controlled User Rights

    The set of solution-specific user rights to be controlled by this workflow. The user rights selected in this property appear in the status permission blacklist of each status in the workflow.

    All user rights related to the selected solution are selected by default. You can customize this set by moving rights from the Workflow Controlled list to the Available list in the Workflow User Rights dialog box.

  4. The new workflow appears in the Workflows screen. Its name is selected in the Workflows list under the selected solution, and the Design pane shows the default workflow diagram for the solution selected.

Next Steps

Once a workflow is created, you can start making the following changes:

  • Edit the status information of the default status boxes.
  • Edit the transition properties of the default transitions.
  • Add new status boxes to the workflow.
  • Add new transitions to the workflow.
