Editing a Status Permission Blacklist

  1. In the Workflow Manager screen, select the workflow containing the permissions you want to edit.
  2. In the Design pane, select the status whose permission blacklist you want to edit.

  3. In the Status Permission Blacklist, select the permissions you want to edit and click Edit.

    In this example, we've selected Edit Price Action and Delete Price Action user rights.

  4. In the Remove User Rights dialog box:

    1. Select the user right/status state combinations you want to customize permissions for.

      In this example, we only want to customize edit and delete permissions for "validated" price actions.

    2. Choose the roles/users you want to remove rights for in those status/state circumstances:

      In this example, only users with Administrator or JustEnough Advanced User roles will be able to edit or delete price actions that are in the Ready for Approval status and have been validated.

    3. Click Save. The Status Permission Blacklist is updated to reflect the changes you made.

      In this example, the blue user icons in the Validated column indicate that Edit Price Action and Delete Price Action rights are removed for one or more users when a price action is in the Validated state during this status.

Note: Alternatively, you can click on a user icon to view/edit permissions for that specific user right/status state combination.
