Workflow Manager Screen

The Workflow Manager screen contains three panes:

  • Workflows list (left side): Select a workflow here to view it in the Design pane.
    • A blue workflow icon indicates the workflow is currently active in the system.

    • A black workflow icon indicates the workflow is currently deactivated/not used in the system.

  • Design pane (middle): Displays the diagram of the selected workflow. At the bottom of the pane are the following zoom options:
    • Fit to Page: Zooms out to display the entire workflow page in the pane.
    • Fit to Width: Zooms in/out to display the entire width of the workflow page in the pane.
    • Fit to Items: Zooms in/out to fit all of the workflow items in the pane.
    • Zoom: Lets you manually zoom in or out.

  • Information pane (right side): Displays information about the item selected in the Design pane.

Actions

You can perform the following tasks from the Workflow Manager screen.

Note: Remember to click Save on the Workflow toolbar after making changes to a workflow design, including adding, deleting, or editing statuses or transitions.

Action

Quick Instructions
Create a workflow Click New on the Workflow toolbar to add a new workflow.
Edit a workflow

Click Edit on the Workflow toolbar to the make the following changes to a selected workflow:

  • Change the workflow name.
  • Activate or deactivate the workflow.
  • Change the user rights controlled by the workflow.
Activate/Deactivate a workflow Click Edit on the Workflow toolbar and then click Active in the Workflow dialog box to toggle activating/deactivating the workflow.
Save changes to a workflow Click Save on the Workflow toolbar to save any changes made to the workflow's design, status information, or transition information.
Delete a workflow To delete a workflow, select it in the Workflows list and click Delete on the Workflow toolbar.
Add a new status to the workflow Click New on the Status toolbar to add a new status box to the open workflow.
Edit a status Select a status in the Design grid to edit its properties in the Status Information pane.
Edit permissions for a status Select a status in the Design grid to edit its permission blacklist.
Delete a status Select the status in the Design grid and click Delete in the Status toolbar to remove the status from the workflow. Click Save on the Workflow toolbar to save the change to the workflow.
Add a transition between statuses

Click Draw on the Transition toolbar to toggle "draw mode" in the Design pane. When in draw mode, click on the "from" status and drag the cursor to the "to" status. (For example, to create a transition from a "Planning" status to a "Review" status, click on the Planning status and then drag the cursor to the "Review" status.) The transition arrow appears as you drag the cursor from one connection point to another.

Once the transition is created, edit its properties in the Transition Information pane.
Edit a transition Select the transition in the Design grid and edit its properties in the Transition Information pane.
Delete a transition Select a transition in the Design grid and click Delete in the Transition toolbar.
