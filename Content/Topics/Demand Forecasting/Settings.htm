Settings

The settings section will display some of the information from the System Policy module which will affect the forecasts.

  • Backtrack/Simulation Ratio – The ratio for weighting the optimistic backtrack to the pessimistic simulation during the forecast engine run. It is set to 0.5 by default (50%) and is considered best practice to keep it there.
  • 0 = Only consider simulation
  • 0.5 = Equal split
  • 1 = Only consider backtrack
  • Prevent Negative Forecast –A system Policy Parameter Setting (Disallow Negative Forecast Data) which prevents, or allows Negative Forecast values to be calculated and displayed. It is set by default to “true” and is considered best practice to keep it there.
  • Error Method – The method by which the forecast engine grades the tournament winners.
