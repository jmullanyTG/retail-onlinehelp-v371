Shape Forecasting

Shape Forecasting applies a pre-selected curve to an item's moving-average sales history in order to generate a moving-average based, season-affected demand forecast for that item.

The Shape Forecasting method appears in the Forecast task dialog box and participates in the forecast tournament when selected.

Shape forecasting requires the following setup tasks:

  • Define appropriate curves in the Curves module.
  • Identify the desired curve for each item in the Shape Forecast Maintenance view.

Shape Forecast Maintenance View

The Shape Forecasting Maintenance view is where you identify the curve that JustEnough should use to generate an item's shape forecast.

  1. In the Views module, open the Shape Forecast Maintenance view.

  2. In the Filter list, choose the filter containing your desired item(s). The items appear in two grids. The top grid lists items at sites. The bottom grid lists items by channel. (The same items appear in both grids.)

  3. To define an item's curve, select the item in a grid and click in its Curve cell to display the "..." button. Click the "..." button to open the Curve Selection dialog box.

  4. In the Curve Selection dialog box, choose the curve you want to apply by selecting its row in the grid and clicking OK.

  5. Click Save above the grid to save the change.

  6. Repeat steps 2-5 for all desired items.

    The next time that JustEnough generates demand forecasts for these items, it will use these curves to create their shape forecasts.
