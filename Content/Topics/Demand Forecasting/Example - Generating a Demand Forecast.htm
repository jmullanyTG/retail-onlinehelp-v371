Example - Demand Forecast

In this example, we are going to generate a demand forecast for the item, "High Collar Angelina Jacket".

Setup

The following objects are used in this example.

  • A filter that includes the "High Collar Angelina Jacket" item.
    • Filter used in this example:New Items (Creation <=14 Days).

Instructions

  1. In the JustEnough Manage tab, click Forecasts.

  2. In the Items - Level View dialog box, select the New Items (Creation <=14 Days) filter and click OK.

  3. In the Forecasts screen, select the new item, "High Collar Angelina Jacket_Grey - L":

    1. Click the Items list.
    2. In the Item Master Code filter, enter 708.

    3. Double-click the row with the Channel value, "Retail" to select it.

  4. Create a demand forecast for the new item:

    1. In the Forecast toolbar, click Forecast.
    2. In the Forecast Task dialog box, click OK to star the forecast tournament.

    3. Once the forecast is complete, it appears in the Forecast screen.
