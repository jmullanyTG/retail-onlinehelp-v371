Prerequisites

Before beginning any activities in which the cluster sets you create will be needed, the following steps must be completed:

  1. Definition of product and site hierarchies: Plan > Views > Maintenance > Hierarchy Filtering
  2. Optional: If you desire to use product and site attributes for the creation of site clusters, Attribute Sets, Families, and Values must be defined, and at a minimum, at least one set must be assigned to the topmost level of both the product and site hierarchy: Maintain > Attributes
  • Note: site and product attribute associations can be set via your configured data import processes, but they can also be managed manually in the following places:
  • Site Attributes: Manage > Site Manager > Site Attribute Management
  • Product Attributes: Manage > Variant Manager > Variant Attribute Maintenance
