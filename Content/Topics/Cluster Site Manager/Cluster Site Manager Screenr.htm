Cluster Site Manager Screen

In the Cluster Site Manager screen, users can launch the Cluster Set Management window, calculate and view specific cluster sets in both detail and chart modes, make manual adjustments to site cluster assignments, and arrange multiple cluster sets in the view for easy comparison.

  • Manage: launches the Cluster Set Management dialog
  • Save: saves changes made to the User Cluster and Apply User Value fields since the last Save
  • Undo: undoes changes made to the User Cluster and Apply User Value fields since the last Save

  • Calculate: calculates the cluster assignments for all stores included in the cluster set (based on the parameters and criteria specified during cluster set configuration) and populates the Cluster field with these assignments.

    Note: You can calculate multiple cluster sets at once from the Cluster Set Management window. From there, just click Allow Multiple and select the desired cluster sets and/or cluster set partitions.

  • Configuration: launches the cluster set Configuration dialog
  • Refresh: refreshes the current view
  • Cluster Set Viewing Area

  • Cluster Set Documents: cluster sets the user has chosen to view appear as tabs with the cluster set name just below the toolbar

  • Document Options: by right-clicking a document tab, you gain access to the following options:
  • Close: closes the cluster set document you right-clicked
  • Close All But This: closes all open cluster set documents except the one you right-clicked; only available when multiple documents are open
  • Float: converts the document tab to a window that can be sized and positioned on the screen (or other screens, in multiple monitor configurations)
  • Note: floating can also be achieve by simply clicking and dragging the document header without right-clicking and selecting Float
  • After floating, click the document header and drag the document to the desired location
  • Right click the document header and click Dock as Tabbed Document to return the document to its tabbed position

  • You can also hover over the directional buttons of the positioning reticle that appears while dragging to see possible docked locations for the floated window

  • New Vertical Tab Group: configures all documents currently being viewed vertically side by side; after clicking this option, right click the document header again and select Move to Previous Tab Group to return to the last screen configuration

  • New Horizontal Tab Group: configures all documents currently being viewed horizontally stacked; after clicking this option, right click the document header again and select Move to Previous Tab Group to return to the last screen configuration

  • Name: non-editable alpha-numeric; displays the name of the cluster set being viewed
  • Description: non-editable alpha-numeric; displays the description of the cluster set being viewed
  • Cluster Type: non-editable dropdown; displays the cluster set type (see Cluster Set Configuration later in this manual for more information on types)
  • Last Computed: non-editable date picker; displays the date the cluster set was last calculated

  • Detail Tab: displays the numbers and descriptions of all stores in the cluster set and their cluster assignments; also allows manual editing of cluster assignments

  • Item Name: non-editable numeric; displays the store/site number
  • Item Description: non-editable alpha; displays the store/site description
  • Cluster: non-editable alpha-numeric; displays the system calculated cluster assignment for the store
  • User Cluster: user editable dropdown; displays/sets the user override cluster assignment for the store; not required
  • Apply User Value: user editable checkbox; sets whether the system will use User Cluster in place of Cluster as the Effective Cluster or not; defaults to checked
  • Effective Cluster: non-editable alpha-numeric; displays the effective cluster assignment for the store

  • Chart Tab: displays all stores in the cluster set as data points against X and Y axes for all Attributes defined in Cluster Set Configuration

  • X-Axis Attribute: user editable dropdown; selects an Attribute from those defined in Cluster Set Configuration to plot on the X (horizontal) axis of the graph
  • Y-Axis Attribute: user editable dropdown; selects an Attribute from those defined in Cluster Set Configuration to plot on the Y (vertical) axis of the graph
  • Legend: user editable checkboxes; displays one color-coded checkbox per cluster, corresponding to the colored data points that are assigned to that cluster on the chart; unchecking hides the related data points

  • Data Points: colored dots and stars within the chart area; dots correspond to clusters of the same color in the legend, stars represent the centroids (K-Means types) or medoids (K-Medoids types) for the distance calculations used to assign stores to clusters; aligning the crosshairs on the chart to a specific data point (done simply by moving the cursor on the chart) shows a pop-up of the store associated with that point as well as the specific values associated with the X and Y axes at the left and bottom of the chart, respectively

  • Show Labels: user editable checkbox; toggles the display of data point labels for all points on the chart; defaults to unchecked
