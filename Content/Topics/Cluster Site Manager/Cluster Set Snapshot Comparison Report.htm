Cluster Set Snapshot Comparison Report

This report allows users to compare cluster set snapshots across a specified timeframe.

Object

Description

Time Frame

Select the timeframes that include the snapshots that you want to compare.

Parameters

Toggles displaying/hiding the Parameters area of the report.

Execute

Generates the report comparing the snapshots selected in the Time Frame list.

Cluster Set Snapshot Comparison Report

For each site, a snapshot classification is highlighted orange when it is different than the classification for the snapshot to its left.

Column

Description

Site Code

The unique code for the site in the system.

Site Name

The description for the site.

Classification Name

The assigned classification for the site during the corresponding snapshot's effective timeframe.
