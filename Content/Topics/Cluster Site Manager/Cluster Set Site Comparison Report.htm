Cluster Set Site Comparison Report

This report compares multiple cluster sets against each other to see how sites rank across them. It is most useful when comparing cluster sets that contain the same sites, classification structure, and clustering attributes. This is most useful when comparing cluster sets that contain the same sites, classification structure, and clustering attributes. For example, by comparing cluster sets created by product category with the classification A, B, and C that were calculated based on the same historical sales timeframe, you can identify strengths and weaknesses of each site across your product spectrum.

Object

Description
Cluster Set List Where you select the cluster sets to compare in the report.
Parameters Toggles displaying/hiding the Parameters area of the report.
Execute Generates the report comparing the cluster sets selected in the Cluster Set List.

Cluster Set Site Comparison Report

The conditional formatting in this report is determined by the Most Frequent field in the Cluster Set Site Classification Report. Any cluster set classification rankings that deviate from the Most Frequent classification(s) are highlighted orange.

The data displayed in the report is not editable.

Column

Description
Site Code The unique code for the site in the system.
Site Description The description for the site.
Classification Name The classification assigned to each site for the corresponding cluster set.

Cluster Set Site Classification Comparison Report

The conditional formatting in this report is determined by the Most Frequent field in the Cluster Set Site Classification Report. Any cluster set classification rankings that deviate from the Most Frequent classification(s) are highlighted orange.

The data displayed in the report is not editable.

Column

Description
Site Code The unique code for the site in the system.
Site Description The description for the site.
Classification/ Cluster Set Count The number of times the site was ranked in the corresponding classification across all cluster sets being compared.
Most Frequent

The most frequently occurring classification(s) for the site based on the data in the Classification/ Cluster Set Count columns.

If there is a tie for most frequent classification, all tying classifications are listed.
