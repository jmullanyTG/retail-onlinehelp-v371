Cluster Set Management Window

In the Cluster Set Management window, users can view a list of all the cluster set templates, and their related cluster sets, to which they have access. Here, they can select templates or sets to view, edit, delete, and create cluster set snapshots for, as well as initiate the creation of new templates and sets.

To open a cluster set document for a given cluster set, simply double-click on the cluster set in the grid or select it and click OK.

Toolbar

Cluster Set toolbar actions are described below.

Action Description
New Launches the Add Cluster Set dialog box to create a new cluster set or template.
Clone Launches the Update Cluster Set dialog box with all parameters and criteria from the selected cluster set or template pre-populated and "Cloned:" pre-pended to the beginning of the cluster set name.
Edit Launches the Update Cluster Set dialog for the selected cluster set or template.
Delete Deletes the selected cluster sets.

Notes:

  • Validation exists to prevent you from deleting cluster sets that are currently in use by assortment plans.
  • Cluster sets are not truly deleted until the garbage collection procedure runs on the database; the frequency and timing of this procedure are determined during software implementation.

Warning: Deleting cluster set templates also deletes all cluster sets that were created using the template.
Allow Multiple Toggles showing/hiding checkboxes in the grid. When enabled, you can select multiple cluster sets and/or cluster set partitions in the grid and then click Delete, Snapshot, or Calculate to perform the action on all of the selected cluster sets.

Notes:

  • Selecting a cluster set partition automatically selects all of its cluster sets.
  • Multi-cluster set operations may be submitted as tasks to the workload queue. You will receive a notification once the operations are done.
Snapshot Creates snapshots of selected cluster sets.
Calculate Calculates the cluster assignments for all stores included in a selected cluster set, based on the parameters and criteria specified during cluster set configuration.
Report Launches the Cluster Set Site Comparison Report.
View Collection Set Configuration Launches the View Collection Set Configuration dialog.
Refresh Refreshes the list of cluster sets displayed in the grid below.

Grid

Grid fields are described below.

Field Description
Name The name of the cluster set.
Description

indicates there is a description for this cluster set. Click the cell to view it.

indicates there is no description for this cluster set.
Hierarchy The site hierarchy that the cluster set is based on. Is blank if a site filter was used instead of a site hierarchy.
Level The specific level of the site hierarchy that the cluster set is based on. Is blank if a site filter was used instead of a site hierarchy level.
Filter

The site filter that the cluster set is based on. Is blank if a site hierarchy was used instead of a site filter.
Last Modified The date that the cluster set was last changed.
Last Computed The date that the cluster set was last calculated.

Allocation

Indicates whether or not the cluster set is configured for use with Allocation modules and allocation scenarios in the Rate of Sale module.

Assortment

Indicates whether or not the cluster set is configured for use with the Assortment Planning module and assortment planning scenarios in the Rate of Sale module.

Events

Indicates whether or not the cluster set is configured for use with the Events module.

Promotions

Indicates whether or not the cluster set is configured for use with the Promotions module.

Price

Indicates whether or not the cluster set is configured for use with the Price Manager module.

Snapshot pane

Select a cluster set in the grid to open the Snapshot pane.

Action Description
New Creates a new snapshot of the selected cluster set.
Edit Edits the snapshot selected in the snapshot grid.
Delete Deletes the snapshot selected in the snapshot grid.
Report Launches the Cluster Set Snapshot Comparison Report.
Refresh Refreshes the data in the snapshot grid.

The snapshot grid displays the following snapshot details. Its data is not editable.

Column Description
Cluster Set Name The name of the cluster set for which the snapshot was taken.
Cluster Set Description indicates there is a description for this snapshot. Click the cell to view it.

indicates there is no description for this snapshot.
Snapshot Date Time The date that the snapshot was taken.
Snapshot Note indicates there is a note for this snapshot. Click the cell to view it.

indicates there is no note for this snapshot.
Classification columns For each classification column ("A", "B", "C", etc.), the count of stores within each classification at the time the snapshot was taken.
Effective Date Time

The date that the cluster set snapshot became/will become effective.

Warning: Creating a new cluster set snapshot initiates recalculation of all assortment plans in which the snapshotted cluster set is active. This recalculation occurs as apart of regularly scheduled overnight processing; the frequency of which is determined during software implementation.
