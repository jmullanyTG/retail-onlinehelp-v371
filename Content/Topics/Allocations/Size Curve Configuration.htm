Size Curve Configuration

Size curve configuration allows the users to create the curves, define the ranges, and determine their constituent items.

  • Size Curve Name – User generated name for the size curve.
  • Size Curve Description – Optional description for the size curve.
  • Size Range – The pre-defined set of acceptable sizes to be included in the curve. The size range data must be imported.
  • Filter – The product filter which will be used as a base of historical data to generate the size curve.
  • Access Level – Access levels determine which users may use and/or edit the size curve.
  • User Select – Displays the specific users for a “shared” access level size curve.
  • Sales Type – Type of sales in historical sales which will be used to generate the curve.
  • Start Date – The beginning history date from which sales data will be used to calculate the size curve.
  • End Date – The ending history date from which sales data will be used to calculate the size curve.
  • Last X Days – Number of preceding days of sales history data which will be used to calculate the size curve.
  • Calculate – If checked, the size curve will calculate upon application.
  • Override Allowed – If checked, users may override calculated values within the size curve detail screen.
