Field Descriptions

  • Size Similarity ID: non-editable integer; displays the unique system ID number associated with the similarity set
  • Size Similarity Set Name: user editable alpha numeric; displays/sets the name for the similarity set
  • Size Type: user editable single-select dropdown; displays/sets the size type to define the similarity set for; populated with Size Types defined in the Allocation module
  • Display Size: user editable single-select dropdown; displays/sets the size name under which all similar sizes in the similarity set will be displayed in JustEnough after migration has occurred; populated with sizes contained in the selected Size Type
  • Similar Size: user editable multi-select dropdown; displays/sets the sizes to be grouped together in the similarity set; populated with sizes contained in the selected Size Type
  • Queue For Migration: non-editable checkbox; indicates that changes made to the similarity set have been saved and queued for migration in the next migration script run
  • Note: similarity sets that have been queued for migration but which contain sizes that have yet to be migrated will have this cell conditionally formatted red
  • Has Migrated: non-editable checkbox; indicates whether or not migration has been completed for the similarity set
  • Last Migrated Date: non-editable calendar control; displays the date the similarity set was last migrated
  • Last Migrated User: non-editable single-select dropdown; displays the user who last ran the migration script for the similarity set
  • Create Date: non-editable calendar control; displays the date the similarity set was created
  • Create User: non-editable single-select dropdown; displays the user who created the similarity set
  • Last Modified Date: non-editable calendar control; displays the date the similarity set was last modified
  • Last Modified User: non-editable single-select dropdown; displays the user who last ran the modified the similarity set
