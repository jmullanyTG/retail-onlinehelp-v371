Validating Enhanced Size Curves

You can set up pass/fail tests to ensure each of your sites had enough qualified data left over after data cleansing to calculate a "valid" size curve.

Note: Size curve validations are for informational purposes only and do not affect whether or not a curve is used by JustEnough. See Next steps for suggestions on what to do with size curves that fail validation.

Thresholds

Validations are based on the following user-defined thresholds, which appear as columns in the Size Curves grid.

Note: These thresholds measure the data-cleansed variants, sales units, and sizes at a site within the item filter selected for the size curve.

Threshold Description
# Variants Qualified The minimum number of variants required at a site for its size curve to be valid.

If, after data cleansing, the number of remaining variants at a site is less than this value, the curve validation fails for that site.

% Variants Qualified The minimum percentage of variants at a site that must pass data cleansing for the size curve to be valid.

If, after data cleansing, the remaining percentage of variants at a site is less than this value, the curve validation fails for that site.

Qualified Sales Units The minimum number of sales units required at a site for its size curve to be valid.

If, after data cleansing, the number of sales units at a site is less than this value, the curve validation fails for that site.
# Sizes Under The maximum number of sizes allowed at a site whose contribution to the size curve falls below the value defined in the % Sizes Under threshold.
% Sizes Under

The % contribution value to test against when validating # Sizes Under. (A size's'% contribution value is shown in the Size Curve Detail pane's Contribution column.)

For example, if this value is set to 10, and the # Sizes Under value is set to 2, then JustEnough checks the number of sizes with Contribution values below 10%. If more than 2 sizes at a site have contribution values below 10%, then the validation fails for that site.

# Sizes Over The maximum number of sizes to allow at the site whose contribution to the size curve is greater than the value defined in the % Sizes Over threshold.
% Sizes Over

The % contribution value to test against when validating # Sizes Over. (A size's'% contribution value is shown in the Size Curve Detail pane's Contribution column.)

For example, if this value is set to 25, and the # Sizes Over value is set to 2, then JustEnough checks the number of sizes with Contribution values greater than 25%. If more than 2 sizes at a site have contribution values greater than 25%, then the validation fails for that site.

You do not have to use all of these thresholds to test your size curves, just choose the ones that make sense for your data. Also, you can customize validations per curve, choosing different sets of thresholds and values for each curve as desired.

If you use more than one threshold to test a size curve, note that a curve must satisfy all thresholds in order to pass validation. JustEnough stops testing a site's curve once any threshold test fails.

Once you save changes to a size curve's thresholds, JustEnough automatically validates your site-specific curves against those thresholds and displays the results in the Size Curve Detail pane.

Setup

This section describes how to set up validations for your enhanced size curves.

  1. From the Manage tab, click Allocation to open the Allocation window.
  2. Click Size Curves on the toolbar to open the Size Curves grid.

  3. Find the row of the enhanced size curve you want to validate. (To tell if a size curve is enhanced or not, look at the curve's read-only Used Advanced Calc column. If the checkbox is marked, it is an enhanced size curve.)

  4. In the size curve row, enter the desired threshold values in the threshold columns described in the table above.

    Note: You can enter any combination of thresholds you choose. However, if defining Sizes Under or Sizes Over thresholds, you must enter values in both the # Sizes and % Sizes columns.

  5. Click Save to save the thresholds and execute the validations against the site-specific size curves. Results are shown in the Size Curve Detail pane.

    Note: Size curves are not automatically recalculated prior to validation. They are run against the current/existing size curve.

  6. Repeat steps 3-5 for each enhanced size curve you want to validate.

Validation results

Select a size curve in the Size Curves grid to view its validation results in the Size Curve Detail pane

The Pass? column indicates whether or not the size curve for each site passed validation.

Value Description
The size curve for this site passed validation.
The size curve for this site failed validation.

In addition, the site's Total Contribution column value appears red. (This is a visual indicator only and does not reflect anything "wrong" with the total contribution value itself.)

Next steps

If a size curve fails validation, you have the following options:

  • Manually override the size curve.
  • Substitute a size curve from another site.
  • Do nothing. JustEnough will continue to use the size curve "as is".
