Making Size Curve Substitutions

To sub one site’s curve for another, do the following:

  • Select the size curve header you want to make substitutions in from the list in the top area of the Size Curves Screen

 

  • Ensure the Override Allowed? field is checked

 

  • In the lower area of the Size Curves screen, select the desired site from the drop-down menu in the Override Site field of the site whose curve you want to override then click Save
  • Note: sites that have an Override Site assigned and Saved will not appear for selection in the Override Site drop-down menu
  • Note: to clear the Override Site prior to Saving, simply select the drop-down entry with a blank Site ID and No Substitution in the Description

 

  • The Contribution values of the overriden site’s size curve will now match those of the Override Site; the Clear Override field will be conditionally formatted orange, indicated an override has been made

 

  • Note: Is Eligible? settings for the Override Site will also be copied
  • Error: if you try to assign an Override Site to a site that is already being used as an Override Site, the following error will raise:

 

This is done to prevent circular and/or difficult to follow logic with the curves being substituted
