Strategy View

As there are many fields in the strategy view, it is advised to create saved layouts based on the item sets and how they are managed (planned demand, reactive demand modeling, etc.) and in these layouts, include or exclude the relevant metrics.

The strategy screen will be populated based on the filter. To create actionable allocation strategies, users must select a size curve, a location strategy, a product model, and set the demand signal.

Product Model

Selecting the ellipses (â€¦) in the product model field will launch the product models selection wizard.

Here, users may choose from the available product models which have been defined in the product model screen.

Size Curve Assignment

Within this field, users may select from the available size curves which have been created in the size curve screen using a drop down menu.

Location Strategy

Within this field, users may select from the available location strategies which have been created in the location strategy screen using a drop down menu.

Once selected, users may choose to refine the settings by location by using the location strategy exceptions.

Here, users may select a different size curve, set of trigger dates, demand shape, and other settings by location. This allows users full control of how the allocations are managed on a location by location basis.
