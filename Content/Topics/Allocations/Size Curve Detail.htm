Size Curve Detail

Size curve detail will display the contributions of each size to the variant, by store.

  • Size Curve Name – Name of the highlighted curve in the size curve configuration grid.
  • Site Name – Name of the site referenced in the site specific contributions.
  • Channel Description – Configurable site grouping.
  • Clear Overrides –Check box will appear with a highlight to indicate that there are override contributions within the curve. If checked, upon application, the calculation will clear the override contributions, but will not alter the eligibility flag.
  • Size Listings – Each size in the selected size range will appear as a column within the size curve detail screen.
  • Is Eligible – If selected, the size is considered to be eligible for allocation to the listed site. All eligible sizes will have an associated contribution percentage.
