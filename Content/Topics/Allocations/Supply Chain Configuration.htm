Supply Chain Configuration

Supply chain mapping is controlled through a maintenance and configuration view (Supply Chain Map Configuration). Both “ongoing item” and “replenishment” strategy selection types share a supply chain. The default supply chain for Adaptive Allocations is also controlled within this maintenance view. Values within the supply chain map configuration view will override any default values within system policy (e.g. allocations default active supply chain).

 

Note:   It’s important to note that if there is a different supply chain mapped to New Items vs. Ongoing items, the user must ensure that the correct AllocateStrategyName has been selected for each Allocation Strategy in which they are generating orders to stores.
