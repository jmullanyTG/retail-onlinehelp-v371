Product Models

Product models are built using calculation parameters and product based (item master) filters.

  • Product Model – The user supplied name for the product model.
  • Product Model Description – Optional description for the product model.
  • Model Time Frame Selection – The applied Define Time Frame for model calculation.
  • Sales Type – Included sales types (listed in sales history by type) for model creation.
  • Model Metric – Defines the measure used to calculate the model.
  • History Units – Quantity of historical sales units.
  • History Revenue – Historical revenue.
  • History Margin – Historical sales margin.
  • Future Plan Revenue – Planned revenue from Location Planning.
  • Future Plan Units – Planned sales units from Location Planning.
  • Future Plan Receipts – Planned receipts from Location Planning.
  • Future Forecast Revenue – Forecasted revenue.
  • Future Forecast Units – Forecasted sales units.
  • Future Forecast Margin – Forecasted sales margin.
  • Model Type - Model Type may be static (contribution with respect to its location only) and dynamic (contribution with respect to the hierarchy).
  • Product Selection – Product selection is built off of a product specific (item master) filter. These filters may be built using the JustEnough Filter Manager to incorporate the desired items used to build the model.
  • Default Product Selection – A supplemental filter to ensure that product models do not have “zero” values for a store. Should be set to a high enough level of the product hierarchy to ensure that values are present, or all itemmasters.
  • Access Level – Access levels determine which users may use and/or edit the model.
  • User Select – Displays the specific users for a “shared” access level product model.
  • Created – Revision history metric showing the creation date of the model.
  • Created By – User ID of the model creator.
  • Last Updated – Revision history metric showing the date of the most recent revision.
  • Updated By – User ID of the most recent editor.
