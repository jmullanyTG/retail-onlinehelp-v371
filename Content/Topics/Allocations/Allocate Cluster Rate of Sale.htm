Allocate Cluster Rate of Sale

For allocation strategies using the rate of sale by grade demand type, the included rate of sale grades must be selected and applied. This feature will allow users to bypass the need to associate a location strategy with an allocation strategy and further enhance the integrated process of assortment plans carrying into allocation strategies.

  • Included – If selected, the cluster set rate of sale grades will be included.
  • Cluster Set – The description of the cluster set.
  • Classification – The value of the classification/grade to be included or excluded.
  • Calculated Rate of Sale – The rate of sale value for the grade as calculated by the Rate of Sale Calculator module. This is the value which has been “pushed” to the strategy.
  • Rate of Sale Override – The by grade user override rate of sale value. If present, will be consumed instead of the calculated rate of sale.
