Add media

Media from a local or network source can be added to Media Manager repository.

  1. Click on Add to load the Explorer or Finder window.

 

  1. Select the Media file to add and click Open.
  2. Media Manager will store all relevant information in the software to access the Media in the future.
  3. The new Media will display in the Repository grid and will be available for users across the JustEnough solution suite.

 

The following file types are supported in Media Manager:

  • doc
  • docx
  • xls
  • xlsx
  • ppt
  • pptx
  • pdf
  • txt
  • jpg
  • gif
  • bmp
  • png
  • tiff
