Media Manager Home Screen

The Media Manager home screen displays all Media stored in the software in alphabetical filename order. Placeholders for New Image Requests are also displayed here. The screen displays a thumbnail of the Media along with other file attributes. Buttons are located above the Repository grid that provide the ability to add and remove Media.

 

The Repository grid displays the following information for each stored media:

Field

Description

Preview

A thumbnail image representing the image or document.

Name

The filename of the media.

File Type

If known, the software indicates the file type of the media.

Storage Type

Indicates the type of storage where the Media was originally located.

  • Local DB Storage – Displayed for all Media added manually using the software
  • File Storage – Displayed for Media imported via Repository import
  • Read Only From File System – Displayed for read only Media imported via Repository import

 

There are two buttons that provide additional information regarding each existing Media:

Action

Description

View

For images, a window displaying a larger preview of the image. See “Images window” on page 9 for details. For documents, clicking View will open the document, assuming the user has the necessary software installed locally to view the document.

Attributes

The file attributes will load in a window.
