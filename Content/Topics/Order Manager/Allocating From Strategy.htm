Allocating From Strategy

  1. From the Manage tab, click Order Manager.
  2. In the Allocate section of the toolbar, click From Strategy.

  3. In the Item dialog box, select the Item Master filter and Site filter to use for this order.

  4. In the order form, enter values for the following parameters:

    Parameter Description
    Workflow The workflow to follow for this order.
    Name The order document name.
    Description The order description.
    Owner The owner of this order.
    Security

    Whether this order is public, private, or shared with select others.

    • Private: This order is only visible to you.
    • Public: This order is visible to all users.
    • Shared: This order is visible only to the roles/users selected by clicking the Change button.
    Today's Orders
    This Week's Orders
    Future Recommended Orders
    		 The order horizon to use for this order.
    Include Zero Orders Select to enable this option for this order.
    Exclude Uploaded Orders Select to enable this option for this order.
    Macro Optional. The macro to run with this order. Click Disable to toggle enabling/disabling the selected macro.
  5. Click OK to generate the order.
