Before any prepack analysis can be done within Order Manager, prepack configurations must be defined for all variants you wish to prepack. To define prepack configurations, click Manage > Variant Manager

When prompted, select or define a product filter that includes the variants you want to define prepacks for and click OK

Once the document is visible on the screen, select the variant you want to create prepack configurations for and click the Details button in the upper right of the screen

This will show the Details area of Variant Manager. Click the Prepack tab to show the prepack configurations work area

  • Header Area Controls
  • Save: saves all changes made to the currently selected prepack configuration or template since the last time it was saved
  • Undo: undoes all changes made to the currently selected prepack configuration or template since the last time it was saved
  • Delete: deletes the selected prepack configuration or template
  • Warning: prior to deletion, the system will prompt you with a warning message asking for confirmation to delete

 

  • Add Prepack: creates a new blank prepack configuration in the Prepack > Manual folder at the left of the screen
  • Add Prepack Template: creates a new blank prepack template in the Template > Manual folder at the left of the screen
  • Prepack Folders (Left side)
  • Prepack
  • Import: prepacks configurations that have been imported into JustEnough from external sources will appear in this folder
  • Manual: prepack configurations created by clicking the Add Prepack button will appear in this folder
  • Process: future functionality; prepack configurations created by the prepack optimization engine will appear in this folder
  • Template
  • Import: prepack templates that have been imported into JustEnough from external sources
  • Manual: prepack templates created by clicking the Add Prepack Template button will appear in this folder
  • Process: future functionality; prepack templates created by the prepack optimization engine will appear in this folder
  • Prepack/Template Info (Center)
  • Code: user editable alpha-numeric; displays/sets the prepack configuration/template name to be used in reference throughout the system
  • Size Range (templates only): user editable dropdown; selects the range of sizes to be used in the prepack template
  • Description: user editable alpha-numeric; displays/sets a description for the prepack configuration/template
  • Prepack/Template Details (Right Side)
  • With a prepack configuration selected
  • Add: launches the Add Variant dialog

 

  • Variant: user editable dropdown; selects a variant to add to the prepack configuration
  • Size Curve: user editable dropdown; selects a size curve to apply to the variant being added
  • Template: user editable dropdown; selects a prepack template, instead of a size curve, to apply to the variant being added
  • Included: user editable checkbox; sets whether a given size will be included in the prepack configuration
  • Size: non-editable alpha-numeric; displays the size codes contained in the selected size curve or template
  • Unit Count: user editable integer; displays/sets the number of units per size to include in the prepack configuration
  • Cancel: closes the Add Variant dialog without adding a variant to the selected prepack configuration
  • OK: adds the variant, size, and unit count information defined in the Add Variant dialog to the prepack details area
  • Remove: removes the selected variant or size from the prepack configuration details
  • Warning: the system will not ask for confirmation prior to removing variants/sizes, however, prior to being saved, these changes can be reversed with the Undo button
  • Variant: non-editable alpha-numeric; displays the variant codes of those present in the selected prepack configuration
  • Size: non-editable alpha-numeric; displays the size codes included in the selected prepack configuration for the corresponding variant
  • Unit Count: user editable integer; displays/sets the number of units per size for the corresponding variant to be included in the prepack configuration
  • With a prepack template selected
  • Size Range: non-editable alpha-numeric; displays the size range set for the selected prepack template
  • Size: non-editable alpha-numeric; displays the size codes included in the selected prepack template
  • Unit Count: user editable integer; displays/sets the number of units per size for the corresponding variant to be included in the prepack configuration
