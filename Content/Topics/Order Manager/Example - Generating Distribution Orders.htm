Example - Generating Distribution Orders

In this example, we are going to generate distribution orders for the new item, "High Collar Angelina Jacket". These orders will deliver the items to stores in the United States from DCs stocked in the Generating Suppler Orders example.

Workflow

Click to open the Generating Orders workflow.

Related video

Instructions

  1. From the Manage tab, click Order Manager.
  2. In the Order toolbar, click Distribution.

  3. In the Items dialog box, select the filter with the items and sites you want to generate orders for.

  4. In the Add Supplier Order dialog box, enter the parameters for the order:

    • Supply Chain: select the [Redistribution] supply chain
    • Filter: Ordering: High Collar Angelina Jacket/USA
    • Select Today's Orders
  5. Click OK.

  6. Once JustEnough calculates the recommended orders, they appear in the Order screen. Each row represents a different recommended order, and contains basic details of the order.

  7. Double-click on an order row to view the demand forecast, stocking strategy, stock holding projection, and other details that drove the order recommendation.
