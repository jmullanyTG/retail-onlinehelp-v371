Creating Commitment Orders

Commitment orders can be created within Assortment Planning by using the Create Commitments button within the Receipt Plan supporting view (see Line Visualization > Supporting Views > Receipt Plan View for details), but doing so only allows you to create orders for one placeholder or collection at a time. While this may be perfectly fine in many cases, Order Manager gives you the ability to create commitment orders from assortment plans for entire sections of the product and site hierarchies and windows of time at once. To initiate the creation of new commitment orders in Order Manager, simply click the Commitment button in the top toolbar:

Doing so will launch the Add Assortment Plan Order dialog
