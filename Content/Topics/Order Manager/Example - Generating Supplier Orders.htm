Example - Generating Supplier Orders

In this example, we will generate supplier orders for the new item, "High Collar Angelina Jacket". These orders will deliver the items from the supplier to DCs in the United States for later distribution to stores.

Workflow

Click to open the Generating Orders workflow.

Instructions

  1. From the Manage tab, click Order Manager.
  2. In the Order toolbar, click Supplier.

  3. In the Items dialog box, select the Ordering: High Collar Angelina Jacket/USA filter

  4. In the Add Supplier Order dialog box, enter the parameters for the order:

    • Supply Chain: Select the [Supplier - DC - Store] supply chain.

    • Select Today's Orders.

  5. Click OK.

  6. Once JustEnough calculates the recommended orders, they appear in the Order screen. Each row represents a different recommended order, and contains basic details of the order.


  7. Double-click on an order row to view the demand forecast, stocking strategy, stock holding projection, and other details that drove the order recommendation.
