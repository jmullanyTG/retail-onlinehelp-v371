Selecting a Different Forecast Method

If the forecast method used for a given forecast has a low method fit or just does not make sense (for example, the item is not seasonal but a seasonal method was used), you can "force" a different method to be used for the forecast.

Related video

Instructions

  1. In the Forecasts screen, click Lock on the toolbar.

  2. In the Forecast Task dialog box, select only the methods that you want JustEnough to consider when forecasting demand for the item.

    For example, if seasonal forecast methods shouldn't be used, make sure their checkboxes are not selected.

  3. Click OK to close the dialog box.
  4. Click Forecast in the toolbar to re-generate the forecast using only the selected forecast methods.

Results

The selections made in the Forecast Task dialog box remain valid until new selections are made or the Lock Expiration date defined in the dialog box is reached.

To select different methods after a forecast has been "locked", click Unlock on the toolbar to re-open the Forecast Task dialog box.
