Current Consumption Exceptions

The Current Consumption exception flags level forecasts when the sales for the current period do not match the forecasted demand for the current period.

Formula

This exception is based on the variance between the demand forecasted for the current period and the sales in the current period. The formula for determining the percentage of variance is: 

(Current Forecast Rate - Current Demand Rate) / Current Demand Rate

Where:

  • Current Forecast Rate= the result of dividing the current period's Effective Forecast quantity by the number of days in the forecast period. (For example, if the forecast period is a week long, divide by 7).
  • Current Demand Rate = the result of dividing the current period's Total Sales quantities (which represents week-to -date sales) by the number of days that have passed since the first day of the period (excluding the current day).

A positive value resulting from this calculation occurs when the forecast for the current period is greater than the current demand, which may result in excess stock. A negative value occurs when the demand for the current period is greater than the forecast, which may result in stock outs.

Whether the resulting value constitutes an exception depends on the variance threshold defined in the exception filters.

Notes:

  • If either the Current Forecast Rateor Current Demand Rate value is equal to zero, the variance percentage will equal 100%.

Filter conditions

To filter for forecasts with Current Consumption over-forecast exceptions

From the Level Forecast Exceptions field group:

    • Field LevelForecastCurrentConsumptionPercent is more or equal to 0.9 (or whatever positive percentage value you want to be the threshold).
    • Field LevelForecastReviewed is False.

To filter for forecasts with Current Consumption under-forecast exceptions

From the Level Forecast Exceptions field group:

    • Field LevelForecastCurrentConsumptionPercent is smaller or equal to -0.9 (or whatever negative percentage value you want to be the threshold).
    • Field LevelForecastReviewed is False.
