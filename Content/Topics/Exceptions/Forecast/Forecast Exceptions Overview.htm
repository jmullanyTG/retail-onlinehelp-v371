Forecast Exceptions Overview

JustEnough pro-actively tests demand forecasts against an array of criteria to determine if a forecast needs additional human review. When it identifies such a forecast, JustEnough flags the forecast with an exception so that it can be easily found later. You can use filters to find and review these forecasts by exception type.

Forecast exception types

JustEnough tests for the following types of forecasts exceptions. Click the link for each exception type to view examples and more information about the exception.

  • Zero forecast - The total forecast for an upcoming period is zero.
  • FC vs LY Demand Variance - The forecast for one or more upcoming periods is exceedingly above or below the item's sales history for the same period(s) last year.
  • FC vs TY Demand Variance - The forecast for one or more upcoming periods is exceedingly above or below the item's sales history for the previous number of periods.
  • Current Consumption - Sales fore the current period are exceedingly above or below the current period forecast.
  • Model Fit - The forecast has a poor MAPE (Mean Average Percentage Error).
  • Sales Spikes - Sales greatly exceeded forecasted demand in one or more periods.
  • Lost Sales - Sales did not match forecasted demand in one or more periods due to stock outs.
