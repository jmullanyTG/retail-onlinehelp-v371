Manually Adjusting a Forecast

You can manually adjust a forecast by entering new values in the grid or by dragging the Manual Level Forecast data series on the graph.

Related video

Instructions

Manually adjusting forecasts via the grid

  1. In the grid, enter a new value in the Manual Level Forecast field for the desired time period.

    The graph updates automatically to reflect the newly entered value.

  2. Click Save and Forecast to save the changes and re-generate the forecast using the manual adjustments. The Manual Level Forecast data series in the graph reflects the manual adjustment.

 Manually adjusting forecasts via the graph
  1. In the grid, make sure the Manual Level Forecast field checkbox is selected. (If it is not, the Manual Level Forecast data series does not appear in the graph).

  2. In the graph, click on the Manual Level Forecast value you want to adjust and drag it to the desired value. In the grid, the Manual Level Forecast field is updated automatically to reflect the manually adjusted value.

  3. Click Save and Forecast to save the changes and re-generate the forecast using the manual adjustments. The Manual Level Forecast data series in the graph reflects the manual adjustment.

 

Notes

  • Level forecasts have two Manual Forecast fields: Manual Forecast and Manual Level Forecast. The Manual Forecast field reflects manual forecast quantities captured at the Inventory Forecast level. This field will not be editable in the Level Forecast view.
  • You must re-forecast an item to push the Level Manual Forecast quantities to the Inventory level.
