Zero Forecast Exceptions

The Zero Forecast exception flags level forecasts when the total forecast for an upcoming period is equal to zero.

Note:
  • If an item as never sold, it will not be marked as a Zero Forecast exception.

Filter conditions

To find forecasts with Zero Forecast exceptions, create a filter that includes the following fields and conditions.

From the Level Forecast Exceptions field group:

  • Field LevelForecastZeroForecast is True
  • Field LevelForecastReviewed is False.
