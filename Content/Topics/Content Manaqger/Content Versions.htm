Content Versions

Content versions let you create variations in product content due to language, geography, or other advertising considerations. Each content version can have its own set of fields and be assigned to one or more site clusters.

Notes:

  • Site clusters are logical groupings of stores created in JustEnough's Cluster Site Manager.
  • When adding a promotion to an event, all content versions are associated to the event by default unless Version Prompt has been enabled for the event slot set.

  • Content versions are configured by JustEnough Professional Services and should not be modified without assistance from JustEnough Professional Services, JustEnough Support, or a trained on-site resource.

Content Versions tab

You can manage content versions from the Content Versions tab of the Content Manager Configuration window.

Content Version Actions

Action Description
New Creates a new content version.
Edit Edits the selected content version.
Delete Deletes the selected content version.
  • You cannot delete a content version if it is associated with a promotion.

Content Version Fields

Field Description
Content Version Name The name of the content version.
Content Version Description The description of the content version.
Geography list The site clusters assigned to the content version.
