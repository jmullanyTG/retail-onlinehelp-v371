Modify a Content Field

To modify a Content Field, select the field and click on Edit.

Modify the following fields as needed:

Field

Description

Content Field (Required)

Modify the descriptive label of the Content Field. Content Field labels must be unique.

Content Type (Required)

If the Content Field can be configured further, this column will appear in the grid as a dropdown list. The following Content Types are available for a Content Field. For fields that can be modified, the default value is listed first:

  • Checkbox (Read Only)
    This option, determined by the database configuration (i.e. not modifiable), will display a checkbox for marking in the user interface.
  • Date and Time (Read Only)
    This option, determined by the database configuration (i.e. not modifiable), will display a date and time field that displays a calendar and clock interface when edited in the user interface.
  • Droplist (Editable)
    This option displays either a single droplist value or multiple droplist values in the user interface.
  • Droplist (Single), Droplist (Multiple)
  • Blue pencil icon indicates that the Droplist values are editable
  • Hyperlink (Read Only)
    This option, determined by the database configuration (i.e. not modifiable), will display single textbox with a clickable hyperlink.
  • Numeric (Read Only)
    This option, determined by the database configuration (i.e. not modifiable), will display a field that only accepts numeric entry in the user interface.
  • Text (Editable)
    This option displays either a multiline textbox or a single line textbox in the user interface.
  • Text (Long), Text (Short)

Read Only (Optional)

Mark this checkbox if the field must not be editable by users within the Content windows accessed from Promotion, Event and Content Manager.

Active (Optional)

Ensure this checkbox is marked for all active fields. To hide a Content Field from the user, unmark this checkbox. Content Fields cannot be deleted using Content Manager configuration since they are created directly in the database.
