Add

If the Image is stored locally or on the network:

  1. Click Add to load the local computer’s Explorer window.
  2. Navigate to the desired Image(s) and click Open.

 

 

Note: The ability for a user to add a local or network image is controlled by the system policy Suppress Content Image Card’s Add Option. If this policy is set to True, users can only add images that already exist in the database (the Add button is removed from the Images toolbar).
