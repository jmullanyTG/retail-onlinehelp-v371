Edit a template filter product group

When a product group is created using a Template Filter, the template values are transformed into a standard filter. Modifying a product group created using this method is the same as modifying a product group created using a Product Filter. The Edit Group window will display the filter created from the Template Filter.

Action

Description

Group Name

Modify the defaulted Group Name as needed.

Product Filter

Select a different existing filter from the dropdown list if needed.

Modify the Filter

If a Product Filter was selected from the dropdown list, a blue hammer icon will display. To modify the selected filter click on the blue hammer icon. The black hammer icon provides the ability to modify the existing or create a new filter.

 

Click Ok to save the updates.
