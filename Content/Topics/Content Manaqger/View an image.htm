View an image

The Images tab displays a preview of the Image.

The following actions can be used to view Images or information about the Image:

Action

Description

View

Clicking View or double-clicking the image will display a larger preview of the Image in a window.

Attributes

Clicking this button will display the Image Attributes window for the selected Image. If more than one Image is selected, Attributes will display for the last selected Image in the grid.

NOTE: Image Attributes are not available for New Image Requests.

 

The Images window also provides additional buttons to view different sizes of the Image.

Action

Description

Fit Image

Adjust the Image dimensions to fit in the window.

Actual Size Image

Displays a full size version of the Image in the window.
