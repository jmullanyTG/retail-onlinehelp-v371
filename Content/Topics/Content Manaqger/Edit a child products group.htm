Edit a child products group

If the product group consists of other product groups, the Edit Group window will display the child groups.

Action

Description

Group Name

Modify the defaulted Group Name as needed.

Add Group

Select one or more child groups to add to the product group.

Edit

Select a child group before clicking on this button.

If the selected child group contains Item Masters or Variants, the child group filter will load in a second Edit Group window.

If the selected child group contains other product groups, a second Edit Group window will load listing selected childâ€™s product groups and modifications can be made as needed.

NOTE: Modifying an existing filter will permanently update the filter and will affect other product groups using this filter.

Remove

Select a child group in the grid and click Remove to remove it from the product group. This button is available when a child group is selected.

View Products

Select a child group and click View Items or View Variants to view the Item Masters or Variants respectively that comprise the child group.

NOTE: This button is not available if the child group contains other product groups.

Click Ok to save the updates.
