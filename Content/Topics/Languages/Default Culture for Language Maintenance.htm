Default Culture for Language Maintenance

A single language can have multiple different Cultures, each with its own set of translations. 

Essential Business Concepts

The culture setting will allow users to further localize their settings within a specific language.

  • Language – The language which has been selected from the drop down menu.
  • Culture – The culture setting specific to the language.
  • Default Flag – The default culture if the language is selected.

Practical Application

Languages may be selected from the drop down menu.

For languages with available culture settings, the cultures will appear once the language has been selected.

Technical Execution

Changes made to the culture settings will require a “save” at the end of the session.
