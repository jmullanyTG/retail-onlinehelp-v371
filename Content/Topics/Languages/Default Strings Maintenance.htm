Default Strings Maintenance

The default strings maintenance screen will show current string names and their “friendly name” or translated string value, along with the date of change.

Essential Business Concepts

This is used to add default strings to the selection available for translation.  Strings added need to be valid. 

  • String Name – The string name as is it used by the application itself. Without a translation, this is the value which will appear in the user interface.
  • String Value – The translated value or friendly name.
  • Date Time – The date on which the translation was created or edited.
  • Note – Any applicable notes.

Practical Application

Users may use the search bar in order to locate individual strings (or use a contextual keyword). New entries may be made by clicking on the “Click here to add a new row” section in the grid.

Sting values may be edited by clicking on the “string value” field.

Technical Execution

The defaults languages files may be imported or exported as a JustEnough Defaults Language (.dlan) file type.
