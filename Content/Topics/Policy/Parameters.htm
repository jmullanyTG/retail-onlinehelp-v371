Parameters

The parameters used to return data to the pivot grid will all be used to narrow the scope of inventories.

  • Filter – An inventory filter. If ItemMaster Hierarchy and Site Hierarchy parameters are left blank, all inventories within the filter will be returned to the grid.
  • ItemMaster Hierarchy – Segments of the product hierarchy may be selected in order to narrow the scope within the inventory filter selected in the “Filter” parameter.
  • Site Hierarchy - Segments of the location hierarchy may be selected in order to narrow the scope within the inventory filter selected in the “Filter” parameter.
