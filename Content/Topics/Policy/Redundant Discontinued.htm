Redundant/Discontinued

The following settings apply to inventories set to normal as an order policy type.

  • Return Rate – The percentage of products which are expected to be returned to the store.
  • Days Before Return – The expected number of days between the sale of a product and the return of the product to the store.
  • Reuse Rate – The percentage of products which have been returned and may be considered as saleable stock again.
  • Days Before Reuse – The expected number of days between the return date of a product and it being considered as regular saleable stock again.
  • Order Multiple – The multiple of units in which the stock must be ordered.
  • Order Multiple Rounding – The rounding rules used to determine how order quantities which do not precisely align with the order multiple should be adjusted.
  • Up – Rounds up to the nearest order multiple.
  • Down – Rounds down to the nearest order multiple.
  • Nearest – Rounds to the nearest order multiple (round natural).
  • Down Protected - Rounds down unless the action of rounding will create a projected lost sale, in which case it will round up.
  • Nearest Protected – Rounds nearest (natural) unless the action of rounding will create a projected lost sale, in which case it will round up. This is the recommended setting.
  • Excess for Redistribution Days – They days of excess which should be considered to be “protected” from a redistribution action. As a “days” value, the calculated unit value will be a resolved function of projected demand.
  • Expedite Freeze Days – The forward looking days within which orders should not be considered eligible for an expedite action.
  • Surplus Freeze Days – The forward looking days within which orders should not be considered eligible for a surplus action.
