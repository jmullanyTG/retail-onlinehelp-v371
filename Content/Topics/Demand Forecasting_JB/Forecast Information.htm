Forecast Information

The Forecast screen shows the following information for each forecast:

  • Forecast Date
  • Review Date
  • Forecast Method
  • Method Fit

The forecast method name displayed will represent the tournament winner and its respective method fit based on MAPE (mean absolute percentage error). Using the forecast method drop-down list, users may view the results of the forecast tournament for that particular item. They will be ordered by their respective method fit scores. Any method with a result of 0.00% is considered to have been disqualified from the tournament and the reason for its disqualification will be listed in the “forecast message” column.

Forecast parameters will display similar information to the details tab, but gives users an opportunity to compare the different methods at a glance.

The forecast method fit is a percentage value indicating how well a forecast method matches sales history. As part of the forecast tournament process, a method fit array is generated starting with the first sales history value and progressively determining what the forecast would have been with the selected method.  This array and the actual sales history values for the same period are compared using mean absolute percentage error (MAPE), to calculate a forecast model fit.

  • A = actual demand for (t) time
