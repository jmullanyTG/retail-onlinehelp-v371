Technical Execution

System configuration must be done before users may create level forecasts.

Level Definition

To define the level, users must create a specific inventory hierarchy for level forecasts, then describe that hierarchy in a system policy (Engine: Forecasts â€“ Level Forecast Hierarchy). The inventory hierarchy must be created using the hierarchy filtering maintenance view.

Split Factors

Split factors are created one of two ways: they may be system generated based on the existing inventory level forecasts, or they may be written in as custom split factors. There is no standard user interface for custom split factors.
