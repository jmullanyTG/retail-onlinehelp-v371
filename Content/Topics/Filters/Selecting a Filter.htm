Selecting a Filter

A typical filter selection dialog box is shown below.

From this dialog box, you have the following options:

  • Open the Filter list to select an existing filter.
    • Static data filters appear in orange text.
    • Static condition filters appear in black text.
    • Dynamic condition filters appear in blue text.
  • Click the Conditions button () to select the dynamic condition(s) to apply to the filter.
    • Appears only when a dynamic filter is selected.
  • Click the Refresh button () to rebuild the filter using the most current data.
    • Refreshing static filters is allowed.
  • Click the Maintain button () to open the Maintain Filters dialog box.
    • Choose Select only some items from the current filter to return only a subset of the data included in the selected filter.

Selecting dynamic conditions

When you select a filter with dynamic conditions, click the Conditions button () to select the condition to apply for this instance of the filter.

In the Filter Conditions dialog box, select the value you want to use for the filter and click OK.
