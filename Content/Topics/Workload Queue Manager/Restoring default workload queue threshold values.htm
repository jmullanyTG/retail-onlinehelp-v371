Restoring default workload queue threshold values

If you want to stop using the customized thresholds for a given process type, you can reset the thresholds to their default values.

Note: This activity requires a user to have the Allow Access to Workload Queue Configuration right.

  1. In the Workload Queue toolbar, click Configuration.
  2. In the Workload Queue Configuration dialog box, expand the desired process type.
  3. Click the button in the Reset column to restore all priority thresholds for the process type to their default values (as shown in the Current Values column).

  1. Click Save to save the changes.
  2. Close the dialog box when finished making changes.
