Workload Queue Configuration

Workload Configuration should only be accessed by JustEnough Professional Services and Support to ensure the integrity of corporate strategies and guiding policies to enforce standards before transmitting exported data to downstream systems. The following solutions are integrated with the Workload Queue:

  • Allocations
  • Assortment (Receipt) Plan
  • Assortment (with Order Management) Prepack
  • Inventory Forecast
  • Inventory Replenishment (with Order Manager)
  • Group Forecast
  • Price Management

While the Workload Queue comes with ‘out of the box’ configuration, for the above JustEnough solutions, it is expected that tuning will be required based on available hardware resources. Only the JustEnough Professional Service and Support teams should implement tuning modifications.

Workload Queue Configuration dialog box

The Workload Queue Configuration dialog box makes it easier to fine-tune workload thresholds for supported process types.

For each supported process type (Allocations, Assortment Prepack, etc.), you can view the default and customized threshold ranges for each priority level (Immediate, ASAP, Low Priority, Night, or Weekend). Each column in the dialog box is further described below.

Column

Description

Priority

Supported process types are listed in alphabetical order. Grouped under each process type is the set of priority levels for that process type. These levels are listed from most- to least-urgent: Immediate, ASAP, Low Priority, Night, and Weekend.

Current Threshold

The default min-max value range for a priority level. These values do not change.

Override Min

The customizable minimum value for a priority level.  If the default thresholds are being used, this field is empty.

Override Max

The customizable maximum value for a priority level.  If the default thresholds are being used, this field is empty.

Reset

Click to delete the customized values in the Override Min and Override Max columns and return to using the default values shown in the Current Threshold column.
