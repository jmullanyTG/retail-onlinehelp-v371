Customizing workload queue priority thresholds

You can override the default priority thresholds for any given process type.

Note: This activity requires a user to have the Allow Access to Workload Queue Configuration user right.

  1. In the Workload Queue toolbar, click Configuration.
  2. In the Workload Queue Configuration dialog box, expand the process type you want to customize thresholds for.

  1. Define the new thresholds by entering the desired minimum and maximum threshold values in the Override Min and Override Max columns for each priority level.

  1. Click Save to save the threshold overrides.
  2. Close the dialog box when finished making changes.
