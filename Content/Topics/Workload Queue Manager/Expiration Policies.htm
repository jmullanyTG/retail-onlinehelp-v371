Expiration Policies

Global Policy

Description

Default Expiration Time

Daily queued tasks will expire being moved from the active queue to archive without being processed.

  • Priorities – Immediate, ASAP, Low Priority

Default Nightly Expiration Time

Nightly queued tasks will expire being moved from the active queue to history list without being processed.

  • Priority – Nightly

Default Weekend Expiration Time

Weekly queued tasks will expire being moved from the active queue to history list without being processed.

  • Priority – Weekend
