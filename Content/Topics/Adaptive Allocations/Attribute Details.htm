Attribute Details

The attribute details section allows users to select which of the initially included attribute families should be considered as when determining allocation quantities.

  • Include in Weighting – If selected, the attribute family will be considered as part of the draft process.
  • Attribute Name – The name of the attribute family.
  • Attribute Description – The optional description for the attribute family.
  • Weighting Factor – A factor by which attributes will be weighted against one another. Defaults to 100% (even weighting for all included attributes).
