Size Ranges and Default Curves

Orthodox configuration of size ranges is required in order to correctly sample supporting data and create selection options. A size range should be configured to support the full breadth of possible sizes for a variant across all locations. As a new size range is generated, a default curve will be created (system generated) to cover size curve option values for all sites. Users wishing to specify a default curve may generate one by giving specific size curve creation parameters (sales types, etc.), then populating the “Default Size Curve” column within the size range manager view.
