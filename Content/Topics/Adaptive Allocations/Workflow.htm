Workflow

A simple workflow related to scenario plan configuration is enable in order to prevent premature execution of a scenario plan.

The scenario plan may be saved at any time (and changes reversed by using the “Undo” button) and the configuration status and plan status fields will display information which may be used to complete the construction of the scenario plan. Specific details on configuration exceptions may be viewed by clicking on the ellipses (“…”) in the configuration status field; this will launch the configuration exceptions pop-up.
