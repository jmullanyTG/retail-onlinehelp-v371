Location Plan

If “location plan” is selected as the demand source, the following fields will appear within the detail configuration window (please note that there is no need for a product model as financial targets have already been set by store):

  • Product Hierarchy – This is a read-only field which will display the system policy defined product hierarchy. It is purely informational.
  • Product Selection – A mapping of the applicable levels within the product hierarchy which will be sampled to create the demand plan.
  • Plan Name – A selection of the available strategic merchandise financial plans.
  • Include Core Products – If core products are contained within the planned financial targets, they may have their contribution removed by de-selecting the “include core products” flag. Core products are defined as any product-location combination with a demand forecast value greater than zero for a future period through management in JustEnough Demand Forecasting and Replenishment or Allocations.
