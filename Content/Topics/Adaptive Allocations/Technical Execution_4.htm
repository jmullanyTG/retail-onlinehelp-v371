Technical Execution

The order creation process will create an order document with values for the following fields based on the allocation volume results:

  • OrderHeaderID
  • InventoryID
  • ItemMasterID
  • SiteID
  • CurrencyID
  • CostPrice
  • PrePackOrderLineID
  • PrePackItemMasterID
  • PrePackRatio
  • SupplierOrderLineID
  • UserOrderDate
  • UserShipDate
  • UserDeliveryDate
  • UserOrderQuantity
  • UserSourceSiteID
  • UserSourceSupplierID
  • UserFromSupplierFlag
  • UserCostPrice
  • UserPrePackOrderQuantity
  • RecommendedOrderDate
  • RecommendedShipDate
  • RecommendedDeliveryDate
  • RecommendedOrderQuantity
  • RecommendedSourceSiteID
  • RecommendedSourceSupplierID
  • RecommendedFromSupplierFlag
  • RecommendedPrePackOrderQuantity
  • ConstrainedOrderDate
  • ConstrainedShipDate
  • ConstrainedDeliveryDate
  • ConstrainedOrderQuantity
  • ConstrainedSourceSiteID
  • ConstrainedSourceSupplierID
  • ConstrainedFromSupplierFlag
  • ConstrainedPrePackOrderQuantity
  • CreatedUserID
  • CreatedDateTime
  • LastModifiedUserID
  • LastModifiedDateTime
  • IsDeletedFlag

These fields may be altered as needed through the application of an Order Manager macro.
