Choice Counts

The choice count results view will display the calculated choice count by store, alongside the historical choice count for the same store as used in the sampled time frame. This is informational and should be used for reference when validation choice count results. All stores within the location strategy will be displayed.

  • Site Code – The unique code for the site.
  • Site Description – The description for the site.
  • Choice Count – The calculated choice count result.
  • Historical Choice Count – The historical choice count used within the calculation.
