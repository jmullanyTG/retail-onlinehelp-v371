Applying a Change to Multiple Records in a Grid

These instructions show you how to change a field value in multiple records at a time.

  1. In the grid, select the records that you want to change.

  2. Right-click on the field value you to change and choose Multiple Changes.

  3. In the Multi Data Change Form dialog box, enter the new value for the field.

  4. Click Save to apply the change. The new values appear in the selected records.
