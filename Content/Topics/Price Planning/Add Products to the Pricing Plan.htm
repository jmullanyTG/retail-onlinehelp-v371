Add Products to the Pricing Plan

To include additional products in the Pricing Plan, click Add on the Pricing Plan Variant grid. The solution provides two methods to identify additional products: using filter criteria or using the system clipboard.

Add from Filter

Clicking Add from Filter will display the filter window so that you can choose an existing filter or click Maintain Filters to create a new filter. Once a filter is selected click OK to append any new products to the existing product listing and click Save to commit any changes.

If active Pricing Action(s) exist, the system will allow you to apply the new products using the Actionâ€™s Summary level entered values. If you are unable to add products, the Pricing Action may be in a status that does not allow modification with your rights.

Add from Clipboard

You will be able to add products directly from their system clipboard with the Add from Clipboard button. The copied list should have one (1) product (i.e. Variant or Item Master, global policy) per line, followed by a carriage return line feed.

The system will validate all entries and will log all failed products for investigation by you. Successful entries will be added to the Pricing Plan and the system will allow you to act immediately on the new products by adding to a Pricing Action.
