Create Rounding Rule Criteria (Standard)

Once a Rounding Rule has been created and selected, the system will allow the creation and association of multiple criteria, by currency, which may be applied to a Calculation. From the Price Manager Configuration Rounding Criteria area, click New, enter the required fields and click Save.

Field

Description

Currency (Required)

The dropdown is populated by the available currencies specified by each Site. Select a currency for the criteria configuration which will be applied only to the applicable Site(s) when the system is calculating Price Actions.

Position (Required)

The numeric position the system will modify based on the configured ‘high’ and ‘low’ criteria being.

Mid-Point (Required)

The low range value to be evaluated by the system.

Active (Optional)

Determines if the Rounding Criteria will be made used when a Rounding Rule is configured with a Calculation.
