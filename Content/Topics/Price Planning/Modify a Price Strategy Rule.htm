Modify a Price Strategy Rule

Modifications can be made directly to the Price Strategy Rule grid. Modifications cannot be made to the Rule Type and Rule Calculation fields. Modification of a Price Strategy Rule will cause associated Price Actions to be marked for recalculation/evaluation, unless already exported.

Clicking Save will save any modifications. Click Cancel to discard any modifications.

 

Modifying the Priority Order

Enter the new priority order value for the Price Strategy Rule to be modified and click Save. The system will re-order the existing priorities accordingly.

Example: Priority Order 1 – 7 and you modify #5 to become #2

  • #1 will remain #1
  • #2 becomes #3
  • #3 becomes #4
  • #4 becomes #5
  • #6 stays #6
  • #7 stays #7

Modifying the Validation Severity
Select from the dropdown a new Severity Type. If ‘error’ is selected you will have the ability to mark the field to allow ‘error overrides’.
