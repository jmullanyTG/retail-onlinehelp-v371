Add Sites to the Pricing Plan

To include additional sites in the Pricing Plan, click Add on the Pricing Plan Site grid, which will display the Sites Filter window. Select an existing filter or click Maintain Filters to create a new filter. Once a filter is selected click OK to append the new sites to the existing site listing and click Save to commit any changes.

If active Pricing Action(s) exist, the system will allow you to apply the new sites using the Actionâ€™s Summary level entered values. If you are unable to add sites, the Pricing Action may be in a status that does not allow modification with your rights.

Modify Plan Start Date
When modifying the Plan Start Date, the system will restrict moving the date forward any further than by any Pricing Action Start Date.

Example:
Original Plan Start Date = 01/01/2016
Earliest Price Action Start Date (already exported) = 02/01/2016
Result: System will not allow a new Plan Start Date greater than 02/01/2016

Remove products from the Pricing Plan
To remove products from the Pricing Plan, click Remove on the Pricing Plan Variant grid. The solution provides two (2) methods to identify products to be removed: manually selected or using the system clipboard.

Manual Selection

To remove products from the Pricing Plan, select and mark the checkbox adjacent to the appropriate product(s) in the Pricing Plan Variant grid and click Remove Selected.

The system will not allow removal of product(s) from the Pricing Plan if the product(s) are being used in a Pricing Action.

Remove from Clipboard

You will be able to remove products directly from their system clipboard with the Remove from Clipboard button. The copied list should have one (1) product (i.e. Variant or Item Master, global policy) per line, followed by a carriage return line feed.

The system will validate all entries and will log all failed products for investigation by you. For successful entries and upon your confirmation, the system will remove the products and recalculate the Pricing Action(s).

Remove Sites from the Pricing Plan
To remove Sites from the Pricing Plan, select and mark the checkbox adjacent to the appropriate site(s) from the Pricing Plan Sites grid and click Remove.

The system will not allow removal of site(s) from the Pricing Plan if the site(s) are being used in a Pricing Action.
