Create a Price Strategy

Within the Price Manager Configuration Strategy screen, click New to create a new Price Strategy, enter the required fields and click Save.

Field

Description

Price Strategy Name (Required)

Enter a descriptive label to summarize the purpose of the Price Strategy.

Price Strategy Description (Optional)

Enter a summary of the Price Strategy.

Default Price Strategy (Optional)

Ability to designate a Price Strategy to be set when creating a new Pricing Action.

Price Strategy Active (Optional)

Unmarking the checkbox, will remove the ability to configure the Calculation in any new Price Strategies.
