Create a Rounding Rule (Standard & Price Ending)

Within the Price Manager Configuration Rounding Rule screen, click New to create a new Rounding Rule, enter the required fields and click Save.

Field

Description

Name (Required)

A descriptive label to summarize the purpose of the Rounding Rule.

Description (Optional)

A summary of the Rounding Rule.

Active  (Optional)

Determines if the Rounding Rule will be made available for configuration with a Calculation or not.
