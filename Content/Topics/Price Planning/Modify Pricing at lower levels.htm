Modify Pricing at lower levels

Entering data in the Price Strategy Variable grid will populate to all Price Action Variants and Sites included in the Price Action (except excluded)

To specify different values, from what is configured at the Price Action level (i.e. Price Strategy Variable grid), modifications may be made at three lower levels: Action Summary, Item Detail and Location

  • Action Summary
    Values are populated down from the Price Action Variable grid down to the Action Summary, but you have the ability to modify those values within the Action Summary itself.
    Enter a new value for the applicable Variant(s) variable(s) within the Price Action Summary grid. Modifying a value for a specific Variant will apply the modified entry to all lower levels (i.e. Item and Item Location Detail). The system will also denote any differences within a level by shading the cell. Lower level differences will be noted by a marked ‘exception’ checkbox.
  • Item Detail
    Click Detail for a specific Variant, within the Price Action Summary grid, to display the Variant Detail values at the Item and Item Location levels. The Item Detail (top) grid displays the individual products across all valid Sites.
    Enter a new value for the applicable Item’s. Modifying a value will cause the system to shade the cell and apply to all sites which can be viewed in the Item Location Detail (bottom) grid. Modifications may also affect calculated results which will be aggregated in the Price Action Summary grid as well as the marking of the ‘exception’ checkbox for ease of filtering.

  • Item Location Detail
    Selecting a row, in the Item Detail grid will populate the Item Location Detail (bottom) grid which displays the individual product(s) by each Site
    Enter a new value for the applicable Product(s)/Site combination. Modifications may also affect calculated results which will be aggregated in the Price Action Location Detail and Summary grids as well as the marking of the ‘exception’ checkboxes for ease of filtering.
