Add a Price Strategy Rule

Once a Price Strategy has been selected, click New (i.e. in the Strategy Rule grid) to add a Price Strategy Rule, enter the required fields and click Save.

Field

Description

Sequence Order (Required)

Enter a descriptive label to summarize the purpose of the Price Strategy.

Price Strategy Description (Optional)

Enter the priority order of the Strategy Rule. The system will re-order the existing priorities accordingly, if appropriate.

Rule Type (Required)

Select whether a ‘Calculation’ or a ‘Validation’ will be configured. Selection of ‘Type’ will populate the Rule Calculation dropdown.

Rule Calculation (Required)

Select the calculation/validation to be used in the Price Strategy from the dropdown. Only ‘active’ calculation and validations will be populated to the list.

Calculation Description (Read Only)

The system will populate the Calculation Rule Description, which is not editable in this screen.

Validation Severity (Required)

Select how the system should evaluate Calculation Rule failures. Severity can only be set for ‘validation’ types.

  • Information – Upon failure (Low Priority), the system will provide notification, in the validation log, but will continue calculation and evaluation of any following Price Strategy Rules
  • Warning – Upon failure (Medium Priority), the system will provide notification, in the validation log, but will continue calculation and evaluation of any following Price Strategy configured Calculation Rules
  • Error – Upon failure (High Priority), the system will provide notification, in the validation log, and will not continue calculation and evaluation of any following Price Strategy configured
  • Error Override (Optional) – This checkbox may be marked to allow users, with permissions, to override validation failures so the system will continue calculations and validation evaluation.

Calculation Rule Active (Optional)

Unmarking the checkbox, will cause the system to not evaluate the configured Calculation Rule when calculating/evaluating Price Strategies.
