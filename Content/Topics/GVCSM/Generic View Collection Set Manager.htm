Generic View Collection Set Module Overview

The Generic View Collection Set Module is a top-level dock for custom view collections and tabs which can be used as shortcuts to Views that are useful and frequently accessed. This module provides a View Collection Set structure independent of other features so that the included views and widgets will only load when needed. Mi9 Technical Consultants may add new view collection sets to this dock as needed for custom functionality.

Access rights

The access right Access To Generic View Collection Set Configuration is required.

Case sensitivity

If users wish to add a new View to an existing page or group, they must ensure that they enter the existing plugin, page, or group name exactly as is. Entering a plugin, page, or group name that does not exactly match case results in a new plugin, page, or group being created. For example, Influence and influence would be two different tabs on the ribbon.

Creating a custom view

To create a custom view:

  1. Navigate to Generic View Collection Set Module.
  2. Click Add on the toolbar.

  3. Complete the fields as required.

    To add a View to an existing group or tab, enter the existing group’s name in the Plugin Group Name field.

  4. In the Plugin Image field, click the drop down arrow and select an icon for the plugin.
  5. Click the ellipsis in the View Collection Set field and select a view from the Maintain View Collection Set dialog.
  6. Check the check box in the Enabled field to display the plugin when the new plugin is saved.

  7. Set the access level by clicking the drop down arrow in the Plugin Access Level field and choosing the appropriate option.

    If the plugin access level is set to other than Shared, then the Plugin Users field is disabled.

  8. Optionally, if the plugin access level was set to Shared, then In the Plugin Users field, click Change to display the Maintain Users And Roles dialog:

    1. Set the Roles and Users as required.
    2. Click Ok to close the dialog.
  9. Click Save.

Field definitions

Field Description
Plugin Name The plugin name as it displays in the Plugin Group in the ribbon.
Plugin Description The description that displays when a user hovers the mouse over the icon.
Plugin Display Order The position in which the plugin displays. The left most position is Plugin Display Order 0 (zero).
Plugin Group Name

The name of the group as it will appear when the module is selected.

Plugin Group Description The plugin group description.
Plugin Group Display Order The position on the ribbon where the new plugin group will appear.
Plugin Page Name The name of the module as it will appear in the ribbon.
Plugin Page Description The plugin page description.
Plugin Page Display Order The position on the ribbon where the new plugin page will appear.
Plugin Author The user who created the plugin.
Plugin Image An image selected by the user.
View Collection Set The collection set to display.
Enabled

Default is unchecked. If checked, then the new plugin displays when Save is clicked.
Plugin Access Level Whether the plugin is private to the user who created it or shared with other users.
Plugin Users The Roles and Users with access to the plugin. The Change button is enabled when Plugin Access Level is set to Shared.
