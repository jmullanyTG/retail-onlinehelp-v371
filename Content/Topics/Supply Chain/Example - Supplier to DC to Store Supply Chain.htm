Example - Supplier to DC to Store Supply Chain

This example shows a supply chain where external suppliers deliver to distribution centers (DCs), which then distribute stock to stores.

When we first create the supply chain in this example, it will be flat. You'll see all sites (stores and DCs) on the same level in the supply chain hierarchy, and the parent source for each site is the external supplier. To create a supplier > DC > store supply chain, we'll have to organize the stores "below" their regional DCs in the hierarchy, with each store having a DC as its parent source.

We'll finish the example by choosing a second DC to be an alternate source for a store.

Example

The JustEnough Standard Demo database includes an example [Supplier - DC - Store] supply chain based on the instructions provided below.

Instructions

1. Create the initial supply chain.

  1. From the Maintain tab, click Supply Chain.

  2. In the Supply Chain toolbar, click New.

  3. In the Create Supply Chain dialog box, enter a name for the supply chain and select Site/Supplier as the supply chain type.

  4. The supply chain appears in the Supply Chain screen. In the Hierarchy pane, note that the organization is flat (stores and DCs are on the same level). In the following steps, we'll change the inventory source for stores from "External Supplier" to their actual DCs.

  5. In the Hierarchy toolbar, click Switch View to change the hierarchy view to a grid.

  6. Next, we'll add columns to the grid to make it easy to identify which sites should move to which DCs In the Hierarchy grid, right-click on a column header and choose Column Chooser.

  7. In the Customize pop-up, double-click on Channel Description, Country Description, and Site Region Description to add these columns to the grid.

  8. Filter the Channel Description column and select Retail, then filter the Country Description column and select Canada.

  9. Right-click in the grid and choose Select All to select all stores.
  10. Right-click in the Parent Site column and choose Multiple Changes.

  11. In the Multi Data Change Form for Column Parent Site dialog box:

    1. Select the New Value option.
    2. Choose DC - Canada as the new value.

    3. Click Save.

  12. The Hierarchy grid now shows DC-Canada as the new parent site for Canadian stores.

  13. Click Save in the Supply Chain toolbar to save the changes.
  14. Continue changing inventory sources for your other regions until all stores have their primary DC as their inventory source.
 2. Define alternate sources for stores.

In this example, we will set DC-North America as an alternate source of inventory for the Toronto store.

  1. Click Switch View to return to the Hierarchy view. You will see that Canadian stores are now children of DC-Canada in the hierarchy.
  2. Click on Toronto in the hierarchy.

  3. In the Alternate Source grid, find DC-North America and enter 2 in its Priority column.

  4. Click Save in the Supply Chain toolbar to save the change. DC - North America is now set as the second source of inventory for Toronto.

  5. Continue assigning alternate sources for your other stores.
  6. Click Reprocess in the Supply Chain toolbar to process the saved changes and rebuild the supply chain.

Results

With a supplier to DC to store supply chain, JustEnough will:

  1. Calculate projected demand versus projected stock on hand for each item at each store.
  2. Calculate stock needed in each DC to meet the demand of stores stocked from that DC.
  3. Generate orders from external suppliers as needed to stock the DCs with enough inventory to meet the collective store demand.
  4. Generate distribution orders to deliver the right amount of inventory from DCs to stores to meet each store's projected demand.
