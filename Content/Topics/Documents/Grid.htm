Grid

The grid button in the upper right section of each Site Manager document will toggle the top grid widget on and off. Upon creation of the document, the grid will be populated by the default columns. Additional columns may be added using the “choose columns” functionality, or through creation of calculated columns.

Default Columns

  • Site Code – Code used to identify the site throughout the organization. This is an imported field.
  • SiteID – A unique identifier for the site used within JustEnough.
  • Site Name – The name for the site.
  • Priority – The global site priority value. It is read only within the top gird.
  • Priority Group – The priority group value. It is read only within the top grid.
  • Currency – The default currency for the site.
  • Channel Code – Code used to identify the associated explicit channel attribute.
  • Channel Description – Optional description for the channel.

Choose Columns

Several other columns are selectable for inclusion in the grid section of the document. The “choose grid section of the document. Saved views may be saved and loaded using the “choose fields” window. Additional columns may be added to the choose columns functionality through configuration of the filter metadata.

Site Capacity

Site capacity data may be returned within the grid view of the site manager. It is strictly informational.

  • Total Selling Capacity – Total quantity of stock (in units) which may be considered “for sale” at the site.
  • Total Stockroom Capacity – Total stockroom capacity indicates the quantity of stock (in units) which may be stored in the stockroom.

Site Group

All relevant site group information may be returned within the grid view.

Attribute

Any imported or created site attributes. These are defined by each organization.

Site Schedules

Site shipping and receiving schedules may be set within the grid view of the site manager by selecting the shipping and receiving schedule columns from the “choose columns” option list.

  • Receiving Schedule – Determines the set of dates (inclusive or exclusive) on which the site may receive shipments of stock. Schedules must be created within the Schedules Module.
  • Shipping Schedule – Determines the set of dates (inclusive or exclusive) on which the site may ship stock. Schedules must be created within the Schedules Module.

Site Exclusion

If site product exclusions exist (Site Product Exclusion tab), they may be filtered down for further investigation based on the value in this field.

  • Product Exclusion Exists – Will display a checked box if any exclusions exist for the site.

Site Type

Site type data will define the role of the location within organization as well as its existence within the plane of reality (if it is virtual or a logical partition within a real location).

  • Site Type – Defines the role of the site within the organization (e.g. store or DC).
  • Is Contract – Checked if a site is a contract site (i.e. a persistent logical partition).
  • Contract Site – If a site is marked as a contract site, this field lists the associated physical site to which is it linked.

Calculated

Users may create calculated columns using any logical combination of the existing columns. These columns are built using the expression editor.
