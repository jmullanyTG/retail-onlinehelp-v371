As stated above, the Rate of Sale Calculator can be used to model scenarios to help a user determine how much, in units or revenue, a particular placeholder/collection might sell per site in a given week. All of the functionality essential to Assortment Planning is available in this stand-alone version of the tool, however a scaled down version also exists directly in the assortment planning module.  The stand-alone version, can be accessed via Plan > Rate of Sale Scenario.

 

There are three main options in determining rate of sale for a placeholder/collection in planning demand with assortment planning:

  • Smart Start: this method leverages a marginal return curve to determine rates of sale for each placeholder required to fulfill a target that is set in Target Creation
  • Like Style Modeling: manually create placeholders/collections and model those choices after historical like items
  • Rate of Sale Calculator: using this tool after a user has either smart started their assortment or manually created placeholders/collections, users can refine assortment plans by using historical data to determine an appropriate rate of sale to apply to placeholders/collections within them; modeling in the Rate of Sale Calculator is extremely flexible in terms of the product, cluster set, and time dimensions that can selected in order to model demand with confidence

The Rate of Sale Calculator can be broken down into three main components.

  1. ROS Scenario Creation
  2. ROS Calculation
  3. ROS Application
