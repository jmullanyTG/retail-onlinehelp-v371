Insert a Token within a Message

Tokens can be used to provide dynamic content. To insert a Token:

  1. Place your cursor where the dynamic content should appear in the Notification Message.
  2. Select the ‘Token’ option from the Insert menu.

The system will insert a generic ‘token’, with the required token formatting (i.e. [%...%]) encompassing the generic ‘token’ text. Replace the ‘token’ text with the matching parameter that will be used to provide the dynamic content.
