Insert a Source Link within a Message

Source Links are used to direct the user to the desired Solution. To insert a Source Link:

  1. Place your cursor where the link should appear in the Notification.
  2. Select the ‘Source Link’ option from the Insert menu.

The system will insert the Source Link formatting (i.e. *[]*) at the required point and the cursor will be placed between the square brackets. Anything between the brackets will be underlined in the message. Please note that it is possible to insert a Token between the brackets as well so that the Source Link has dynamic text.
