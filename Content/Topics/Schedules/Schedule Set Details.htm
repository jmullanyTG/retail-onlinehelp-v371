Schedule Set Details

Highlighting a schedule set within the navigation pane will display the schedule set details.

  • Name – The schedule set name.
  • Active Flag – Users may control the enabled behavior for the schedule set within the details view.
  • Description – An optional description field for the schedule set.
  • Schedules – Information related to the schedules contained within the schedule set will be displayed.
  • Enabled – Users may control the enabled behavior for the schedules within the details view.
  • Schedule Name – The name of the schedule.
  • Mode – The current behavior of the schedule within the context of the schedule set.
  • Summary – A summary description of the schedule behavior.
  • Resolved Dates – A listing of the resolved dates within the schedule set. Range for the resolved dates is from today’s date to ten years out.
