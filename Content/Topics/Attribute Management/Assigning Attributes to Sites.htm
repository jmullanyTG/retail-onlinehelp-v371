Assigning Attributes to Sites

Product attributes can be assigned to sites through Site Manager

Site Manger is structured and functions very similarly to Variant Manager, with users specifying a site filter, or selecting specific sites from a site filter, to create a document in which to work with those sites.

 

 

One a site manager document is open, users can select the site(s) to be worked with in the top, or Grid, section of the document. Details for the selected sites will appear in the bottom, or Details, section.

 

Users can work with a variety of aspects of sites within a variant manager document by clicking the appropriate tab in the Details section. To assign or edit site attributes for the selected site(s), click the Site Attribute Management tab.

 

Note: both the Grid and Details sections of the view can be toggled on and off by clicking the corresponding buttons () in the upper right of the view.

 

  • Site Attribute Management
  • Site Hierarchy: user editable dropdown; selects the site hierarchy for which to assign attributes
  • Note: because it is possible to create alternate site hierarchies which contain the same sites, it is also possible to have different attributes assigned to a site across those hierarchies; ensure the correct site hierarchy is selected before proceeding
  • Parameters: shows/hides the Parameters area of the Details section
  • Execute: updates the Details section of the view according to the selected Site Hierarchy and the site(s) selected in the Grid section
  • Family Name: non-editable alpha numeric; displays the name of the attribute family
  • Family Description: non-editable alpha numeric; displays the description for the attribute family
  • Single Values: user editable dropdown; displays/sets the attribute value of the adjacent single-select attribute family for the selected site(s)
  • when sites are selected that have different attribute values assigned, this field will show “(Multi)” to indicate this
  • Multiple Values: user editable multi-select; displays/sets the attribute values of the adjacent multi-select attribute family for the selected site(s)
  • when multiple sites are selected that have different attribute values assigned, this field will show the combination of all values present
  • Apply: saves all changes made to attribute values for the selected site(s) since the last Apply
  • Cancel: cancels all changes made to attribute values for the selected site(s) since the last Apply and reverts them to previously saved values
  • Data Population: the view will populate with all attribute families that any of the selected sites is eligible for; attribute values available for selection will include all values that ANY of the selected sites is eligible for
  • Applying Changes Across Multiple Items: when multiple sites are selected in the Grid section, the system will automatically resolve any conflicts that exist between the Applied values and the attribute value eligibility of the sites the change is being applied to
  • Single-select attribute value changes will be applied to all sites that are eligible for them; ineligible sites will remain unchanged
  • Multi-select attribute value changes will be applied additively to sites that are eligible for them; if only some of the applied values are eligible, those will be applied; ineligible sites will remain unchanged
