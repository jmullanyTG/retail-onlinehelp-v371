Attribute Management and Assignment Overview

Attribute management has traditionally been restricted to defining core attributes in host systems and pushing these to the planning solution for use in reporting and other grouping functions. JustEnough has introduced the attribute management module which allows for the maintenance of new attribute families and storage of imported attribute families. In addition, JustEnough has introduced the concept of attribute sets which allow users to define groups of attributes which are allowed to be assigned to certain hierarchies of product.

Assignment of attribute values now becomes a simple task for users to accomplish throughout the JustEnough system. This section will cover attribute management as it relates to assortment planning.
