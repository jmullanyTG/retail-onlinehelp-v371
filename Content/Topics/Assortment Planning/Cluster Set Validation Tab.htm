Cluster Set Validation Tab

  • Function: Displays the validation status of cluster sets for the current plan

  • Standard JustEnough Pivot Grid functionality, including field customization, saved layouts, and ability to Export or Open analysis in a variety of popular formats, including Microsoft Excel and Adobe PDF
  • Window of Time: user editable tree multi-select; sets the window of time to display cluster set validations for in the Assortment Cluster Set Validation section
  • Execute: populates the Assortment Cluster Set Validation section with validation statuses for the selected Window of Time

Note: Cluster sites must be configured for use with Assortment Planning (their Module property must include Assortment Planning).
