Promotion Application Details Section

This section displays placeholder/cluster details for the application selected in the Promotion Application section. Users can make placeholder/cluster level changes to the application Start and End Dates, Final Indices, and Hold Sales parameters.

  • Fields: opens the fields Customization dialog; users can drag fields to/from the tabâ€™s header area and this dialog to customize the view

 

  • Custom layouts can be saved/restored by right-clicking in the grid and selecting Save Layout or Restore Layout, respectively
  • Apply: saves all changes to promotion application details made since the last Apply
  • Cancel: undoes changes to promotion application details made since the last Apply
  • Product Eligibility: non-editable alpha-numeric; displays the Product Filter used to define the placeholders to include in the selected promotion application
  • PH/PLM Phase Dates: non-editable alpha-numeric; displays a timeframe representing the Phase In and Phase Out dates of each placeholder included in the selected promotion application
  • PH/PLM Code: non-editable alpha-numeric; displays either the placeholder codes or PLM codes (for placeholders with variants assigned) for all placeholders included in the selected promotion application
  • PH/PLM Description: non-editable alpha-numeric; displays either the placeholder descriptions or PLM descriptions (for placeholders with variants assigned) for all placeholders included in the selected promotion application
  • Approximation button: launches the Assortment Promotion Approximation dialog
  • Note: All fields are the same as at the promotion application level, but the data displayed is only for the specific placeholder on the line the button was clicked on; aggregations are only of clusters assigned to that specific placeholder rather than all placeholder/clusters included in the entire application
  • Cluster Sections
  • Start: user editable single select tree; sets the start date (as a period) for promotion application for the specific placeholder/cluster; defaults to the Start Date for the promotion application
  • Note: this date cannot be set prior to the Start Date for the promotion application
  • End: user editable single select tree; sets the end date (as a period) for promotion application for the specific placeholder/cluster; defaults to the End Date for the promotion application
  • Note: this date cannot be set after the End Date for the promotion application
  • Hold Sales: user editable checkbox; sets whether the promotion application has the ability to increase total sales of the specific placeholder/cluster
  • Checked: the total sales of the placeholder/cluster will remain the same as before the promo was applied, and only magnitude of sales in each period will be adjusted
  • Unchecked: the total sales of the specific placeholder/cluster can increase above the magnitude prior to promo application; this is effected through recalculation of the placeholder/cluster rate of sale
  • Included: user editable checkbox; sets whether the specific placeholder/cluster will be included in the promotion application once it is made active
  • Final Index: user editable index (1.01); sets the Promotional Index for the specific placeholder/cluster; defaults to the Final Index of the selected promotion application
  • Note: Cluster sections for placeholders that are not currently assorted to the corresponding cluster will be grayed out and locked for editing
