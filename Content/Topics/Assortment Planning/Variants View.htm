Variants View

For users who prefer to work in Collection mode, the Variants supporting view is provided to still give access to the individual placeholders that exist within each collection. Variants view will automatically update to show the placeholders within the selected collection

 

Note: Variants view is only available in Collection mode

 

 

  • Header area controls
  • The header controls in Variants view are essentially the same as those in the Main Planning view, with the exception of having the ability to change the data filter parameters, since the view is linked to the selected collection in the Main Planning view
  • For details on the available functionality, see Header Area Controls and Main Planning View earlier in this section
  • Planning Area and Metrics
  • The planning area in Variants view functions the same as the Main Planning view in Placeholder Mode, including all metrics, the ability to add/remove columns via the column chooser in View Options, and the ability to save and restore layouts
  • See Main Planning View above for details
  • Style Bank
  • Users are even able to use the style bank to assign variants to placeholders in Variants view in the exact same manner as in Placeholder mode of the Main Planning View
  • See Style Bank above for details
