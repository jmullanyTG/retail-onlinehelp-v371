Scenario Details

This section of the Scenario Manager screen displays details about the segments users have defined for the scenario(s) selected in the Target Creation Scenarios section. If multiple scenarios are selected, segment data for each will be stacked in the view.

  • Scenario: Non-editable alpha-numeric; name of the selected scenario(s)
  • Target Segment: Non-editable alpha-numeric; names of target segments defined for the selected scenario(s)
  • Target Filter: Non-editable dropdown; product filter used to define historical data for the segment
  • Target Filter Generated: non-editable checkbox; using Segment By Hierarchy to define segments checks this box, which indicates that the system will use a temporary filter based on the product hierarchy level corresponding to the segment in place of a standard Target Filter (see Target Creation – Target Segment Definitions below for details)
  • Model Filter: Non-editable dropdown; product filter used in place of the Target Filter for the purpose of recommended choice count generation for the segment
  • Not required
  • Product Hierarchy: Non-editable tree dropdown; level of the plan’s product hierarchy to which placeholders generated for the segment will be assigned to
  • Cluster Sets: Non-editable multi-select; cluster set(s) used to define the site component of the segment
  • Limits the historical data returned when calculating scenarios in Target Creation
  • Comparison Time Frame: Non-editable time frame selector; specifies the time frame used to gather historical data for the segment
  • Note: a value of “LY” indicates the system will use the direct last year equivalent periods for the Window of Time defined for the scenario.
  • Collection Option: Non-editable dropdown; specifies the configuration of collections to which placeholders generated for the segment will belong to; see Target Creation for details
  • New Variants: Non-editable integer; maximum number of new choices defined for the clusters assigned to the segment
  • Total Variants: Non-editable integer; maximum number of new choices + carryover choices defined for the clusters assigned to the segment
  • Active Variant Threshold: non-editable percentage (10.00%); displays the % of deseasonalized average sales removed during step 3 of data cleansing (see Rationalization above
