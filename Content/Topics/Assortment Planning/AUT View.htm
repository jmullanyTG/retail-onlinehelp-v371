AUT View

AUT view allows the user to make period level changes to AUT (average unit ticket price) for the purpose of planning mid-season permanent price adjustments; this view can be used to make such changes even after a placeholder/collection has begun to actualize

 

  • Apply: saves all AUT changes made since the last Apply and initiates recalculation of blended AUR values for the selected placeholder
  • Note: placeholder/collection level AUT calculates as a weighted average based on sales units of the periods in the phase window for each cluster
  • Cancel: cancels all changes made since the last Apply and reverts them to previously saved values
  • Export: standard Pivot Grid functionality; exports and saves the Pivot Grid in many common formats including Microsoft Excel and Adobe PDF
  • Open: standard Pivot Grid functionality; opens the Pivot Grid in many common formats including Microsoft Excel and Adobe PDF â€“ does not require save prior to viewing in the desired format
  • Graph: standard Pivot Grid functionality; graphs the selected data in the view
  • Fields: standard Pivot Grid functionality; allows the user to customize the field layout for the view
  • Custom layouts can be saved and restored by right-clicking in the Pivot Grid and selecting Save Layout or Restore Layout, respectively
  • Period: indicates the period being changed; this aligns with the period type for the current assortment plan
  • Cluster Set: indicates the cluster set containing the cluster to be changed
  • Cluster: indicates the cluster to be changed
  • Conditional Formatting: orange shaded cells in the AUT view indicate a change in AUT from the prior period in the column
