Target Creation Scenarios

  • Scenario Name: User editable alpha-numeric; name for the scenario
  • Scenario Description: User editable alpha-numeric; description for the scenario
  • Scenario ID: Non-editable integer; System ID number for the scenario
  • Create Date: Non-editable date; date the scenario was created on
  • Create User: Non-editable alpha-numeric; the username of the person who created the scenario
  • Plan Basis: Non-editable dropdown; name of the MFP plan used as a basis for sales volume in the scenario; not required
  • Start Date: Non-editable date; beginning date of the scenario’s Window of Time
  • End Date: Non-editable date; end date of the scenario’s Window of Time
  • Product Hierarchy: Non-editable alpha-numeric array; Product Hierarchy level(s) planned within the scenario
  • Site Hierarchy: Non-editable alpha-numeric array; Site Hierarchy level(s) planned within the scenario
  • Scenario Target: Non-editable currency or integer (based on the Scenario Target Type); the planned sales volume in revenue or units for the scenario
  • Scenario Target Type: Non-editable dropdown; the type of target sales volume for the scenario; can be either Revenue or Units
  • Sales Types: Non-editable alpha array; comma delimited list indicating the sales types included in the scenario
  • New Variants: Non-editable integer; sum of the maximum number of new choices for each segment within the scenario
  • Total Variants: Non-editable integer; sum of the maximum number of new choices + carryover choices for each segment within the scenario
  • Has Been Smart Started?: Non-editable checkbox; indication of whether the scenario has even been Smart Started; default is unchecked
  • Status: Non-editable dropdown; indicates the current state of the scenario within the established workflow; valid values are Created, Calculated, Rationalized, and Smart Started
  • Privacy: User editable dropdown; indicates the privacy setting for the scenario; valid values are Public, Private, and Shared
  • Sharing Settings: Non-editable alpha-numeric array; comma delimited list of roles and/or users who can view/edit the scenario; only valid if Privacy is Shared
