Assortment Planning Dashboard

The assortment planning dashboard can be found in the Assortment section of the software. There are three standard dashboards that ship with the assortment module, each containing information specific to Assortment Planning.

This dashboard contains four tabs Assortment Buy Sheet Report, Assortment Choice Count by Period Report, Assortment Choice Count Report, and Assortment Upcoming Commitments Report. The first three tabs are utilized to aggregate the item level plans to provide more holistic views as it relates to key measures contained within the Assortment module, and the fourth is used to track upcoming due dates for key milestones in the commitment process for orders generated from assortment plans. Each of these tabs will be detailed in this document.
