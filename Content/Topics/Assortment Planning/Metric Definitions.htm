Metric Definitions

After the user has set the parameter selections at the top of the screen, clicking Execute will return the following metrics in the Assortment Rollup:

  • System Sales– sales generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • System Sales Ex VAT – sales excluding value added tax generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Sales – sales coming from the sales plan view; this value will match the system sales line unless a user has overridden the sales plan in the sales plan view
  • Sales Ex VAT – sales excluding value added tax coming from the sales plan view; this value will match the system sales line unless a user has overridden the sales plan in the sales plan view
  • Comparison Sales – sales coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • Comparison Sales Ex VAT – sales excluding value added tax coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Sales – last year sales coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • LY Sales Ex VAT – last year sales excluding value added tax coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • System Sales % +/- LY – percentage increase or decrease of the system sales versus LY sales
  • Sales % +/- LY – percentage increase or decrease of sales versus LY sales
  • System Sales Build Ratio – ratio of system sales of current period versus previous period system sales
  • Sales Build Ratio – ratio of sales of current period versus previous period sales
  • LY Sales Build Ratio – ratio of last year sales of current period versus last year sales of previous period
  • System Sales Units – sales units generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Sales Units – sales units coming from the sales plan view; this value will match the system sales line unless a user has overridden the sales plan in the sales plan view 
  • Sales Units (Comparison) – sales units coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Sales Units – last year sales units coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • System Sales Units % +/- LY – percentage increase or decrease of the system sales units versus LY sales units
  • Sales Units % +/- LY – percentage increase or decrease of the sales units versus LY sales units
  • System Sales Units Build Ratio – ratio of system sales units of current period versus previous period system sales units
  • Sales Units Build Ratio – ratio of sales units of current period versus previous period sales units
  • LY Sales Units Build Ratio – ratio of last year sales units of current period versus last year sales units of previous period
  • AUT – the average unit ticket price of all products contained in a given aggregate level
  • Comparison AUT -- the average unit ticket price of all products contained in a given aggregate level from the selected comparison snapshot
  • LY AUT – last year average unit ticket price of all products contained in a given aggregate level
  • AUR – the average unit retail of all products contained in a given aggregate level
  • Comparison AUR – the average unit retail of all products contained in a given aggregate level coming from the selected comparison snapshot
  • LY AUR – last year average unit retail of all products contained in a given aggregate level
  • AUR % +/- LY – percentage increase or decrease of AUR versus last year AUR
  • Sellthrough – by period sellthrough percentage.  (Sales Units/BOP Units)
  • Note: this is not a rolling calculation and looks at each period individually
  • LY Sellthrough – last year by period sellthrough percentage.  (LY Sales Units/LY BOP Units).  This is not a rolling calculation and looks at each period individually
  • System COGS – cost of sales generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • COGS – cost of sales coming from the sales plan view; this value will match the system cost of sales line unless a user has overridden the sales plan in the sales plan view
  • Comparison COGS – cost of sales coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY COGS – last year cost of sales coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • COGS % +/- LY -- percentage increase or decrease of cost of sales versus LY cost of sales
  • System AUC – aggregated average unit cost price generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • AUC – average unit cost price coming from the sales plan view; this value will match the system AUC line unless a user has overridden the AUC in the receipt plan view
  • Comparison AUC – average unit cost price coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken.
  • LY AUC -- last year aggregated average unit cost price coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • AUC % +/- LY -- percentage increase or decrease of average unit cost price versus LY average unit cost price
  • System Margin -- margin generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Margin – margin coming from the sales plan view; this value will match the system margin line unless a user has overridden the sales plan in the sales plan view or AUC in the receipt plan view
  • Margin (Comparison) – margin coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Margin – last year margin coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • Margin % +/- LY -- percentage increase or decrease of margin versus LY margin
  • System Margin % -- margin % generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Margin % -- margin % coming from the sales plan view; this value will match the system margin % line unless a user has overridden the sales plan in the sales plan view or AUC in the receipt plan view
  • Margin % (Comparison) – margin % coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Margin % -- last year margin % coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • Margin Rate % +/- LY -- percentage increase or decrease in margin % versus last year margin %
  • System BOP Units – Beginning of Period units of inventory generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • BOP Units – Beginning of Period units of inventory coming from the sales plan view; this value will match the system BOP Units line unless a user has overridden the sales plan in the sales plan view or receipt plan in the receipt plan view
  • BOP Units (Comparison) – Beginning of Period units of inventory coming from the selected comparison snapshot; this value would include any overrides from the user at the time the snapshot was taken
  • LY BOP Units – last year Beginning of Period units of inventory coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • BOP Units % +/- LY -- percentage increase or decrease of BOP Units versus LY BOP Units
  • System BOP Cost -- Beginning of Period cost of inventory generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • BOP Cost -- Beginning of Period cost of inventory coming from the sales plan view; this value will match the system BOP Cost line unless a user has overridden the sales plan in the sales plan view or receipt plan in the receipt plan view
  • Comparison BOP Cost -- Beginning of Period Cost of inventory coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY BOP Cost – last year Beginning of Period cost of inventory coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • BOP Cost % +/- LY -- percentage increase or decrease of BOP Cost versus LY BOP Cost
  • System Avg BOP Unit Cost – average cost of inventory per unit generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the product filter selected for this report
  • Avg BOP Unit Cost – average cost of inventory per unit coming from the sales plan view; this value will match the System Avg BOP Unit cost line unless a user has overridden the sales plan in the sales plan view or receipt plan in the receipt plan view
  • Avg BOP Unit Cost (Comparison) – Avg BOP Unit Cost of inventory coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Avg BOP Unit Cost -- last year Avg BOP unit cost of inventory coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • Avg BOP Unit Cost % +/- LY -- percentage increase or decrease of Avg BOP Unit Cost versus LY Avg BOP Unit Cost
  • System Receipt Units – receipt units generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Receipt Units – receipt units coming from the sales plan view; this value will match the system Receipt Units line unless a user has overridden the receipt plan in the receipt plan view
  • Receipt Units (Comparison) – receipt units coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Receipt Units -- last year receipt units coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • Receipt Units % +/- LY -- percentage increase or decrease of Receipt Units versus LY Receipt Units
  • System Receipt AUC -- aggregated average unit cost price of receipts generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Receipt AUC -- average unit cost price of receipts coming from the sales plan view; this value will match the system receipt AUC line unless a user has overridden the AUC in the receipt plan view
  • Receipt AUC (Comparison) -- average unit cost price of receipts coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Receipt AUC -- last year aggregated average unit cost price of receipts coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • Receipt AUC % +/- LY -- percentage increase or decrease of average unit cost price of receipts versus LY average unit cost price of receipts
  • System Receipt Cost – receipt cost generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Receipt Cost – receipt cost coming from the sales plan view; this value will match the system Receipt Cost line unless a user has overridden the receipt plan in the receipt plan view
  • Receipt Cost (Comparison) – receipt cost coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Receipt Cost -- last year receipt cost coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • Receipt Units % +/- LY -- percentage increase or decrease of Receipt Cost versus LY Receipt Cost
  • Minimum Presentation – number of units in aggregate bought for presentation
  • On Order Units – receipt units for which orders have been submitted to external systems and for which order confirmations have been received
  • On Order units var to Receipt Units – the difference between planned receipt units from the receipt plan view and On Order Units
  • LY On Order Units – last year receipt units for which orders have been submitted to external systems and for which order confirmations have been received
  • Note: relevant for planned values only in scenarios where planning is occurring further than 12 months out from the current date; actual receipts delivered will be actualized and displayed as LY Receipt Units
  • On Order Units % +/- LY -- percentage increase or decrease of On Order Units versus LY On Order Units
  • On Order AUC -- average unit cost price of On Order Units coming from the sales plan view
  • On Order Cost – cost of receipt units for which orders have been submitted to external systems and for which order confirmations have been received
  • On Order Cost % +/- Receipt Cost -- % increase or decrease of On order cost versus the receipt cost coming from sales plan view.
  • LY On Order AUC – last year average unit cost price of LY On Order Units coming from the sales plan view.
  • LY On Order Cost -- last year receipt cost for receipt units for which orders have been submitted to external systems and for which order confirmations have been received
  • Note: relevant for planned values only in scenarios where planning is occurring further than 12 months out from the current date; actual receipts delivered will be actualized and displayed as LY Receipt Units
  • On Order Cost % +/- LY -- percentage increase or decrease of On Order Cost versus LY On Order Cost
  • System Stock Cost to COGS -- ratio expressing the relationship of BOP Cost to COGS generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Stock Cost to COGS -- ratio expressing the relationship of BOP Cost to COGS coming from the sales plan view; this value will match the system Stock Cost to COGS line unless a user has overridden the sales plan in the sales plan view or receipt plan in the receipt plan view
  • Stock Cost to COGS (Comparison) – ratio expressing the relationship of BOP cost to COGS coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Stock Cost to COGS – ratio expressing the relationship of Last year Stock Cost to Last Year COGS coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • System Stock Units to Sales Units -- ratio expressing the relationship of BOP Units to Sales Units generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Stock Units to Sales Units -- ratio expressing the relationship of BOP Units to Sales Units coming from the sales plan view; this value will match the system Stock Units to Sales Units line unless a user has overridden the sales plan in the sales plan view or receipt plan in the receipt plan view
  • Stock Units to Sales Units (Comparison) – ratio expressing the relationship of BOP Units to Sales Units coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Stock Units to Sales Units – ratio expressing the relationship of Last year Stock Units to Last Year Sales Units coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • Shrink Units – amount of inventory units lost through theft, damages, spoilage, etc. that are planned or actual; shrink units are subtracted from EOP Units.
  • Shrink Units % of Sales Units – percentage of total sales units incurred from planned or actual shrink units
  • LY Shrink Units – amount of inventory units lost last year through theft, damages, spoilage, etc. that are planned or actual; LY Shrink units are subtracted from LY EOP Units
  • LY Shrink Units % of Sales Units -- percentage of LY total sales units incurred from planned or actual LY shrink units
  • Shrink Cost -- amount of inventory at cost lost through theft, damages, spoilage, etc. that are planned or actual; Shrink Cost is subtracted from EOP Cost
  • Shrink Cost % of COGS -- percentage of total COGS incurred from planned or actual shrink at cost
  • LY Shrink Cost – amount of inventory at cost lost last year through theft, damages, spoilage, etc. that are planned or actual; LY Shrink cost is subtracted from LY EOP Cost
  • LY Shrink Cost % of COGS – percentage of LY total COGS incurred from planned or actual LY shrink at cost
  • Inventory Adjustment Units – planned/actual inventory adjustment units for the period; this value can add to or subtract inventory units from EOP units
  • LY Inventory Adjustment Units – last year planned/actual inventory adjustment units for the period; this value can add to or subtract inventory units from EOP units
  • Inventory Adjustment Cost -- planned/actual inventory adjustment at cost for the period; this value can add to or subtract inventory at cost from EOP Cost
  • LY Inventory Adjustment Cost -- last year planned/actual inventory adjustment at cost for the period; this value can add to or subtract inventory at cost from EOP Cost.
  • System EOP Units – End of Period units of inventory generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • EOP Units – End of Period units of inventory coming from the sales plan view; this value will match the system EOP Units line unless a user has overridden the sales plan in the sales plan view or receipt plan in the receipt plan view
  • EOP Units (Comparison) – End of Period units of inventory coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY EOP Units – last year End of Period units of inventory coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • EOP Units % +/- LY -- percentage increase or decrease of EOP Units versus LY EOP Units
  • System EOP Cost -- End of Period cost of inventory generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • EOP Cost -- End of Period cost of inventory coming from the sales plan view; this value will match the system EOP Cost line unless a user has overridden the sales plan in the sales plan view or receipt plan in the receipt plan view
  • Comparison EOP Cost -- End of Period Cost of inventory coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY EOP Cost – last year End of Period cost of inventory coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • EOP Cost % +/- LY -- percentage increase or decrease of EOP Cost versus LY EOP Cost
  • System Avg EOP Unit Cost – average cost of inventory per unit generated by smart start in target creation or from the work a user has done in the main planning view in line visualization for all the variants/placeholders in the selected product filter
  • Avg EOP Unit Cost – average cost of inventory per unit coming from the sales plan view; this value will match the System Avg EOP Unit cost line unless a user has overridden the sales plan in the sales plan view or receipt plan in the receipt plan view
  • Avg EOP Unit Cost (Comparison) – Avg EOP Unit Cost of inventory coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY Avg EOP Unit Cost -- last year Avg EOP unit cost of inventory coming from the sales plan view; this value could be a planned value or actualized value depending on the comparable time period last year
  • Avg EOP Unit Cost % +/- LY -- percentage increase or decrease of Avg EOP Unit Cost versus LY Avg EOP Unit Cost
  • VAT Sales – amount of sales revenue generated through value added tax
  • Comparison VAT Sales – VAT Sales coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • VAT % -- value added tax rate applied to the revenue generated from the items at a given level
  • VAT % (Comparison) -- VAT % coming from the selected comparison snapshot; this value includes any user overrides present at the time the snapshot was taken
  • LY VAT Sales -- amount of sales revenue generated last year through last year value added tax
  • LY VAT % -- last year’s value added tax rate applied to the revenue generated from an organization for same period last year
