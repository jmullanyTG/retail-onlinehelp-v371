Create a Role

Choosing to create a role will bring up the “role information” window.

Filling in the role name (required) and role description (optional) are done within the “role details” tab.

Role memberships may be added if desired through the “memberships” tab.

For the role to be an effective tool, the proper rights must be assigned. This is managed through the “rights” tab.

Expanding the header level rights groups will allow administrators to select certain rights within each module to include or exclude from the standard access rights for the role.
