Allow Editing of Promotion after Sale Start Date

Provide access to edit a Promotion after the Sale Start Date has occurred. You will be granted the User Rights configured for the Promotion’s Workflow Status State in conjunction with any other global User Rights not controlled by Workflow. The “Allow Editing of Promotion” User Right must also be granted.

Note: This User Right should only be granted to trusted Super Users
