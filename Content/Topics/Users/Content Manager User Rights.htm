Content Manager User Rights

Content Manager User Rights provides the ability to grant permissions to update text and images for products and product groups at the Master, Promotion and Event levels. If Content is updated for a product at any level, it can be pushed to other levels for the same product or where the product is included in a grouping of products.

Below is a list of global permissions and the purpose of each. From Maintain, click Users to configure permissions.

User Right

Description

Access To Content

Provides access to the Content Manager solution.

Allow Updating of Copy from Event to Events

Provides access to save Copy Text from one Event instance to the Copy Text for one or more other Events with Promotions for the same product.

Allow Updating of Copy from Event to Master

Provides access to save Copy Text from an Event to the Master Copy Text for the same product.

Allow Updating of Copy from Event to Promotions

Provides access to save Copy Text from an Event instance to the Copy Text for one or more Promotion’s for the same product.

Allow Updating of Copy from Master to Events

Provides access to save Copy Text from the product Master Copy Text to one or more Events with Promotions for the same product.

Allow Updating of Copy from Master to Promotions

Provides access to save Copy Text from the product Master Copy Text to the Copy Text for one or more Promotions for the same product.

Allow Updating of Copy from Promotion to Events

Provides access to save Copy Text from a Promotion to the Copy Text for one or more other Events with Promotions for the same product.

Allow Updating of Copy from Promotion to Master

Provides access to save Copy Text from a Promotion to the Master Copy Text for the product.

Allow Updating of Images from Event to Events

Provides access to save Images from one Event instance to the Images for one or more other Events with Promotions for the same product.

Allow Updating of Images from Event to Master

Provides access to save Images from an Event to the Master Images for the same product.

Allow Updating of Images from Event to Promotions

Provides access to save Images from an Event instance to the Images for one or more Promotion’s for the same product.

Allow Updating of Images from Master to Events

Provides access to save Images from the product Master Images to one or more Events with Promotions for the same product.

Allow Updating of Images from Master to Promotions

Provides access to save Images from the product Master Images to the Images for one or more Promotions for the same product.

Allow Updating of Images from Promotion to Events

Provides access to save Images from a Promotion to the Images for one or more other Events with Promotions for the same product.

Allow Updating of Images from Promotion to Master

Provides access to save Images from a Promotion to the Master Images for the product.

Edit Content

Provides access to edit product Content in Content Manager.
