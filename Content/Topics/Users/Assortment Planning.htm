Assortment Planning

Right Name

Location Controlled

  

Access To Assortment Planning

Plan > Assortment Planning

 

Add Plan

Plan--> Assortment Planning-->Click Manage and then Click Add to create a plan

 

Edit Plan (Owner)

Plan--> Assortment Planning-->Click Manage then select a owned plan and click Edit

 

Edit Plan (All)

Plan--> Assortment Planning-->Click Manage then select a plan created by another user and click Edit

 

Delete Plan (Owner)

Plan--> Assortment Planning-->Click Manage then select a owned plan and click Delete

 

Delete Plan (All)

Plan--> Assortment Planning-->Click Manage then select a plan created by another user and click Delete

 

Manage Plan User Access (Owner)

Plan--> Assortment Planning-->Click Manage then select a owned plan and click User Access

 

Manage Plan User Access (All)

Plan--> Assortment Planning-->Click Manage then select a plan created by another user and click User Access

 

Edit Plan Status (Owner)

Plan--> Assortment Planning-->Click Manage then select a owned plan and click Status

 

Edit Plan Status (All)

Plan--> Assortment Planning-->Click Manage then select a plan created by another user and click Status

 

Add Plan Snapshot (Owner)

Plan--> Assortment Planning-->Click Manage then select a owned plan and click Snapshot

 

Add Plan Snapshot (All)

Plan--> Assortment Planning-->Click Manage then select a plan created by another user and click Snapshot

 

Delete Plan Snapshot (Owner)

Plan--> Assortment Planning-->Click Manage then click on snapshot tab. Select a owned snapshot and then click delete

 

Delete Plan Snapshot (All)

Plan--> Assortment Planning-->Click Manage then click on snapshot tab. Select a snapshot created by another user and then click delete

 

Revert Plan from Snapshot (Owner)

Plan--> Assortment Planning-->Click Manage then click on snapshot tab. Select a owned snapshot and then click Revert

 

Revert Plan from Snapshot (All)

Plan--> Assortment Planning-->Click Manage then click on snapshot tab. Select a snapshot created by another user and then click Revert

 

Initialize Plan (Owner)

Plan-->Financial Planning-->Click Manage-->Select a owned Plan In created status and then click Initialize

 

Initialize Plan (All)

Plan-->Financial Planning-->Click Manage-->Select a Plan In created status that created by another user and then click Initialize

 

Re-Initialize Plan (Owner)

Plan-->Financial Planning-->Click Manage-->Select a owned Plan In created status and then click Re-Initialize

 

Re-Initialize Plan (All)

Plan-->Financial Planning-->Click Manage-->Select a Plan In created status that created by another user and then click Re-Initialize
