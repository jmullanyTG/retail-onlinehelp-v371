Allow Overriding Slot Set Status

Provides access to override the Slot Set Status to grant User Rights from a previous Status.

Note: This permission may be configured to be enabled or disabled during specific Event Workflow Status’ States

Note: This User Right should only be granted to trusted Super Users

Note: The User Right “Allow Editing of Slot Set Header” must also be granted

Note: This functionality and its placement on the screen is the same whether accessed from the List or Layout
