Events Management User Rights

Events Management User Rights can be configured by clicking Users from the Maintain tab, selecting a user or role and clicking Assign Rights. The > and < buttons can be used to add or remove rights for the user or role. User Rights can be configured for a Role, by clicking Users from the Maintain tab, selecting a user or role and clicking Modify.

Some User Rights may be incorporated into Event Workflow Status States, while others are not affected by Event Status. User Rights need to be granted at a global level, but can be temporarily taken away by Event Workflow Status State using the User Rights Blacklist. For more information, see “Restrict User/Role Permissions” in Workflow State Manager Standard Documentation to understand blacklisting User Rights during specific Event Status States.
