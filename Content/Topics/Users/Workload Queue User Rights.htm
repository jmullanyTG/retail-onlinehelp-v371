Workload Queue User Rights

Permissions can be configured by clicking Users from the Maintain tab and using the > and < buttons to add or remove rights for a user or role.

User Right

Description

Allow Access to Workload Queue Module

This permission will display the Workload Queue icon under the Maintain tab.

Allow Job Reprioritization and Removal

This permission will enable the ability to change a Batch’s prioritization and removal from the ‘active’ list to be processed.

Allow Workload Queue Configuration

This permission enables the Configuration button on the Workload Queue toolbar, which allows users to configure threshold overrides and optimizations.
