Price Manager

There are two sets of permissions within Price Planning; One set not affected by a Price Action Status (i.e. global), the other affected by status. Below is a list of global permissions and each’s purpose. Permissions can be configured using the ‘Maintain > Users’ menu action.

Right Description
Access to Price Planning

Provides the ability to view the Price Planning module, via ‘Plan > Price Manager’ menu action, and all Pricing Plans and Actions configured

Access to Price Planning Configuration

Provides the ability to view the Price Planning configuration screen: Rounding Rules, Variables, Calculations, Validations, Strategies and Status.
Create/Edit a Pricing Plan

Provides the ability to create and edit (i.e. Start Date, add/remove products and sites) associated to a Pricing Plan
Delete a Pricing Plan

Provides the ability to delete Pricing Plans. Pricing Plans cannot be deleted if there are associated Price Actions.
Create a Pricing Action

Provides the ability to create Pricing Actions which set pricing for site products.
Emergency Pricing Action

Provides the ability to modify the ‘Pricing Plan Start Date’ and ‘Pricing Action Start Date’ to occur tomorrow. By default, the system will restrict the ‘Plan Start Date’, and associated ‘Price Action Start Date(s)’ by the number of days, from today, configured in ‘Maintain > Policies > Module:Price Manager > PriceManager.PlanStartDateNumberOfDaysOut’.
