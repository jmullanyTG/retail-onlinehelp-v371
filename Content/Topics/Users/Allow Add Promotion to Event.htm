Allow Add Promotion to Event

Provides access to add an existing Promotion that has effective dates that overlap the Event’s Sale Dates. If this User Right is not granted, you may still be able to create a new Promotion from the Event if you have the “Allow Creation of Promotion from Event” User Right.

Note: This permission may be configured to be enabled or disabled during specific Event Workflow Status’ States
