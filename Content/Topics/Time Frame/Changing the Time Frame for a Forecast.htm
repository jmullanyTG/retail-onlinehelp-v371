Changing the Time Frame for a Forecast

If you want to change the time frame used to generate demand forecasts for an item (for example, "April 1 - 31" instead of "the next 90 days"), simply apply the desired forecast time frame to the item. The next time that a forecast is generated for that item, it will be for that time frame.

Note: An item's forecast time frame is currently visible only in the Forecast Phase Time Frame Maintenance view.

Setup

The following objects are used when changing an item's forecast time frame.

  • An item filter containing the desired item(s).
  • The time frames you want to use.

Instructions

  1. From the Plan tab, click View to open the View screen.

  2. In the Navigation Menu pane, find the Forecast Phase Time Maintenance view, then double-click on it to open it.

  3. In the Forecast Phase Time Frame Maintenance document:

    1. Select a filter containing the relevant items.

    2. In the items' Phase Time Frame field, select the time frame you want JustEnough to use when generating forecasts for the item.

      Note: You can change the value for multiple records at a time.

  4. Click Save in the grid toolbar to apply the changes. The next time a forecast is generated for the changed items, it will use the time frames you just applied.
