Time Frame Maintenance Window

The Time Frame Maintenance window is where you can create, edit, and delete time frames.

It contains a toolbar and 3 areas:

  • A grid listing all available time frames (upper-left).
  • A time frame definition pane listing the effective time ranges for the time frame selected in the grid (upper-right).
  • A calendar area showing the effective date ranges for the time frame selected in the grid (bottom).

Getting here

To open the Time Frame Maintenance window:

  • From the Maintenance tab, click Time Frame; or
  • From a Time Frames selection dialog box, click (Maintain Time Frames).

Toolbar

The Time Frame toolbar contains the following actions.

Action Description
Add Creates a new time frame.
Edit Edits the selected time frame.
Delete Deletes the selected time frame.
Refresh Refreshes the Time Frame Maintenance window.

Grid details

The read-only grid lists all of your available time frames. (Time frames for which you do not have access do not appear in the grid.)

Field Description
Time Frame ID The unique ID that JustEnough assigned to the time frame.
Name The name given to the time frame.
Description An optional description of the time frame.
Dynamic Rule

When selected, the time frame covers "rolling" time ranges, such as "last 90 days" or "next 30 days".

If not selected, the time frame covers static dates only, such as "March 1, 2018 - March 31, 2018".
Allow Multiple Definitions

When selected, the time frame can include more than one defined time range. Each time range appears in its own row.

For example, a time frame covering the week after Christmas for the years 2015-2017 would include the following static time ranges: "December 26-31, 2015", "December 26-31, 2016", "December 26-31, 2017".

User Access

Which JustEnough users have access to this time frame.

  • Private: Only you have access.
  • Public: All users have access.
  • Shared: Only selected users have access.

Time Frame Definition pane

This pane displays the definitions of the selected time frame. The fields shown depend on the time frame definition type (static, dynamic, or schedule) that was selected for the time frame when it was created.

Calendar area

The calendar area contains two calendars that show the effective time periods covered by the time frame selected in the grid. The calendar on the left shows time periods by week. The calendar on the right shows time periods by month. Both calendars track each other and, if the selected time frame allows multiple definitions, the calendars use the same color-coding for each included time range (see screenshot below).
