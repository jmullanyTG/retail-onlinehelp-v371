Example - Creating a Dynamic Time Frame

In this example, we'll create a time frame that covers the last 90 days before an operation is run.

  1. In the Time Frame Maintenance window, click Add.

    '

  2. In the Time Frame Editor, enter the following values:

    • Name: Last 90 Days
    • Security: Private

    • Time Frame Definition Type: Dynamic

  3. In the Time Frame Definition toolbar, click Add to add a time frame definition row to the editor.This lets us define the time period for the time frame.

  4. Click in the Offset Days cell, and enter the value -90. (Make sure to enter the "-" sign.) This tells JustEnough to go back 90 days to start the time period.

  5. Click in the Number of Rolling Days cell and enter the value 90.This tells JustEnough to include 90 days from the offset day in the time range.

  6. Click OK.

  7. The new time frame appears in the Time Frame Maintenance grid. Select it to view its details.

    Note the following details:

    • In the grid, the Dynamic Rule box is checked, indicating that the time frame is dynamic.
    • In the grid, the Allow Multiple Definitions box is not checked, indicating that only one time period is defined for the time frame.
    • In the upper-right corner, the Offset Days and Number of Rolling Days values we entered for the time frame are displayed, showing us the values that JustEnough uses to calculate the effective time period for the time frame.

    • In the Calendar area, you can see the actual days covered by the time range defined for the time frame (displayed in red, in this example). The time range, being dynamic, is based on "today" being the day from which the start date of the time frame is offset, so the start date will be 90 days before today.

      • In the weekly calendar (left side), the Time Range goes back 90 days from the current day (February 13, in this example). Its effective start and end dates are displayed in the calendar for your convenience.
      • In the monthly calendar (right side), the dates covered by the time range are also shaded red.
      • You can scroll backward and forward in either calendar and the other one tracks along for convenience.
  8. You can now use the time frame in other JustEnough modules. When executed, the time frame will include the 90 days before the current date. For example, if an operation using this time frame is executed on July 1, its time frame will cover the 90 days prior to July 1. If it is executed again on December 1, it will cover the 90 days prior to December 1.
