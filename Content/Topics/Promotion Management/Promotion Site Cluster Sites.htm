Promotion Site Cluster Sites

Sites can be added or removed from Site Clusters and new Site Clusters can be created or deleted from the Promotion.

To create a new Site Cluster

  1. Click on Price Cluster New.
  2. Enter the Site Cluster Name, select a Promotion Strategy, and select the Site Cluster currency.
  3. Click Ok.
  4. Add Sites to the new Site Cluster.

To delete a Site Cluster

  1. Select the Promotion Site Cluster.
  2. Click on the Price Cluster Delete.
  3. Click Yes.

Note: Any Sites assigned to the deleted Site Cluster will be removed from the Promotion.
