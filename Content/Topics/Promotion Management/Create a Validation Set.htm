Create a Validation Set

Within the Promotion Validations tab, click New above the Promotion Validation Set Configuration grid to create a new Promotion Validation Set. Enter the required fields and click Save.

Field

Description

Validation Set Name (Required)

Enter a descriptive label to summarize the purpose of the Validation Set. This will be used to identify the Validation Set when configuring Promotion Strategies.

Validation Set Description (Optional)

Enter a summary of the Validation Set.

Clone (Optional)

Leave this checkbox unselected..
