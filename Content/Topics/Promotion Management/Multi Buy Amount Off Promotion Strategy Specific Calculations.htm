Amount Off - Low – Promo Aggr

Set the Promotion Aggregate level LowestDollarDiscount field to the minimum DollarDiscount of the products (Item, Variant or groups) in the Promotion. For example, if the Promotion consists of Items and groups, the Promotion Aggregate value is set by comparing the minimum DollarDiscount at Item level for the Items, the minimum DollarDiscount at the Group level for all groups, then setting to the Promotion Aggregate LowestDollarDiscount to the minimum value of all.

  • Calculation Level – Promotion Aggregate Level
  • Group Type – Buy Get Group
  • Calculation Field – LowestDollarDiscount
  • Aggregation Type – MIN
  • Formula – ISNULL(DollarDiscount,0)

Amount Off - High - Promo Aggr

  • Calculation Level – Promotion Aggregate Level

Multi Amount Off - Promo Low - Item

Set the LowestPromotionPrice field at the Item level to the lowest retail price minus the proportioned amount off. Overrides are not configured for multi buy Promotion Strategies because it is not possible to accurately calculate individual product Promotion Price with overrides.

  • Calculation Level – Item Level
  • Group Type – Buy Get Group
  • Calculation Field – LowestPromotionPrice
  • Formula – LowestPromotionPrice = (CASE WHEN  ISNULL(MultiBuyQuantity,0) <> 0 THEN ( ISNULL(LowestRetailPrice,0) - (ISNULL(DollarDiscount,0)/ ISNULL(MultiBuyQuantity,0)) ) ELSE 0 END)

Multi Amount Off - Promo High - Item

Set the HighestPromotionPrice field at the Item level to the highest retail price minus the proportioned amount off. Overrides are not configured for multi buy Promotion Strategies because it is not possible to accurately calculate individual product Promotion Price with overrides.

  • Calculation Level – Item Level
  • Group Type – Buy Get Group
  • Calculation Field – HighestPromotionPrice
  • Formula – HighestPromotionPrice = (CASE WHEN  ISNULL(MultiBuyQuantity,0) <> 0 THEN ( ISNULL(HighestRetailPrice,0) - (ISNULL(DollarDiscount,0)/ ISNULL(MultiBuyQuantity,0)) ) ELSE 0 END)
