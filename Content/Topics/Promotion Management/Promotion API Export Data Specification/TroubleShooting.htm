Troubleshooting

Executing export.Promotion_Extract_Wrapper with the debug flag set to 1 will output various chunks of data. Of particular interest is the XML payload data. This will show exactly what is being sent to the API process.

 

Example:

EXEC export.Promotion_Extract_Wrapper

@iUserid =1

,@iExportID  =  --<configurable Export ID>

,@iExportBatchID = --<ExportBatchID to pull data from>

,@bDebug  = 1
