Promoted price or promoted discount type Promotion Strategies

For Promoted Price and Promoted Discount type Promotion Strategies there are three ways to add products. The three methods of adding products are not mutually exclusive and all three methods can be used within a Promotion.

  • Add Item / Add Variant – Click this button if you know the product code (e.g. SKU).

    1. Enter individual product code(s) (for Variant or Item Master, depending on the global policy) into the textbox or paste product codes from the clipboard by clicking the Add Items From Clipboard / Add Variants From Clipboard.
    2. Once all the products have been added to the Promotion, they will be displayed in a group called Item Group below the Promotion Group.

  • Select to Add – Use a filter to select products.

    1. A Filter selection box will display. Select a filter from which you would like to select items from the dropdown list.
    2. If an appropriate Filter does not exist click on the black hammer icon to create a new Filter that will better match the list of items to select from.
    3. Once the appropriate Filter is selected click Ok. Selecting the checkbox ‘Select only some items from the current filter’ will allow individual product selection from the selected filter.
    4. Once all the products have been added to the Promotion, they will be displayed in a group called Item Group below the Promotion Group.

  • Add Item Group / Add Variant Group – Click to add a product group.

    1. Select the checkbox for each group to be added to the Promotion, then click OK.

      • If the group you wish to add is not available, first ensure the Filter selected in the pop up is for “All Variants/Item Masters”. If still unavailable, navigate to Content Manager to create the group you require for the Promotion.
      • These groups are dynamic, meaning the system will update the associated products included in the group dependent on the filter criteria used to create the group. This update will only occur as long as the Promotion is in an unlocked state.
      • When adding a group to a Promotion, the system will create a copy of the original group, including the filter criteria used to define the Group, so that changes can be made at the Promotion level without affecting the originating product group.

    1. To modify the product group’s filter highlight the group and click Edit which will display the Edit Group window.

      • Click on the blue hammer.
      • Thereafter the Maintain Filters screen will display. Make the appropriate changes to the filter following the instructions in the Filters section.
    1. Once all the product groups have been added to the Promotion, they will be displayed below the Promotion Group.
