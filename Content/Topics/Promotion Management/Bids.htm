Bids

The Promotion Bids tab allows you to make a bid for an Event. This screen lists all Events that the Promotion could bid for. Only Events with Sale Start and End Dates that overlap with the Promotion Start and End Dates will be displayed.

The Bid Status column indicates if the Promotion already has a Bid has for the Event. The possible Statuses are:

  • No Bid – The Promotion has not made a Bid for the Event.
  • Submitted – A Bid has been submitted.
  • Re-Submitted – A Bid has been resubmitted for the Event.

To place a Bid for an Event, select the Event and click on Edit to display the ‘Edit Bid Details’ window. Enter the required field and click on Ok:

Action

Description

Preferred Bid Event Spaces (Optional)

Use the ‘Select Preferred Spaces’ window to move Spaces to the ‘Included’ pane. Click Ok when finished.

Bid Action (Required)

Select an action from the dropdown list:

  • Submitted – Indicates a new Bid.
  • Re-Submitted – Indicates a previously placed Bid has been updated.
  • Cancelled – Indicates that the Bid is no longer active.

Financial Forecast  (Optional)

Enter the projected financial sales for this Bid.

Unit Forecast  (Optional)

Enter the projected units sold for this Bid.

Vendor Supported (Required)

If the Vendor is providing support for the Bid, select ‘Yes’, otherwise ‘No’.

Comments (Optional)

Enter any supporting information to help ensure the Bid is successful.
