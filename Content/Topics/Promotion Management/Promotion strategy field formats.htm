Promotion strategy field formats

This grid determines how numeric pricing fields will be displayed if they are configured to display in the display grid on the left of a New Promotion Wizard and Promotion.

  

 

Field

Description

Price Field (Read Only)

Names of all available pricing fields.

Field Friendly Name (Optional)

The user-friendly field name displayed in the grid.

Table Name (Read Only)

The database table where this value is stored.

Format String (Optional)

The numeric format to be used to present the field value in the grid.

Is Custom (Read Only)

Indicates if the field is customer specific.

Is Read Only (Read Only)

Indicates if the field is a read only field.
