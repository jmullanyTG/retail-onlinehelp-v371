Create rounding criteria – price ending

With the Rounding Rule selected, click New above the Rounding Criteria grid to add criteria to the Rounding Rule. Enter the required fields and click Save above the Rounding Criteria grid.

Field

Description

Currency (Required)

The dropdown is populated by the available currencies specified by each Site. Select a currency for the criteria configuration which will be applied only to the applicable Site(s) when the system is calculating Price Actions.

Price Ending (Required)

The right most decimal value the system will search for in price fields.

Position (Required)

Select the numeric position the system will modify based on the criteria.

Direction (Optional)

Determines whether the system will force the calculated value to be rounded ‘up’, ‘down’ or to the ‘closest’ based on the calculated mid-point of the Price Ending configured value.

  • Closest - The system will evaluate the configured ‘Position’ to determine whether to round the calculated value up or down to the desired price ending.
  • Down - The system will evaluate the configured ‘Position’ and calculated ‘Mid-Point’ and round the calculated value down to the desired price ending.
  • Up - The system will evaluate the configured ‘Position’ and calculated ‘Mid-Point’ and round the calculated value up to the desired price ending.

Active (Optional)

Leave selected to indicate that this Rounding Criteria is active within the Rounding Rule.
