Change Promotion Status

Promotion Workflows are setup in Workflow Manager. The Workflow will determine the number of Statuses, what transitions will be available at each Status, and what access the users will have at each Status. If you have been given permission to change the Promotion Status for the current Status of the Promotion, the Change Status button will be available for you.

By clicking on Change Status, a window will be displayed that allows you to change the current Status to the one of the available Statuses for this transition.

 

Field

Description

Status

A dropdown list of all available Statuses that you may select from.

Message

An optional area where you can enter a message for the next user to review. If a Message has been entered, a green message icon will be displayed beside the Status.
