Strategies

Within the Promotion Strategy screen, click New to create a new Promotion Strategy. Enter the required fields in the Promotion Strategies Detail grid and click Save.

Field

Description

Strategy Name (Required)

Enter a name for Promotion Strategy that will be used to select a Promotion Strategy when creating a Promotion.

Strategy Type ID (Required)

Select the Strategy Type from the dropdown list that will determine how the Promotion Strategy will be displayed.

  • Promoted Price - A promotional price.
  • Promoted Discount – An amount off or percent off Promotion.
  • Buy Get – A buy X get X or buy X get Y Promotion.
  • Spend Get – A spend X get points, dollars, or percent off.
  • Rebate – A regular retail or promotional price with a rebate.

Method (Required)

The dropdown is populated by the available methods (i.e. advanced processes applied at the database level) configured for use.  Select a method the system will apply for the Promotion Strategy. This will be run before anything else on the Promotion.

Calculation Set (Required)

Select a Calculation Set from the dropdown list. Populated with Calculation Sets defined in the Promotion Validation Configuration grid. Calculations will be performed after methods.

Validation Set (Optional)

Select a Validation Set from the dropdown list. Populated with Validation Sets defined in the Promotion Validation Configuration grid. Validations will be performed after Calculations.

Price Output Set (Required)

Select a Promotion Pricing Detail from the dropdown list. Populated with Pricing Outputs defined in the Promotion Price Output Set Configuration grid.

Default Price Method (Optional)

The dropdown is populated by the available methods (i.e. advanced processes applied at the database level) configured for use.  Select a method the system will apply for the Promotion Strategy. This will be run second, after the Method is run on the Promotion.

Analysis View Collection Set (Optional)

The dropdown is populated by available pivot template to display analysis information. Select a display to use in the Promotion – Analysis tab.

Price View Collection Set (Required)

The dropdown is populated by available pivot template to display pricing information. Select a display to use in the Promotion – Price Details tab.

Is Group Synced (Optional)

Must be selected for all BOGO AA type Promotion Strategies.

Active Flag (Optional)

Selected by default to indicate that the Promotion Strategy will be available when working with Promotions.
