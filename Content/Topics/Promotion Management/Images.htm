Images

By clicking the Images tab in the Content Details window, Images associated with the Promotion Content Version can be viewed, new Image associations can be made or requested as well as removed from future use.

The following buttons are available if needed:

Action

Description

Save

It is beneficial to save changes if a several Images have been newly associated to the Promotion. If the Promotion has been placed on a future Event or the product in the Promotion has Master Content, the Content Update window will display to provide the ability to update Images for the other instances as well. For more information, see Content Update window.

Add Existing

Opens the Repository Explorer window to allow selection of an existing Image.

Add

Opens the local computer’s Explorer window to allow you to navigate to the appropriate Image on your computer or network.

Remove

Selecting an Image(s) and clicking this button will remove the selected Image(s).

Remove All

This will remove all Images for the Content Version.

View

Clicking View or double-clicking the image will display a larger preview of the Image in a window.

Attributes

This will display the Image Attributes window for the selected Image. If more than one Image is selected, Attributes will display for the last selected Image in the grid.

Note: Image Attributes are not available for New Image Requests.

Image Request: New

If the required Image doesn’t exist, it can be requested by clicking this button. A window will a form that can be completed to initiate a request. Click Ok to save the request.

Image Request: Edit

The existing New Image Request will open in a window and can be modified as needed. Click Ok to save the updates.

 

Note: The three methods of adding images are not mutually exclusive and all three methods can be used within a Promotion.

 

Associate a view type

Each new Image added should be associated with a View Type so it will display properly in an Item Display Format. This association can be made directly in the Images grid by selecting the View Type dropdown list and choosing a value for each Image. Each View Type can only be associated to one Image.
