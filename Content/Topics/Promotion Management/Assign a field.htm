Assign a field

Within the Promotion Pricing Details screen, click New above the Promotion Price Output Set Fields Configuration grid to create a new Field Configuration, enter the required fields and click Save.

Field

Description

Price Field/Table (Required)

Select a Price Field from the dropdown list populated with Price Output table fields.

Write Enabled (Optional)

Select if this field should be editable in the Promotion.

Write Enabled Levels (Required/Optional)

If Write Enabled is selected, the Write Enabled Levels should be specified. Clicking on … will open the Write Enabled Levels window.

Write Enabled Levels window:

  • Using the window, select the level(s) that will be written to from the Excluded pane and click on > to move over to the Included pane.
  • Use < to move back any selected fields from the Included pane to the Excluded pane.
  • Clicking Ok will apply the Included Write Enabled Levels to the field.
  • Click Cancel to discard any modifications.

Note: The Promotion Manager system defines Write Enabled Levels as follows:

  • Promotion Level – Price field will be written at the Site Cluster and Promotion Group level.
  • Promotion Aggregate Level – Price field will be written at the Site Cluster and Promotion Group level.
  • Group Level – Price field will be written at the Site Cluster and Product Group level.
  • Item – Price field will be written at the Site Cluster and Item Master level.
  • Variant – Price field will be written at the Site Cluster and Variant level.

Price Field (Read Only)

Defaults to the Price Field Name after changes are applied.
