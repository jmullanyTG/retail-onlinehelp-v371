BOGO AB Percent Off Promotion Strategy Calculation Set

Fill down the keyed BOGO percent off from the Promotion level to all levels. At the Item level, set the LowestPercentDiscount and HighestPercentDiscount equal to the keyed BOGO percent off. Roll up these percent off fields from the Item level to the Promotion Aggregate level.

Calculate and set the LowestPromotionPrice and HighestPromotionPrice for the Get Group at the Item level using the Bogo Percent Discount or override value if available. Set the LowestPromotionPrice and HighestPromotionPrice for the Buy Group to the respective EDLP at the Item level. Roll up LowestPromotionPrice and HighestPromotionPrice from the Item level to the Promotion Aggregate level.
