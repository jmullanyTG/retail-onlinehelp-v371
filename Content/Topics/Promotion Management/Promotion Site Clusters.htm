Promotion Site Clusters

The Site Clusters area displays the Promotion Site Clusters, each with their associated Promotion Strategy and Sites.

If a Site Cluster that contains Sites with more than one currency is used in a Promotion, the Site Cluster will be split to display an instance of the Site Cluster for each currency. For example:

  • Site Cluster ‘A’ consists of Site Alpha (USD), Site Beta (USD), Site Gamma (CAD), and Site Delta (USD).
  • Site Cluster ‘A’ is added to a Site Cluster Set which is used in a Promotion.
  • The top area of the Promotion will display the following rows for Site Cluster A in the list of Promotion Site Clusters: ‘A (USD)’ and ‘A (CAD)’, each with their respective Sites that match the currency appended to the Site Cluster name.

The Site Clusters area allows you to modify the Promotion Strategy at the Site Cluster level.

  • Select the Site Cluster that will be promoted differently.
  • Modify the Promotion Strategy as needed.
  • The Promotion will be saved automatically.
