Item Manager Fields

Upon entering the item manager, users will be prompted to select a filter. This can be a static filter for a select set of items, or one of the many dynamic filters which are capable or returning items which match the specified criteria.

After selecting a filter, users will be prompted to choose the fields to return in the item manager view. These are organized by category and may be included or excluded as needed. Users may also use the search bar near the top of the window to search for applicable fields.

Once the view is created, fields may be added or removed by using the “choose” button.
