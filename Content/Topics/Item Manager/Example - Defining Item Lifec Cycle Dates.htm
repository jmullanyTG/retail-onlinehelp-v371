Example - Defining Item Life Cycle Dates

In this example, we will set a retail channel phase-in date for all style/color variants of the new item, "High Collar Angelina Jacket".

Workflow

Click to open the Managing Product Life Cycles workflow.

Managing Item Life Cycles

  

Setup

The following objects are used in this example.

  • A filter that includes the new item and the old item.
    • Filter used in this example: New Items (Creation <=14 Days).
  • An Item Manager view that includes Site Group 3 Description and phase in/phase out fields.

    • View used in this example: Life Cycle Management.

Instructions

  1. From the Manage tab, click Item Manager.

  2. In the Items dialog box, select the filter that contains the items you want to manage.

    • For this example, select the New Items (Creation <=14 Days) filter.

    Note: If the Items dialog box does not automatically appear, click Inventory Item Management on the toolbar.

  3. In the Choose Fields dialog box, select the Life Cycle Management view.

  4. Click OK.

  5. In the Item Manager screen, filter product and channel columns to identify the new item (in all its variants) and the desired channel (retail).

    1. In the Product Description column filter, enter: High Collar Angelina Jacket.
    2. In the Site Group 3 Description column filter, enter Retail.

  6. To set the same Phase In Date and/or Phase Out Date for all style/color variants of the new item:

    1. Right-click in an item row and choose Select All.
    2. Right-click in an item's Phase In Date field and choose Multiple Changes.

    3. Select a date roughly 30 days in the future and click Save.

    4. Right-click in an item's Phase Out Date field and choose Multiple Changes.
    5. Select the phase out date "12/31/2999" and click Save.

    6. The rows in the Item Manager grid should reflect the new phase-in and phase-out dates.

  7. Click Save on the toolbar to save the new phase-in and phase-out dates.
