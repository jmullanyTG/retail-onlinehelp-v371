Defining Item Life Cycle Dates

You can define phase-in and phase-out dates for each of your items. These dates should reflect the dates an item will first and last appear on shelves in your locations. JustEnough uses these dates to optimize demand forecasts, purchase orders, and other related features for the item.

Setup

The following object should be set up before defining an item's life cycle dates:

  • A filter that includes the items you want to manage.

Procedure

  1. From the Manage tab, click Item Manager.

  2. In the Items dialog box, select the filter that contains the items you want to manage.

    Note: If the Items dialog box does not automatically appear, click Inventory Item Management on the toolbar.

  3. In the Choose Fields dialog box, select or create a view that includes the following fields:

    • Inventory Planning > Phase In Date
    • Inventory Planning > Phase Out Date

    • Fields you can use to identify the items you want to manage. For example, Demand Planning > Site Group 3 Description.

  4. Click OK.
  5. In the Item Manager screen, filter columns to locate the item(s) and/or channels in need of phase-in and/or phase-out dates.

  6. To set dates for a single item, click in its Phase In Date and Phase Out Date fields and select the desired dates.

  7. To set the same Phase In Date and/or Phase Out Date for multiple items:

    1. Filter columns until only items with the same desired dates appear in the grid.
    2. Right-click in an item row and choose Select All.
    3. Right-click in an item's Phase In Date field and choose Multiple Changes.

    4. Select the desired phase-in date for the selected items.

    5. Right-click in an item's Phase Out Date field and choose Multiple Changes.
    6. Select the desired phase-out date for the selected items.
  8. When all phase-in and phase-out dates have been entered, click Save on the toolbar to save the dates to the items.
