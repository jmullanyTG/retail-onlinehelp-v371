Example - Adding Schedules to Items

Schedules help JustEnough determine when orders need to be placed so that inventory can get to your sites in time to meet demand. You can define default schedules for sites in Site Manager, but when you want to override these schedules for specific item-site combinations, you can do so in Item Manager.

You can assign multiple types of schedules from within Item Manager. In this example, we will assign a receiving schedule to an item at a specific store. Once saved, future orders placed for this item at this store will use this receiving schedule instead the site's default receiving schedule.

Instructions

  1. From the Manage tab, click Item Manager.

  2. In the Items dialog box, select the New Items (<=14 days) filter.

  3. Choose the fields in the Inventory Schedules group to add schedule fields to the Inventory Manager document.

  4. In the Inventory Manager document, find the item you want to modify and select it in the grid.

  5. Click in its Receive Schedule field and select the schedule you want to use for this item.

  6. In the Inventory Manager document toolbar, click Save to save the schedule.
