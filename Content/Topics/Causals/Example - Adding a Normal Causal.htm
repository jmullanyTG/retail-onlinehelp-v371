Example - Adding a Normal Causal

In this example, we will create a causal that explains several spikes in sales in the e-commerce channel for the item "143 (All Season Lined Parka_Dark Navy - L). We know these spikes coincided with a recurring promotion, and we want JustEnough to identify the spikes in sales as being due to the promotion rather than part of the item's "normal" demand patterns. Because we are planning to run this promotion in the future, we will create a normal causal for the promotion so that JustEnough will account for the expected boost in sales when generating a demand forecast for the item.

This example includes the following tasks:

  • Review the item forecast - In this step, we look at the original item forecast and see that no causals have been applied to the spikes in sales that resulted from the parka promotions. As a result, the demand forecast for the item assumes the increase in sales is "organic" and not related to any promotions.
  • Create a causal - In this step, we create a causal for the parka promotion and apply it to the past and future dates of the promotion.
  • Review the revised item forecast - In this step, we review the updated forecast to confirm that the causals were applied to the correct periods and that the demand forecast reflects the changes as expected.

Instructions

1. Review the item forecast.
  1. From the Manage tab, click Forecasts.
  2. If the Items - Level View dialog box does not automatically appear, click Level View in the Forecast toolbar to open it.

  3. In the Items - Level View dialog box, select the (All Items) filter.

  4. In the Forecasts screen select the following item in the Item list:

    • Item Master Code; 143
    • Item Master: All Season Lined Parka_Dark Navy-L
    • Channel Code: E-Com

  5. Review the forecast. Note in the screenshot below that there are sales spikes in the following weeks: 2016 Week 30, 2017 Week 18, and 2017 Week 34 (dates in your example may vary). We know these spikes coincide with the online parka promotion. There are also forecasted spikes in 2017 Week 27 and 2017 Week 43 that have no matching spikes in sales. We know these discrepancies are due to the fact that the online parka promotion did not run during those periods. Finally, we know that we are running the same online parka promotion again in 2018 Weeks 8 and 31 but no corresponding spikes in demand are shown in the forecast. To fix all these issues, we must create a causal that resolves the past discrepancies and enables JustEnough to accurately forecast demand for future online parka promotions.

 2. Create a causal.
  1. From the Manage tab, click Causals.
  2. In the Causals toolbar, click New to create a new causal.

  3. In the New dialog box, enter the following information and click OK:

    • Name: Parka Online Sale
    • Type: Normal
    • Bucket Type: Weekly
    • Default Value: 0

  4. The new causal appears in the Causals screen. In the Series Values tab, enter the value 1 in each week that corresponds to the online parka promotion. (In this example, 1 is entered for the following weeks: 2016 Week 30, 2017 Week 18, 2017 Week 34, 2018 Week 8, and 2018 Week 31.)

  5. Click Save in the toolbar to see the results.

  6. In the Level Links tab, click Add in the Level toolbar to link the new causal to the filter with the parka items.

    • For this example, select the (All Items) filter.

  7. The Level Links tab now displays the items that are linked to this causal.

 3. Review the revised item forecast.
  1. From the Manage tab, click Forecasts.
  2. If the Items - Level View dialog box does not automatically appear, click Level View in the Forecast toolbar to open it.

  3. In the Items - Level View dialog box, select the filter that contains the items whose forecasts you want to review.

    • For this example, select the (All Items) filter.

  4. In the Forecasts screen select the following item in the Item list:

    • Item Master Code; 143
    • Item Master: All Season Lined Parka_Dark Navy-L
    • Channel Code: E-Com

  5. Review the forecast. Note that the Parka Online Sale causal now corresponds to the historic spikes in sales and also that upcoming Parka Online Sale promotions have corresponding spikes in demand.
