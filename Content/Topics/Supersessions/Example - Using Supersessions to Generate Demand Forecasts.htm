Example - Forecasting Supersession

In this example, we will use JustEnough's supersessions feature to create a demand forecast for a new item, "High Collar Angelina Jacket", using the sales history of an old item, "Leather Moto Jacket". This example shows how to create a supersession and demand forecast for a single color/size combination of the new item. In practice, you will need to repeat these steps for each color/size combination your new items possess.

Instructions

1. Create the supersession.
  1. From the Manage tab, click Supersessions.
  2. In the Supersessions toolbar, click New.

  3. In the Items dialog box, select the filter that contains the new and existing items.

    For this example:

    1. Select the Supersession Model Management (by Dep) filter.

    2. Add condition: ItemMaster Group 3 Description is One of the Following: Coats & Jackets.

  4. In the Add Supersession wizard, select the following old item to supersede:

    • Item Master Code: 543

    • Item Master Description: Leather Moto Jacket_Dark Navy-L

  5. Click Next to continue.

  6. Select the following new item to supersede the Leather Moto Jacket_Dark Navy-L:

    • Item Master Code: 708

    • Item Master Description: High Collar Angelina Jacket_Grey - L;

  7. Click Next to continue.

  8. Select the following supersession parameters:

    • Old Item Replenishment Strategy: No change.

    • Supersession Type: Forecasting only.

  9. Click Next to continue.

  10. Select the sites where you want to apply the supersession. In this example, all sites are selected.

  11. Click Next to continue.
  12. Click Finish to close the wizard and create the supersession.
2. Generate the demand forecast.
  1. In the JustEnough Manage tab, click Forecasts.

  2. In the Items - Level View dialog box:

    1. Select the Supersession Model Management (by Dep) filter.
    2. Add condition: ItemMaster Group 3 Description is One of the Following: Coats & Jackets..

  3. In the Forecasts screen, select the new item, "High Collar Angelina Jacket":

  4. Create a demand forecast for the new item:

    1. In the Forecast toolbar, click Forecast.
    2. In the Forecast Task dialog box, click OK to star the forecast tournament.

    3. Once the forecast is complete, it appears in the Forecast screen.
