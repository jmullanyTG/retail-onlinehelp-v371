Example - Replacing Old Items with New Items

In this example, we will use JustEnough's supersessions feature to create a demand forecast for a new item, "High Collar Angelina Jacket_Grey", using the sales history of an old item, "Leather Moto Jacket_Dark Navy".

Note: You can perform the steps in this example yourself by connecting to the JustEnough Demo database. All data, filters, and views used in this example have been recreated there for your convenience.

Instructions

  1. From the Manage tab, click Supersessions.
  2. In the Supersessions toolbar, click New.

  3. In the Items dialog box, select the filter that contains the new and existing items. For this example, select the filter "3.2 BC-3 CD-1 SD-1: Supersession Example”.

  4. In the Add Supersession wizard, select the following old item to supersede:

    • Item Master Code: 543
    • Item Master Description: Leather Moto Jacket_Dark Navy-L

  5. Click Next to continue.

  6. Select the following new item to supersede the Leather Moto Jacket_Dark Navy-L:

    • Item Master Code: 708
    • Item Master Description: High Collar Angelina Jacket_Grey - L

  7. Click Next to continue.

  8. Select the following supersession parameters:

    • Old Item Replenishment Strategy: No change.
    • Supersession Type: Forecasting only.

  9. Click Next to continue.

  10. Select the sites where you want to apply the supersession. In this example, all sites are selected.

  11. Click Next to continue.
  12. Click Finish to close the wizard and create the supersession.
