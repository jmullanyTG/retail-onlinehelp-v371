Supersessions Screen

The supersessions screen is where you can create, delete, view, and update supersessions.

Supersession Grid

The supersession grid lists all of the active and inactive supersessions in your JustEnough system. It contains the following details for each supersession..

Detail Description
Old Item The item which is supplying the supersession history. In many instances, this is the item which is being replaced (last year’s model).
New Item The item which will supersede the item described in the “old item” column. This is the item which is replacing the old item (this year’s model).
Superseded for Replenishment If selected, this will supersede the new item over the old item for all replenishment orders
Superseded for Forecasting If selected, this will bring in the demand history of the old item into the forecast engine in order to calculate the projected demand for the new item
Replenishment Ratio Value of the ratio to 1 which will be applied to the old item stock on hand quantities. A value of 1 will give a 1:1 ratio of old item quantities to new item quantities. A value of 2 will give a 2:1 ratio of new item demand to old item demand, a 100% increase. A value of 1.2 would be for a 20% increase.
Forecast Ratio Value of the ratio to 1 which will be applied to the old item sales history amounts. A value of 1 will give a 1:1 ratio of old item quantities to new item quantities. A value of 2 will give a 2:1 ratio of new item demand to old item demand, a 100% increase. A value of 1.2 would be for a 20% increase.
Sales History Start Date This will define the beginning of the measurement period for the old item sales history.
Sales History End Date This will define the end of the measurement period for the old item sales history.
Supersession Start Date This will define when the supersession becomes active. If blank, the supersession will be considered active from the go live date of the system.
Supersession End Date This will define when the supersession becomes inactive. If blank, the supersession will be considered to continue in perpetuity.
Active The active flag will display in the check box if the supersession is currently active.
Old Item Type This will determine the replenishment status of the old item.
Redundant The item will remain in stock, but will no longer be ordered.
Non-Stocked The item will no longer remain in stock (not on the shelf).
User The UserID of the creator of the supersession. Useful for audit purposes.
Date Created The date on which the supersession was created.

Sites Affected

For the supersession highlighted in the supersession grid, this will display the sites affected by said supersession.

Notes

A place for additional notes. Will be populated with the UserID of the note adder and a timestamp.

Actions

The following actions are available from the Supersession toolbar.

Action Description
New Creates a new supersession.
Edit Updates the sites for the selected supersession.
Delete Deletes the selected supersession.
Save Saves property changes made on the Supersessions grid.
