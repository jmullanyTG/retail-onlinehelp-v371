Creating Supersessions

Setup

The following object should be set up before creating a supersession.

  • A filter that includes the new items AND the the old/existing items they are superseding.

Instructions

  1. From the Manage tab, click Supersessions.
  2. In the Items dialog box, select the filter you want to use for the supersession.

    Note: If you are already working in the Supersessions screen, click New on the Supersessions toolbar to open the Items dialog box.

  3. In the Items dialog box, select a filter that contains the old/existing items, the new items, and the relevant sites.
  4. In the Add Supersession wizard, select the existing item(s) to supersede.

  5. Click Next to continue.

  6. Select the new item, which will replace the old item.

  7. Click Next to continue.

  8. Review or edit the supersession parameters

  9. Click Next to continue.

  10. Select the sites where you want to apply the supersession.

  11. Click Next to continue.
  12. Click Finish to close the wizard and create the supersession.

Results

Once created, supersessions appear in the Supersessions screen. You can then generate demand forecasts for the new items manually from the Forecast screen, or wait for the forecasts to be generated by the daily job manager.
