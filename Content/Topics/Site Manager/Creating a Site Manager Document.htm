Creating a Site Manager Document

  1. From the Manage tab, click Site Manager.

  2. In the Items dialog box, select the filter that contains the sites you want to manage.

    Note:The Items dialog box opens automatically when you first open the Site Manager screen. If you are already working in Site Manager and want to manage a different set of sites, click Site Manager Document in the Site Manager toolbar to create a new site document.

  3. The new document opens in Site Manager with a limited set of fields displayed in the grid.

    In the local Columns toolbar, click Choose to select additional fields to display. Choose fields that allow you to easily identify the sites you are working with and which contain the data you are interested in.

  4. In the document, you can compare details across sites in the gird, or select individual sites in the grid to view additional details in the Detail pane.

    :

    Note: If you change any settings in the Details pane, remember to click Save to save the changes.
