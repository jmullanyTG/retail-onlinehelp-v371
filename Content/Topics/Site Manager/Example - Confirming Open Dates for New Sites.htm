Example - Confirming Open Dates for New Sites

In this example, we will find new sites imported from the ERP and confirm their opening dates are scheduled accurately. Site opening dates help determine when item orders need to be placed for the new stores.

Instructions

  1. From the Manage tab, click Site Manager.

  2. In the Items dialog box, select the Identify New Stores filter.

  3. In the Site Manager document, you can see that two new sites have been added (Calgary and Nashville).

  4. In the local Columns toolbar, click Choose to select the additional fields you want to appear in the document. In this example, we will choose fields related to site type, site status, and site group.

  5. In the document, choose one of the new sites and click on its row in the grid to view its details. (If the Details pane is hidden, click Details on the toolbar to show it.)

  6. In the Site Status tab, find the Closed entry. If the Status End Date value is empty, select the day before the store is scheduled to open and then click Save in the Entry toolbar.

  7. If there is not already an "Open" status entry for the site, click New in the Entry toolbar to add a new status.

      1. Enter the following values for the status entry:
        • Site Status = Open
        • Site Status Attribute = Open
        • Status Start Date = The date of the store opening.

      2. Click Save in the Entry toolbar to save the new status entry.

  8. In the Site Manager grid, you can view the values in the Days to First Open, Days to Status Change, Next Site Status Date, Next Status Description, and other status-related fields.
