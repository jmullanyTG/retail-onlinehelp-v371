Site Status

Site status refers to when a site will be open (such as a new store opening) or closed (such as for remodeling or permanent closure). The system uses site status to generate accurate forecasts and order recommendations, so it is important to keep sites updated with status changes as they are planned.

The Site Status tab contains a grid listing when the status of a selected site is scheduled to change. Each upcoming status change is listed as a row in the grid.

Site status fields

You can add fields to the grid by clicking Fields in the Entry toolbar.

Field Description
Site Code

The numeric site code of the selected site.
Site ID The Site ID.
Site Status

Whether the site will be open or closed during this period.
Site Status Attribute

Describes the reason for the status change and specifies whether or not orders will be accepted at the site during this period.

Examples:

  • Remodel (accepting orders)
  • Expansion (no orders)
  • Permanently closed
Notes Optional user-entered notes.
Site Status Temporal ID The site status temporal ID.
Status Start Date The date on which this status is to go into effect.
Status End Date The date on which this status is to end.

Adding a new status entry for a site

  1. Open a Site Manager document containing the desired site.
  2. Select the site in the document grid. Make sure that the Details pane is displayed.
  3. In the Site Status tab, click New on the Entry toolbar. A new row is added to the grid below.

    • Select the desired values in the Site Status and Site Status Attribute fields.

    • Select the desired start date and (if there is one) end date for the status in the Status Start Date and Status End Date fields.

    • Enter any notes for the status change in the Notes field (optional).

  4. Click Save on the Entry toolbar to save the change and populate the entry's ID values.

Modifying a status entry for a site

  1. Open a Site Manager document containing the desired site.
  2. Select the site in the document grid. Make sure that the Details pane is displayed.
  3. In the Site Status tab, modify the entry as desired.
  4. In the Entry toolbar, click Save to save the change or Cancel to discard the change.

Deleting a status entry from a site

  1. Open a Site Manager document containing the desired site.
  2. Select the site in the document grid. Make sure that the Details pane is displayed.
  3. In the Site Status tab, select the status you want to delete.
  4. In the Entry toolbar, click Delete to remove the entry from the grid.
  5. In the Entry toolbar, click Save to save the change or Cancel to discard the change.
