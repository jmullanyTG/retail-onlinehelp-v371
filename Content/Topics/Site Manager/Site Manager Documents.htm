Site Manager Documents

Site Manager views are organized as documents. Documents are a flexible way to view reports and data within JustEnough as they allow users to:

  • Undock the view from the UI shell, thus giving users window management options.
  • Take advantage of multi-screen setups.
  • Organize documents into vertical or horizontal tabs, allowing for side-by-side comparison of filters.
  • Use enhanced navigation with several documents, including the use of “ctrl + tab” to switch between document tabs.
