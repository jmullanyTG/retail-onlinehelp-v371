Site Location Strategies

Within Allocations, store type sites are grouped into location strategies in order to determine allocation quantities; the membership of a specific site is not exclusive to a single location strategy. The site location strategies view allows users to view and maintain the membership of a site within active location strategies.

  • Is Included? – If a site is included within a location strategy, the checkbox will be checked. Users may manipulate a site’s membership within this view (Please note: any updates to location strategies will require recalculation of all linked allocation strategies.). DC site types may not be included within location strategies.
  • Strategy Location Name – The name of the location strategy.
  • Strategy Location Description – The description of the location strategy.
