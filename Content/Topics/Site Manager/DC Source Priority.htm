DC Source Priority

The DC source priority view will display the site specific sourcing priority for distribution centers. This is a read only view and is populated by the Supply Chain module. There is a system policy used to determine the supply chain which will populate this view. All active DC sites which are not included in the site’s priority will be struck-through.

  • Source Site Code – Site code for the DC.
  • Source Site Type – The site type (DC, DC/Store).
  • Source Site – The name of the DC site.
  • Source Site Priority – The priority ranking as established by the supply chain.
  • Default Site Lead Time Days – The default lead time (set at site level; product agnostic).
