Site Source Priority

The site source priority view site specific sourcing priority for other sites which are classified as stores. This is a read only view and is populated by the Supply Chain module. There is a system policy used to determine the supply chain which will populate this view. All active store sites which are not included in the site’s priority will be struck-through.

  • Source Site Code – Site code for the store.
  • Source Site Type – The site type (Store).
  • Source Site – The name of the site.
  • Source Site Priority – The priority ranking as established by the supply chain.
  • Default Site Lead Time Days – The default lead time (set at site level; product agnostic).
