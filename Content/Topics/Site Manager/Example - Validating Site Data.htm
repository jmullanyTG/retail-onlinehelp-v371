Example - Validating Site Data

In this example, we will create a Site Manager document that displays all stores added within the past 60 days. We will then use this document to verify that the information received from the ERP system for each site is valid.

Instructions

  1. From the Manage tab, click Site Manager.

    Note: If the Items dialog box does not automatically open, click + Site Manager Document on the toolbar to open it.

  2. In the Items dialog box, select the filter "Identify New Stores" from the How To Guide Examples filter group.

  3. In the site document, you can see the newly added stores (Calgary and Nashville).

  4. In the document's Columns toolbar, click Choose to select the additional fields you want to appear in the site document. For this example, choose fields related to site type, site status, and site group.

  5. You can now review the site groups, attributes, status, and other data fields for each new site to ensure they are correct. Be sure to also click the tabs in the Detail pane to ensure the values they show are as expected.

    Note: If you change any site settings in the Details pane, remember to click Save to save the changes.
