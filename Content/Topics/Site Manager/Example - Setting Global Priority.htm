Example - Setting Global Priority

In this example, we'll set the global priority values for new stores Calgary and Nashville.

Instructions

  1. From the Manage tab, click Site Manager.

  2. In the Items dialog box, select the filter that contains the sites you want to manage.

    For this example, select the Identify New Stores filter

  3. In the site document, you can see that two new sites have been added (Calgary and Nashville).

  4. Click on either store in the grid to populate the Details pane.
  5. In the Details pane, click the Global Site Priority tab.

  6. Find the Nashville store in the Global and Group Priority grid, and change its Global Priority value to 43.

  7. Click Save in the Global Site Priority toolbar. Nashville is now ranked 43rd in the Global Site Priority list.
