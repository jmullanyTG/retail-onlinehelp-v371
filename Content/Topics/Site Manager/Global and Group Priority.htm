Global and Group Priority

Global and group priority is a globally applicable view and is not driven by the selected site in the top grid. Within this view, users will rank sites based on their priority as it relates to receiving stock in potential shortfall situations.

Priority group will allow users to “protect” a specific set of sites before referring to global priority to resolve shortfall situations.

  • Site Code – Site code for the store or DC.
  • Site Type – The site type (DC, Store, and DC/Store).
  • Site Description– The name of the site.
  • Global Priority – Unique, globally applied priority. Used to resolve shortfall situations. 
  • Priority Group – Associated priority group. Used to resolve shortfall situations.  Takes precedence over global priority.
