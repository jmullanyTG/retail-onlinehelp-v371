Example - Assigning Schedules to Sites

Schedules help JustEnough determine when orders need to be placed so that inventory can get to your sites in time to meet demand.

You can assign multiple types of schedules to a site from within Site Manager. In this example, we will assign a receiving schedule to a new store in Nashville.

Instructions

  1. From the Manage tab, click Site Manager.

  2. In the Items dialog box, select the Identify New Stores filter.

  3. In the site document, you can see that two new sites have been added (Calgary and Nashville).

  4. In the local Columns toolbar, click Choose to select the additional fields you want to appear in the site document. In this example, choose the Receiving fields in the Site Schedules group.

  5. In the Site Manager document, find the row in the grid for Nashville. Click in its Receiving Schedule column and select the schedule US Retail Stores - Week Day Delivery Only.

  6. In the Site Manager document toolbar, click Save to save the schedule assignment.
