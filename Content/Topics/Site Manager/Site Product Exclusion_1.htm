Site Product Exclusion

Only one product exclusion may be set per site at a time. As such, users must either create an appropriate item master filter to fulfil their requirements, or set the exclusion at an applicable level of the hierarchy.

ItemMaster Hierarchy Level

When ItemMaster hierarchy level is selected, users will be presented with a tree wizard which represents the selected hierarchy as set by the system policy.

Levels which should be excluded should be selected with a check mark.

ItemMaster FilterMaster

When ItemMaster filter master is selected, users will choose a filter using the filter chooser, which will then be applied.

Users should apply any desired changes by selecting “Apply” before exiting. If desired, the exclusions may be removed by deleting the entry using the “Delete” button.
