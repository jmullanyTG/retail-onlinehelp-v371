Practical Application

The step rules are managed within the “edit method definition” window. A step rule may be added to a definition by clicking on the “add” button, and “delete” to remove a step rule from the definition.

Clicking on the lower “add” button will populate the step rules stages and default parameter values windows. A step rule may appear in more than one stage by clicking the “add” button multiple times.

A default value for the parameter must be chosen for any series rule which requires it.
