Practical Application

The forecast definition module is located under the “maintain” tab. It is commonly restricted to most users. Upon opening, users will be shown the available forecasting methods. They may choose to create a new method, however as of version 3.1, these custom methods will not be available for the forecasting tournament.

To edit a forecasting method, users may highlight the row and click the “Edit” button, or simply double click on the row.  This will bring up the “edit method definition” window.

Within the “Method Fundamentals” tab, users will be able to edit the name, description and information related to the forecast base.

  • Method Name – The name for the forecasting method. Will commonly match the description in the “base method” field.
  • Method Description – Any additional description for the forecasting method.
  • Method Group – Used for sorting and organization of multiple forecasting methods.
  • Method Active – A flag used to determine if the method is active and may be entered into the forecasting tournament.
  • Base Method – Displays the possible methods for use in forecasting within JustEnough.
  • Forecast Mode – Displays the three possible mode types.
  • Included Parameters – Will display the mathematical parameters associated with the method displayed in the “base method” drop down.
