Versioned Content Output Configuration

Located in the Output tab of the Event Workbench, the ‘Versioned Content Output Configuration’ allows someone with administrative rights to configure the composition of any Versioned Content Export file. In this screen there is a ‘Versioned Content Output File Export Settings’ tab which contains various settings used to export versioned content for use with InDesign, or other creative layout applications.

In the case that versioning is not being computed for a Slot Set, the ‘Non-Export Prefix’ setting on this tab could be the source of the problem. This setting checks the Slot Set Name to see if it starts with the designated prefix, and exclude it from version calculations if it does. If Slot Sets are not getting versioned even though they should, checking this setting may reveal that this is the case because of how the Slot Set is named, or a bad prefix value for the setting. If the prefix is not to be used the easiest way to ensure it will not interfere with versioning is set the value to “Export all Slot Sets” (which must be manually keyed).
