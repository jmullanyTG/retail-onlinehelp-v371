Event Manager Global Policies

Global Policies are configured to set up default settings for all JustEnough solutions. They are configured by Professional Services and should not be modified without assistance from JustEnough Professional Services or Support. Changes to some Policies may require closing and opening the JustEnough software before the changes are available so it is recommended to restart the software after any Policy change. Below is a list of global policies related to events and the purpose of each.

To configure event policies:

  • From the Maintain tab, click Policies and choose Module:Events in the navigation pane.

Global Policy

Description

Allow Sale date range to occur before Ad date range

If set to ‘True’, provides the ability to set an Event Sale Start Date to occur before the Event Ad Start Date.

Allowed Versioned Content Output Methods

Provides ability to customize output and affects in the Output Method dropdown list options in the Workbench > Output > Generate Output for Page Layout tab.

  • Standard – Output for versioned content will be formatted to meet the standard Generate Output process for import into a desktop publishing software.
  • Custom – Output for versioned content will be customized as defined by a custom database procedure to meet specific customer needs.
  • Both – The user is given the option to select either ‘Standard’ or ‘Custom’ for each Slot Set in the Event to determine what formatting method should be used for each.

 

Default new Promotion with Event date range

Provide the ability to set the default start and end dates for Promotions created from within an Event

  • Use Sale date range – Set the Promotion dates to match the sale dates.
  • Use Ad date range – Set the Promotion dates to match the ad dates.

Event Calendar View Collection Set

Assign the View Collection Set to display the Event Calendar by its ID. If this is not set to a valid ID, a blank screen would be displayed in the Event Calendar tab.
Event Level Output

Choose whether to output an event's slot sets to a single file or multiple files:

  • Event – Outputs all of an event's slot sets to a single file in the configured event directory. (Note: If this option is selected, the Allowed Versioned Content Output Methods policy must be set to Standard. Custom output is not supported.)
  • Slot Set – Outputs selected slot sets as separate files in slot set-specific sub-directories.

Event Module View Collection Set

Assign the View Collection Set ID to be display in the third tab of Event Manager by its ID. If this is not set to a valid ID, a blank screen would be displayed for this customized tab.

Gantt View date display type for Events

Ability to set the default Event dates to display in the Event Calendar:

  • Sale dates – Default the calendar to show Events listed by their Sale Start and End Dates.
  • Ad dates – Default the calendar to show Events listed by their Ad Start and End Dates.

Go to Promotion Module after Promotion creation

Provide the ability to go to Promotion Detail screen after a new Promotion is created from within an Event. If this is set to ‘False’, the user will return to the Event screen after the New Promotion Wizard is closed.

New Promotion View Collection Set

Assign the View Collection Set to display customized information in a second tab in the Products grid of the New Promotion Wizard by its ID. If this is not set to a valid ID, the second tab will not display in the New Promotion Wizard screen.

 

Populate Online Proofing Data

Provide the ability to generate additional data sets required for the JustEnough Online Proofing solution.

Promotion Double Click Action

Provide the ability to set what action occurs when the user double clicks on a Promotion in an Event:

  • Go to Promotion module – Open the Promotion Detail tab in the Promotions solution.
  • Open Promotion window – Open the Promotion in a window while remaining in the Events solution.
  • None – Do nothing when the user double clicks on a Promotion.

Note: This setting will limit the user to only modifying or viewing the details of a Promotion from the Promotions solution.

Select Destination Option for Event Generate Output

Provide the ability to set where the generated Event output is sent:

  • Pipe Delimited File – Generate Output will create pipe delimited data stored in files that can be used by page layout software.
  • Database Only – Generate Output will populate database tables.

Show Auto Filter Row for Campaign Data Grid

Provide the ability to display a filter row at the top of the Campaign data grid. The filter will refine the list of Campaigns that begin with the string entered in the filter.

Show Auto Filter Row for Event Data Grid

Provide the ability to display a filter row at the top of the Event data grid. The filter will refine the list of Events that begin with the string entered in the filter.

Show View Collection Set in Event Calendar

If this is set to ‘True’, show the View Collection Set defined in the ‘Event Calendar View Collection Set’ policy, otherwise use the static Event Calendar.

Slot Set display order for Events

If an Event has more than one Composition, the Slot Sets can be displayed in the Event tree in the following ways:

  • Slot Set Name Order – Alphabetically, by Slot Set Name.
  • Composition Order – In the order defined in the first Composition with remaining Slot Sets displayed in alphabetical order.

Validate Promotion Date Range

Provide the ability to ensure the Promotion date range matches the Event start and end sale dates before allowing the user to add it to the Event.
