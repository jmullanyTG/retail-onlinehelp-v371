Geographies and Sites

When the Event Name is selected in the Event Navigation Pane, this screen displays all the Geographies for the Event and their Site assignments. The Geographies and Sites assigned to the Event are defaulted by the Event Type configuration. If needed, the default can be modified for each specific Event.

Note: The label of this tab is configurable by Event Type.

To view the Sites for a Geography, select it in the left pane and the Sites will be displayed in the right pane.
