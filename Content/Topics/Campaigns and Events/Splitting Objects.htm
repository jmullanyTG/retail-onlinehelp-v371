Splitting Objects

Splitting divides a single object into two or more equally-sized objects. It is available to the following object types:

  • Price display format containers
  • Attribute display format containers
  • Item display format containers
  • Slot set templates
  • Event slot sets

Split options

The Split toolbar menu contains the following options:

  • Multiple - Allows you to specify the number of rows and columns you want to split the object into.
  • Horizontal - Splits the selected object into two equally-sized "stacked" objects (one on top of the other).
  • Vertical -Splits the selected object into two equally-sized side-by side objects.

Notes:

  • You can choose a split option from the Split menu on the toolbar or by right-clicking an object and choosing the option from the context menu.

  • You cannot split objects into heights or widths smaller than 6 DIUs (about 1/32 inch). For slots, the minimum slot height or width allowed after a split is 48 DIUs (about 1/2 inch)
  • When splitting slots, the "new" slot resulting from a split will have the same space and item display format as the original slot.

Examples

The following examples are shown using price display formats, but apply to the other supported objects as well.

Split Multiple

  1. Select an object. The selected object is outlined in blue.

  2. Choose Split > Multiple from the toolbar (or right-click and choose Split Multiple).

  3. In the Split Into Multiple dialog box, enter the number of rows and columns you want to split into.

    If splitting a slot, you must also specify a default event space and item display format for the split slots:

  4. Click OK to split the object. It is divided equally into the specified number of rows and columns.

Split Horizontal

  1. Select an object.

  2. Choose Split > Horizontal from the toolbar (or right-click and choose Split Horizontal).

  3. The object is divided into two equally-sized objects, stacked top-and-bottom.

Split Vertical

  1. Select an object.

  2. Choose Split > Vertical from the toolbar (or right-click and choose Split Vertical).

  3. The object is divided into two equally-sized objects, positioned side-by-side.
