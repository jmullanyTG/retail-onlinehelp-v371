Content Update window

When the Ok or Save buttons are clicked to save the updated Promotion Content, the Content Update window is displayed.

In the window, the user can choose to push Event Content (i.e. Copy, Images or both) to the following:

  • Master Content – If the Promotion consists solely of a single Product or Product Group that exists in Content Manager, it will be available to update with the Event Content
  • Promotion Content – If selected, the Event Content changes will update the original Promotion
  • Event Content – If the Promotion is associated with any other Events, they will be listed and the user can select which updates, if any, should be pushed to the other Events

Click Ok to save and update selected instances.
