Attribute Display Formats

Attribute display formats display promotion attribute fields in an item display format.

You can view and edit attribute display formats from the Attribute Display Formats tab in the Event Configuration window.

Previewing an attribute display format

On the left side of the Attribute Display Formats tab is an Attribute Display Format Name list. Select a name to preview its associated attribute display format. You can use the Zoom slider as needed to magnify the preview.

Containers

An attribute display format contains one or more containers. Each container can include its own text and attribute values, and its properties (size, font options, etc.) can be individually customized.

Toolbar actions

The following actions are available from the Attribute Display Format toolbar.

Action

Description
New Creates a new attribute display format from scratch. See Modifying an Attribute Display Format for details.
Clone Creates a new attribute display format by copying the selected attribute display format. The clone contains all the content and properties of the selected attribute display format. You can give it a new name as needed.
Edit Modifies the selected attribute display format. See Modifying an Attribute Display Format for details.
Delete Deletes the selected attribute display format.
