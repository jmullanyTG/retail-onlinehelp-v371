Create a Cost

To add a new cost, select New. This will create a new row in the grid for the user to modify as needed to reflect the new cost. Select Event Save to save the new cost information and to update the Total Event Costs and Available Event Budget fields.

The user can enter the following information for each cost:

 

Field

Description

Cost Type (Required)

Select the Cost Type.

Estimated Amount (Required)

Enter the projected monetary cost for the Cost Type.

Actual Amount (Required)

Enter the actual monetary cost for the Cost Type.

Cost Level (Optional)

Select one of the following from the dropdown list:

  • Event
  • Promotion
  • Slot Set

Applied To (Optional)

This dropdown list will refresh after the Cost Level is selected. Select one from the dropdown list depending on the selection in Cost Level:

  • Event – The system will default the current Event
  • Promotion – A list of all Promotions in the Event
  • Slot Set – A list of all Slot Sets in the Event

Supplier (Optional)

Select the dropdown list from the list of Suppliers that are maintained in the Suppliers solution.

Invoice Number (Optional)

Enter the Invoice Number associated with the cost.

Comments (Optional)

Enter any additional comments for the cost.
