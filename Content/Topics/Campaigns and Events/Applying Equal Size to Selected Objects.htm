Applying Equal Size to Selected Objects

The Equal Size toolbar button lets you resize multiple objects to be sized equally in height and width.

  • Objects must be adjacent to each other along the same dimension (horizontal or vertical).
  • You can select multiple objects as long as each is adjacent to the original or succeeding objects.

Note: The screenshots below are from the Edit Price Display Format window, but the steps are the same for any container or slot.

  1. Use CTRL + left-click to select adjacent objects (above/below/left/right). Selected objects have an orange outline.

    Note: Objects which do not qualify for equal-sizing include:

    • Objects along a different dimension (vertical or horizontal) than previously-selected objects.
    • Objects that do not span the entire border of a previously-selected object.

  2. Click Equal Size to divide the selected containers into equal widths or heights.

Notes:

  • Objects cannot be smaller than 6 DIUs (about 1/32 inch) in width or height.
