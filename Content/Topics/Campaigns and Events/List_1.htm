List

The Slot Set List view, displays the Slots in the selected Slot Set as rows in a grid. Access to this screen depends on the user’s rights.

For each Slot, the following attributes are displayed as columns:

Field

Description

Slot Number (Read Only)

The Position of the Slot in the Slot Set.

Assigned Promotion (Read Only)

If a Promotion has been added to the Slot, the Promotion Name will be displayed.

Content Version (Read Only)

If a Promotion has been added to the Slot and more than one Content Version exists for it, each Content Version will have its own row in the grid. The Content Version Name will be displayed in this column.

Item Display Format (Optional)

Indicates the Item Display Format to be used across all Promotions in the Slot. This may be modified by selecting a new Item Display Format from the dropdown list. All Promotions associated with a Slot must use the same Item Display Format.

Space (Optional)

Indicates the Space assignment for the Slot. This may be modified by selecting a new Space from the dropdown list. Changing the Space assignment may affect user permissions to the Slot.

Status (Read Only)

If a Promotion has been added to the Slot, the Status is ‘Assigned’, otherwise it is ‘Open’. It is possible for a Slot to have Statuses of ‘Assigned’ and ‘Opened’ if not all of the available Geographies are assigned a Promotion.

Notes (Optional)

Enter instructions or additional information related to the treatment of the assigned Promotion directly into the grid. This is not editable if the Slot is ‘Open’.

Start Date (Read Only)

Displays the Promotion Start Date.

End Date (Read Only)

Displays the Promotion End Date.

Every Slot Set has a Position 0 Slot. This is where all unadvertised Promotions will be located for the Slot Set. Position 0 does not have an associated Item Display Format, Space, and the Status is ‘Unassigned’.

Selecting a Slot in the grid, will refresh the Geography and Promotion panes below the grid.

The Geography pane indicates which Geographies will be automatically assigned to the Promotion when it is added to the Slot. The Promotions pane indicates which Promotions are available to add to the Slot. A Promotion with a checkmark, indicate that it has been added to the Slot Set but not necessarily the selected Slot. If a Promotion is assigned to the selected Slot, only it will be the only one displayed in the Promotions pane.

Depending on the Global Policy ‘Promotion Double Click Action’ setting, double clicking on an assigned Promotion in the grid will load the Promotion in either a window or in the Promotions solution.
