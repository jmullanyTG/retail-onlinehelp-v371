Versioned Content Output File Export Settings

The settings of the output file are configured in the ‘Versioned Content Output File Export Settings’ tab of this view. These settings are configured by Professional Services and should not be modified without assistance from JustEnough Professional Services or Support. If changes are made to any of the following settings, click Save to save.

 

Field

Description

Output Root

The root destination directory for exported output files. Subfolders are created for the Event and its associated Slot Sets. The output files are stored under the Slot Set subfolders.

Image DTP Path Attribute

The repository attribute that will be exported for every Image in an Event.

Non-Export Prefix

Specifies a Slot Set prefix that identifies that the Slot Set should not be included in the output file. This should not be set to any value that is likely to be used in Slot Set naming.

Database

This setting will be used to connect to the database during the export process.

DBServer Name

This setting will also be used to connect to the database during the export process.
