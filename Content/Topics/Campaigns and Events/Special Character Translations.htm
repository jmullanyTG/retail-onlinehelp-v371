Special Character Translations

To avoid mistranslations in Adobe InDesign, the Generate Output to File process automatically replaces some special characters with strings.

Character Replacement string
¢ &cent
• &bull
™ &trade
® &reg

To configure InDesign to properly translate these strings back into their intended symbols:

  1. In InDesign, select a field.
  2. Go to Field Properties > General.

  3. In the Cleansing Options area, enter the following translations:

    &cent=¢;&bull=•;&trade=™;&reg=®

  4. Repeat for all fields in which these symbols may appear.

These configurations must be saved with the panel data source to be used to import the files into EasyCatalog.

Note: The syntax for the cleansing options varies between versions of EasyCatalog. Refer to the your EasyCatalog User Guide for the correct translation for your version.
