Editing Slot Set Layouts

Layout Edit Mode, available from the Layout view, is the only place where you can edit the design of event slot slots. From here, you can customize slots, their positions, space assignment, and item display format assignment. Certain actions may be restricted based on user rights.

To toggle entering/exiting Layout Edit Mode from the Layout view:

  • Click the Layout Edit button (located about the slot set preview, next to the Zoom slider).

Once in Layout Edit Mode, the toolbar actions change and new panes appear in the right side of the Layout view.

  • The Slot Information pane, located in the top-right of the view, shows property details of the selected slot.

  • The Space Allocation pane, located in the bottom-right of the view, shows space allocation by slot set and the overall event.

Actions

You can perform the following actions while in Layout Edit Mode.

Action

Description
Edit slot properties In the Slot Information pane, you can edit the following properties of a selected slot:
  • Slot Name
  • Event space
  • Item display format
  • Notes
Display headers Clicking the button next to the Zoom slider toggles showing/hiding slot hearers, which display the position and item display format for each slot.
Display dimensions Clicking the button next to the Zoom slider toggles showing/hiding each slot's dimensions and relative size.
Drag promotions to different slots Dragging a promotion from one slot to another is possible within the same slot set, as long as the promotion has at least one site cluster that matches the destination slot's geographies. Dragging removes the promotion from the original slot.
Renumber slots Clicking the # button next to the Zoom slider toggles entering/exiting slot renumbering mode.
Save changes The Save toolbar button saves changes made to the slot set.
Split a slot Splits the selected slot into multiple slots. Slots cannot be split into widths or heights smaller than 48 DIUs (about 1/2 inch).
Merge slots Merges selected slots.
Equal size The Equal Size toolbar button resizes selected slots into equal heights or widths across their combined area. Slots must be adjacent to each other and aligned vertically or horizontally.
Clear all The Clear All toolbar button removes all slot configurations and resets the slot set to a single slot with default settings.
Undo and redo changes The Undo and Redo toolbar buttons undo and redo unsaved changes to the slot set.
