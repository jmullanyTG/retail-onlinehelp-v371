Create an Event Type

In the Event Type tab, click New to create a new Event Type, enter the required fields and click Ok.

Field

Description

Event Type Name (Required)

Enter a descriptive label to summarize the purpose of the Event Type. This will identify the Event Type when creating an Event.

Media Type (Required)

Select a Media Type for the Event Type from the dropdown list.

Geography Set (Required)

Select a default Geography Set for the Event Type from the dropdown list.

Color (Required)

Select the color to display Events of this Event Type in a calendar.

Workflow (Optional)

Select a default Workflow for Events of this Event Type, from the dropdown list of configured Event Workflows.

Description (Optional)

Enter a summary of the Event Type.

Composition Label (Required)

Enter a descriptive label for the Composition to be used when setting up Event Compositions by Geography.

Allow Multiple Compositions (Optional)

Select the checkbox if the Event Type allows for more than one Composition per Event.

 

Slot Set Label (Required)

Enter a descriptive label for Slot Sets to be used when working in Events.

Geography Label (Required)

Enter a descriptive label for the geographies to be used when working in Events.

Event View Collection Set (Optional)

Select a View Collection Set from the dropdown list to display in the third tab of the Event. ‘Event Workbench’ is the standard View Collection Set provided for this tab.

Slot Set View Collection Set (Optional)

Select a View Collection Set from the dropdown list that will display in the bottom pane of the screen after clicking Details in an Event Slot Set.

Valid Slot Set Templates (Optional)

Click Edit to select the slot set templates that can be used for events of this event type. The selected templates will appear as available options in the Slot Set Information pane when editing existing events as well as in the New Event wizard when reviewing the event's default composition.

The Default checkbox indicates the default template used when adding new slot sets to events of this event type.

Slot Set (Optional)

The Slot Set grid provides configuration for the default Composition (i.e. ordered series of Slot Sets with Templates) for new Events of this Event Type. Using the Add and Remove buttons, set up the default Slot Sets, including the Slot Set Template, and the Slot Set order for Events.
