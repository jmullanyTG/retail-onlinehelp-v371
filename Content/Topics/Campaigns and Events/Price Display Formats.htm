Price Display Formats

Price display formats determine how price fields appear in an item display format. A price display format exists for every promotion strategy defined in Promotion Manager.

You can view and edit price display formats from the Price Display Formats tab in the Event Configuration window.

Previewing a price display format

On the left side of the Price Display Formats tab is a Strategy Name list, which is a list of your defined promotion strategies. Select a strategy to preview its associated price display format. You can use the Zoom slider as needed to magnify the preview.

Containers

A price display format contains one or more containers. Each container can include its own text and price field values, and its properties (size, font options, etc.) can be individually customized.

Modifying a price display format

To modify a price display format, select it from the Strategy Name list and click Edit. The Edit Price Display Format window opens.

From this window, you can perform the following actions on the price display format:

Action

Description
Edit properties The height, width, font properties, and text of a price display format appear on the left side of the window.
Modify text You can add, delete and edit the text in a price display format by clicking in the Text Preview property box and making changes there. The preview updates automatically as you type.
Save changes The Save toolbar button saves changes made to the price display format.
Split a container Splits the selected container into multiple containers. Containers cannot be split into widths or heights smaller than 6 DIUs (about 1/32 inch).
Merge containers Merges selected containers.
Equal size The Equal Size toolbar button resizes selected containers into equal heights or widths across their combined area. Containers must be adjacent to each other and aligned vertically or horizontally.
Clear all The Clear All toolbar button removes all sizing, font attributes, text and price fields from the price display format.
Undo and redo changes The Undo and Redo toolbar buttons undo and redo unsaved changes to the price display format.
Refresh The Refresh toolbar button refreshes the price display format with the last saved changes.
