Modifying an Attribute Display Format

To modify an attribute display format, select it from the Attribute Display Format Name list and click Edit. The Edit Attribute Display Format window opens.

Note: The New Attribute Display Format window is the same as the Edit Attribute Display Format window, except it starts with a blank Name field and a single empty container.

From this window, you can perform the following actions on the attribute display format:

Action

Description
Edit properties The name, height, width, font properties, and text of an attribute display format appear on the left side of the window.
Modify text You can add, delete and edit the text in an attribute display format by clicking in the Text Preview property box and making changes there. The preview updates automatically as you type.
Save changes The Save toolbar button saves changes made to the attribute display format.
Split a container Splits the selected container into multiple containers. Containers cannot be split into widths or heights smaller than 6 DIUs (about 1/32 inch).
Merge containers Merges selected containers.
Equal size The Equal Size toolbar button resizes selected containers into equal heights or widths across their combined area. Containers must be adjacent to each other and aligned vertically or horizontally.
Clear all The Clear All toolbar button removes all sizing, font attributes, text and attribute fields from the attribute display format.
Undo and redo changes The Undo and Redo toolbar buttons undo and redo unsaved changes to the attribute display format.
Refresh The Refresh toolbar button refreshes the attribute display format with the last saved changes.
