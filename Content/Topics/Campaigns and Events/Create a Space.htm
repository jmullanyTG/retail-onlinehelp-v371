Create a Space

In the Spaces tab, click New to create a new Space, enter the required fields and click Ok.

Field

Description

Space Name (Required)

Enter a descriptive label to summarize the purpose of the Space. This will be used to identify the Space when setting up Slot Set Templates or modifying the Layout of an Event Slot Set.

Select Color (Required)

Modify the default color if desired. The color should identify the Space to the user when working in an Event.

Assign Access Rights (Optional)

By default, no users will have access to add Promotions to the Space. To assign access rights, click Assign Access Rights.

Select roles or users that should have access to the Space and click Ok. Both roles and specific users can be assigned to the Space. Event Slot Set access is also restricted by Event Workflow Status and User Rights.
