Images

There are three ways to add Images to the Content Version:

Action

Description

Add Existing

If the Image already exists in the database, click on this button, select the checkbox next to the Image(s) to be associated with the Content Version and click Ok.

Add

If the Image is stored locally or on the network, click on Add and navigate to the desired Image(s) and click Open.

New

If the required Image doesnâ€™t exist, it can be requested by clicking on Image Request New. The New Image Request window will provide a form that can be completed to initiate a request. Click on Ok.

Note: The three methods of adding images are not mutually exclusive and all three methods can be used within a Content Version.

 

Each new Image added should be associated with a View Type so it can display in an Item Display Format. This association can be made directly in the Images tab grid by selecting the View Type dropdown list and selecting a value for each Image. Each View Type can only be associated to one Image.
