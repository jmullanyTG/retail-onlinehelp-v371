Filtering

Action

Description

Show All

By clicking on this button, the user can toggle between displaying all Campaigns or only Campaigns that are assigned to the Events in the Events Filter.

Events Filter

To filter the list of Events, click Filter and the system will display the Events Filter window. The user may select from an existing Filter or click on the black hammer icon to create a custom Filter.

The Calendar view also provides the ability to display Campaigns and Events in monthly or daily (agenda) views. To choose a different display, right click in the calendar heading area, below the menu and above the calendar.

Select ‘Change View To’ and select from the following:

  • Month View – Displays one Campaign and up to three of its Events at a time in a monthly view. The vertical scroll bar changes the month and the horizontal scroll bar changes the Campaign

  • Gantt View – Default for Event Calendar as shown previously.

  • Agenda View – Display Campaigns and Events in a daily view
