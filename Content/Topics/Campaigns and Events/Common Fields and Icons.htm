Common Fields and Icons

There are some fields and icons that are available on this screen regardless of what is selected in the Event Navigation Pane.

Field

Description

Event (Read Only)

The name of the Event.

Campaign (Read Only)

The name of the Campaign associated with the Event.

Ad Date Range (Read Only)

The Ad Start and End Dates of the Event.

Sale Date Range (Read Only)

The Sale Start and End Dates of the Event.

Event Status (Read Only)

The current Workflow Status of the Event. This may affect what actions the user is permitted to perform on the Event. The Event Slot Sets inherit the current Event Workflow Status, unless a Slot Set has had its Status temporarily overridden to provide user permissions for a limited time period using the Slot Set ‘Status Override’ button in the Slot Set Pane.

Event Schedule (Read Only)

By clicking on the calendar icon, beside Event Status, the entire Event Workflow schedule is displayed in a window, including Status projected dates and current Slot Set Status States.

Messages

This icon is green if there’s a message available for the current Event Status.
