Event status change

Depending on the Event Workflow Transition Rights configuration, the user may have the ability to manually advance the Event Status.

To manually change the Event Status, click on Change Status. Select the new Status from the dropdown list, if needed enter a Message for the next person working on the Event and click Ok. The Message icon will turn green to indicate a Message has been entered.

The Slot Sets will inherit the new Status even if the Slot Set Status State is not ‘Complete’ or is currently overridden. The Status State will be reset to ‘In Progress’ for all Slot Sets.

The Event Workflow will advance to the next Status and the Event Schedule will reflect the Status change.
