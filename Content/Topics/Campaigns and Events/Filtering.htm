Filtering

There are multiple ways to filter the list of Campaigns and Events in this screen. Any of the methods listed below may be used to filter Campaigns and Events.

 

Action

Description

Campaign Selection

The simplest way to filter Events is to select a Campaign from the Campaigns Grid and only the associated Events will be displayed below in the Events Grid.

Show All

By clicking on this button, the user can toggle between displaying all Campaigns or only Campaigns that are associated with the Events in the Events Filter.

Events Filter

To filter the list of Events, click Filter and the system will display the Events Filter window. The user may select from an existing Filter or click on the black hammer icon to create a new or modify the current Filter.

 

Auto Filter Row for Campaigns

The list of Campaigns may be filtered by using the auto filter row above the Campaigns grid. The filter will refine the list of Campaigns that begin with the string entered in the filter.

Auto Filter Row for Events

The list of Events may be further filtered by using the auto filter row above the Events grid. The filter will refine the list of Events that begin with the string entered in the filter.

Filter Events by Campaign Name

By selecting the down arrow at the bottom of the screen, the user can select a Campaign and all associated Events are displayed in the Events Grid.

By deselecting the checkmark, all Events are displayed regardless of which Campaign they belong to. Clicking on the ‘x’ will hide the Filter Events by Campaign Name functionality until Event Manager is restarted.

Edit Filter

This action will open a Filter window with the selected Campaign Name defaulted in the Filter Editor.
