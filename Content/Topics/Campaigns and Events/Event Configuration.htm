Event Configuration

It is almost impossible to provide an out-of-the-box configuration that would work for all customers in terms of permissions, event types, slot set (e.g. event page) templates and geographies. JustEnough provides the following areas for you to configure these items to your specific needs.

Permissions, policies, and content configuration

  • Users screen: where you can configure user and role-based access to event-related features and functions.
  • Policies screen: where you can configure global policies related to events.
  • Content Manager Configuration window: where you can configure content fields and content versions.
  • Cluster Site Manager screen: where you can configure geography sets.

The Event Configuration window

The Event Configuration window is where you can configure the following event-related items:

  • Spaces

  • Price Display Formats
  • Attribute Display Formats
  • Item Display Formats
  • Slot Set Templates
  • Cost Types
  • Event Types
  • Geographies and sites

To open the Event configuration window, click Configuration from the Events screen toolbar.
