Price Display Format Properties

Price Display Format properties appear in the Edit Price Display Format window.

Getting there

  1. From the Plan tab, click Events.
  2. In the toolbar, click Configuration.
  3. In the Event Configuration window, click the Price Display Formats tab.
  4. In the Price Display Formats tab, select a strategy name and click Edit.

Modifying properties

You can view and modify the properties of a price display format by clicking a container in the Edit Price Display Format window. Price display formats can include multiple containers, so be sure the correct one is selected before modifying any fields.

  • Click Undo and Redo on the toolbar as needed to undo/redo any unsaved changes.
  • Click Save on the toolbar to save any changes made.

Property

Description

Name (Read Only)

 The name of the price display format is the same as its associated promotion strategy.

Units (Required)

The unit of measure used (centimeters or inches).

Width (Required)

 The width of the selected container. The size of the box within item display formats should be taken into consideration when determining the appropriate dimensions of the price display format. The minimum width allowed is 6 DIUs (about 1/32 inch).
Height (Required) The height of the selected container. The size of the box within item display formats should be taken into consideration when determining the appropriate dimensions of the price display format. The minimum height allowed is 6 DIUs (about 1/32 inch).
Font (Required) The font used for text (including price fields) in the selected container.
Size (Required) The font size used for text (including price fields) in the selected container.
Color (Required) The font color used for text (including price fields) in the selected container.
Order Number (Optional) An order number can be set up for each container in a price display format. The order number determines the order in which price format containers appear in reports. Reports are generally configured by Professional Services and should not be modified without assistance from JustEnough Professional Services or Support.
Price Field (Optional) To add a price field to a container, select the field from the Price Field list and click the down arrow. The selected price field will be appended to the container. You can edit the location of the price field in the Text Preview box.
Text Preview The text displayed in the selected container. You can add, delete, edit, or move text in this box and the changes appear automatically in the price display format preview.
