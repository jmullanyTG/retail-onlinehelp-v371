Versions

A Slot Set Version is the result of a combination of Promotion, Content and Pricing assigned to Geographies. Each Version consists of a distinct set of content and pricing across one or more Geographies.

To view each Slot Set Version, select Versions.

Slot Set Versions are displayed as across the bottom of the screen; each Base with a letter identifier, such as Base A, Base B, etc. By selecting a Base Version from the bottom of the screen, the Slot Set display, Geography pane and Site Names pane will refresh to display the associated Version information.
