Buttons

When a Slot Set is selected in the Event Navigation Pane, the same buttons that are available if the Event is selected, are available as well as some additional buttons. User Rights and Event Workflow permissions for the current user may affect which buttons are available.

 

Action

Description

Save

Allows the user to save any changes that have been made to the Event Information screen.

Clone

Provides the ability to create a copy of the current Event. All associated Promotions will be duplicated for the new Event.

Edit Header

Provides access to modify Event Header details including Campaign, Workflow, Ad Dates and Sale Dates.

Change Status

Provides access to change the Event Status based on the associated Workflow. The user may not have access to this functionality depending on the Event Workflow Transition Rights configuration. For more information, see Event Status Change.

Slot Set: New

New appends a new Slot Set to the Event using the default Slot Set Template defined in the Event Global Policies. The Slot Set is also appended to the first Event Composition.

Note: The label for this button will change depending on the Event Type’s configuration.

Slot Set: Delete

Delete removes the selected Slot Set from the Event. All Promotions that were placed on the Slot Set will be disassociated from the Event but not deleted.

Note: The label for this button will change depending on the Event Type’s configuration.

Promotion: New

If a Slot, has one or more Geographies available and the user has the appropriate Workflow permissions and rights, the ability to create a new Promotion is available. Clicking on New will open the New Promotion Wizard window. Enter the required fields and click Save and Close. For more information, see Create a New Promotion.

Promotion: Remove

If a Promotion is selected in the Slot Set, the Remove button is available. Click on this button to remove the selected Promotion from the Slot. This will only remove the selected Promotion. Any other Promotions will still be associated with the Slot. A different Promotion may be added to the newly open Geographies in the Slot.

Promotion: Remove All

If a Promotion is selected in the Slot Set, the Remove All button is available. Click on this button to remove all Promotions from the Slot. The Slot is considered open after this action.

Promotion: Unassign

If a Promotion is selected in the Slot Set, the Unassign button is available. Click on this button to move the Promotion from an assigned Slot to the Unassigned Position 0. The Promotion will move to Slot 0 and still be associated with the Slot Set.

View

If a Promotion is selected in the Slot Set, the View button is available. Click on this button to view the Promotion in Promotion Manager. If the user has the appropriate rights, modifications can be made to Promotion and saved. Changes will be reflected in the Event after the Slot Set is refreshed.

Content

For any Promotion in an Event, it is possible to modify the Content and Images associated with it using the Content Editor. For more information, see Content Editor.

List

To view the Slot Set in a text based list view, select List. Access to this screen depends on the user’s rights. For more information, see List.

Layout

To view the Slot Set in a more visual view with Images and formatted text, click Layout. Access to this screen depends on the user’s rights. For more information, see Layout.

Versions

To view all versions of the Slot Set that may occur due to variations in Promotions, Content or Pricing for one or more Geographies, click Versions. For more information, see Versions.

Views

This button will display a customized area at the bottom of the screen that is configured for the Event Type.
