Merging Objects

Merging combines two adjacent objects into a single object. It is available to the following object types:

  • Price display format containers
  • Attribute display format containers
  • Item display format containers
  • Slot set templates
  • Event slot sets

Merge options

The Merge toolbar menu contains the following options:

  • Merge Up
  • Merge Down
  • Merge Left
  • Merge Right

Notes:

  • Merge options are only available when there is at least one another object adjacent (above/below or to the right/left) to the selected object.

  • You can choose a merge option from the Merge menu on the toolbar or by right-clicking an object and choosing the option from the context menu.

  • Only the merge options that apply to a selected object will be enabled. For example, in the screenshot above, only the Merge Right option is available because the only other adjacent container is located to the right of the selected container.
  • You can merge one object with multiple other objects if the other objects combine to share the same border width or height as the original object. (See example below.)
  • WARNING: When merging, only the content in the originally-selected object is preserved (see examples below).

Examples

The following examples are shown using price display formats, but apply to the other supported objects as well.

Merging equally-sized objects

  1. Select an object with at least one other object adjacent to it. The selected object is outlined in blue.

    Note: A purple line along the selected object's border indicates that merging in that direction is possible.

  2. Select an available merge option.

  3. The objects merge. Note that the merge removed the content that was in the adjacent object, and only the selected object's contents and properties are preserved.

Merging multiple adjacent objects

In this example, we will merge a container with two other containers that, combined, share the same height as the selected container.

  1. Select an object that has multiple objects along its entire top, bottom, or side. In this example, the selected object is adjacent to two horizontally-stacked containers that combine to be the same height as the selected container.

    Note: The purple line along the selected object's right border indicates that merging in that direction is possible.

  2. Select an available merge option.

  3. The objects merge. Note that the merge removed the content that was in the adjacent objects, and only the selected object's contents and properties are preserved.
