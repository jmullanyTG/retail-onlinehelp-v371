Renumbering Slots

Slots in a slot set layout are automatically assigned slot numbers. You can select a slot to see its assigned slot number in the read-only Slot Num property.

  • In the Edit Slot Set Template window, Slot Num is included in the property list.

  • In the Layout view's Layout Edit Mode, Slot Num is a property in the Slot Information pane.

Instructions

  1. Above the slot set preview, click the # button next to the Zoom slider to turn on the slot numbering feature.

    Numbers will appear over each slot and the slot properties will be grayed out.

  2. Click on the slot you want to be slot number 1. The rest of the slots renumber themselves accordingly

  3. Clicking on additional slots renumbers them in sequence (slot number 2, slot number 3, etc.).

    Note: Click Undo or Redo on the Slot toolbar as needed.

  4. When you are done renumbering slots:

    • Click Save on the Event toolbar to save your changes
    • Click the # button again to turn off the numbering feature and return to normal functionality.
