Module: Supply Chain

Display Name System Name Description
Supply Chain Default Active Supply Chain ID SupplyChain.DefaultActiveSupplyChainID The ID of the default supply chain
Supply Chain Garbage Collection Max Minutes Allowed for Processing SupplyChain.GarbageCollection.MinutesUntilTimeout The maximum number of minutes that garbage collection can be performed. Default value = 60.
Supply Chain Garbage Collection Max Number of Deleted Supply Chain to Process at a Time SupplyChain.GarbageCollection.DeletedEntriesBatchSize

The max number of deleted supply chains to process at one time.

Default value = 100.
Supply Chain Garbage Collection Minutes to Wait Before Removal of Deleted Entries SupplyChain.GarbageCollection.MinutesToWaitOnDeletedEntries

The max number of minutes JustEnough waits after a supply chain is deleted to remove the entry plus related entries from the system.

Default value = 129600 (90 days).
Supply Chain Override With From Supplier Flag SupplyChain.OverrideSupplyChainUsingFromSupplierFlag When set to True, the supply chain hierarchy is ignored for inventory that has a From Supplier Flag set to True. However, supply chain overrides are always used, if defined.
