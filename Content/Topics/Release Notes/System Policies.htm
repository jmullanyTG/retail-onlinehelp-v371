Engine: Replenish

Display Name System Name Description
Order Processing Lead Time Type Replenish.OrderProcessingLeadTimeType

Sets whether the Order Processing Lead Time value specified in an inventory policy is applied according to calendar days or business days.

Example: if an inventory policy specifies Order Processing Lead Time = 3, this means 3 days are required after delivery to process an order. If this policy is set to 0 (Calendar days), then exactly 3 days will be allotted to process the order. If this policy is set to 1 (Business days), then the actual number of days allotted to process the order may be greater than 3 to account for days the business is scheduled to close (weekends, holidays, etc.).

  • 0 = Calendar days.
  • 1 = Business days. Business days are counted according to the Order Processing Lead Time Schedule set for a site or item.

Default value = 0
