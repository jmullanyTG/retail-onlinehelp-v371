Price Manager

The following enhancements are now available in the Price Manager module.

Site cluster sets replace site filters

Site cluster sets have replaced site filters as the method of selecting sites for pricing plans and price actions. (See the related Cluster Site Manager enhancement for making site cluster sets eligible for selection in Price Manager.)

This change is reflected in the following areas of the Price Manager module:

Pricing plan changes

  • New Pricing Plan dialog box - The Site Filter field has been replaced with a Site Cluster Set field.

  • Information tab header - A pricing plan's Information tab now includes a Site Cluster Set field in the header.

  • Sites grid - A pricing plan's Sites grid now displays the sites included in the selected site cluster set. You can choose to further exclude/include individual clusters and/or sites from this grid.

Price action changes

  • Create New Price Action dialog box - The Site Filter field has been replaced with a Site Clusters field. You can now select individual clusters from the pricing plan's site cluster set to apply to the price action.

  • Price Action Sites grid - A price action's Sites grid now displays the individual sites in the clusters selected for the action. You can choose to further exclude/include sites from this grid.

    Migration of site data from existing plans and actions

    If upgrading from a previous version of Demand Management, site data from existing pricing plans and actions will be migrated to a new default site cluster set during the upgrade process. For more information, see the Demand Management Installation Guide.
