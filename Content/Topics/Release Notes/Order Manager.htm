Order Manager

The following Order Manager enhancement is now available.

Available Date fields

Available date refers to the date on which an ordered product is expected to be "in stock" and ready for sale/distribution at the receiving site or DC. (Availability is counted at the immediate start of the day, before any demand is drawn off the available stock.) Available date differs from delivery date in that it accounts for any order processing time that occurs after the delivery date but before the product is considered in-stock.

When generating forecasts, lost sales projections are now calculated against available date, not delivery date.

Calculation

In 3.7.1, available date is calculated as "Delivery Date + Order Processing Lead Time Days".

"Order Processing Lead Time Days" is based on the Order Processing Lead Time value of the order's governing inventory policy. It is resolved using either calendar days or business days, depending on the Order Processing Lead Time Type system policy.

Examples

The following examples show how order processing lead type affects the available date of an item. Both examples have the same delivery date and order processing lead time, but Example 1 uses calendar days to resolve lead time days and Example 2 uses business days. (For Example 2, assume that Saturdays and Sundays are not scheduled business days.)

Because Saturdays and Sundays are counted as processing days in Example 1 but not in Example 2, the available date in Example 1 is 2 days earlier than in Example 2.

Delivery Date Friday Feb 1 Friday Feb 1
Order Processing Lead Time
(inventory policy value) 		3 days 3 days
Order Processing Lead Time Type (system policy setting) Calendar days Business days
Scheduled business days Monday - Friday Monday - Friday
Available date calculation Friday Feb 1 + 3 calendar days Friday Feb 1 + 3 business days

Fields

The following Available Date fields are now available for display on the Order Manager grid.

Field Name Description
User Available Date A date entered by a user to override the Recommended Available Date and Constrained Available Date values.
Recommended Available Date The available date calculated by the ordering engine before any constraints are applied. This date is derived from the delivery date, order processing lead time, and order processing lead time type.
Constrained Available Date The recommended available date after factoring in ordering constraints.
