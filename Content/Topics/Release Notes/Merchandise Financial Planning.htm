Merchandise Financial Planning

The following enhancements are now available in the Financial Planning module.

Consolidation of planning views

A single new planning view replaces the 3 planning views (Main Planning View, Comp/Non-Comp Planning View, and Sales By Type Planning View) that appeared in previous versions. All fields from the previous views are available in the new Hybrid Financial Planning Grid.

Auto Lock on Cell Change

You can now have sales-related cell values automatically lock themselves after you update them. When the new Auto Lock on Cell Change is enabled, each time you change the value of a sales-related metric in the grid, the system automatically locks the changed cell as well as its related sales cells. This saves you from having to stop and lock each cell after each change, which is especially convenient when you want to to change a number of values in a plan.

The Auto Lock on Cell Change option is located in the Calculation Direction Toggles pane. Simply click its checkbox to toggle enabling and disabling the option.

Save/Restore Data Layout

To complement the existing Layout feature, which saves and restores the pivots, filters, and data fields displayed in the Hybrid Financial Planning Grid, a new data layout feature is now available.

The Save/Restore Data Layout context menu option lets you save and restore only the fields used as metrics in the data area of the planning grid, independently of the pivot fields used to aggregate the data into rows and columns. This lets you easily switch metrics in a layout without having to recreate your current product/location/time dimensions and filters.

Data layouts take the pain out of switching between different areas of a financial plan. For example, you can choose a layout that contains the pivots (channel, product, and time periods) and filters for the areas of the plan you are responsible for. As you work on your plan, if you want to see a different set of metrics while preserving the current pivots and filters, you can simply select a new data layout and only the data fields displayed in the grid will change. Your workspace, pivots, and filters will persist until you choose to change them. This versatility lets you use layouts to drive your work, making it easy to switch scenarios without sacrificing time or convenience.

New folder structure for organizing saved layouts

Using different layouts for different tasks can make working on financial plans much easier. Now you can better organize your layouts by saving them in folders.

When saving a layout, simply choose the group you want to save the layout in.

The next time you want to use that layout, you can choose it from that group in the Restore Layout context menu.

Note: Creating a layout group requires administrator privileges.

Supply chain validation behavior

Supply chains are now validated on master plans only, and their validation status is passed down to branch plans.

Also, the Supply Chain Status value is now colored red when the supply chain fails validation. This replaces the warning message that would pop up when opening a plan in previous versions.
