Event Manager

The following enhancements are now available in the Event Manager module.

Slot renumbering improvements

  • You can now undo and redo slot numbering changes by clicking Undo and Redo on the toolbar.
  • The UI now asks if you want to save or cancel changes when exiting the Slot Renumbering feature.

Resizing slots

You can now adjust the borders of individual and multi-selected slots to create custom-sized slots on a slot set.

Note: Only one border can be re-sized at a time.

Resizing a single slot

  1. Click in a slot to select it. The "resizable" borders are indicated in purple.

  2. Hover your mouse pointer over the border you want to resize. Wait for the mouse pointer to change to the "resize" cursor and then click-and-drag the border to its new position.

  3. Release the mouse to resize the slot.

Resizing multiple slots

You can resize multiple slots at once as long as the slots are adjacent and aligned across at least one border.

  1. Hold the Ctrl button while clicking in the slots you want to select. Slot borders turn orange when a slot is selected.

  2. Hover your mouse pointer over the shared border. Wait for the mouse pointer to change to the "resize" cursor and then click-and-drag the border to the desired position.

  3. Release the mouse to resize the selected slots.
