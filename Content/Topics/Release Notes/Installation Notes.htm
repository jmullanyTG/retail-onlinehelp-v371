Installation Notes

The upgrade process has changed for version 3.7.

Refer to the Demand Management Installation Guide for detailed instructions and information related to upgrading from prior versions (3.6.2 and earlier) to 3.7.x as well as installing new instances of 3.7.x.

Minimum requirements

Software

.Net Framework 4.7.1 and Dot Net Core 2.1.3 are required on the Application Server and on Demand Management clients. These can be installed with the services or downloaded and installed from Microsoft.

Application Server

Supported operating systems:

  • Windows Server 2016 (recommended)
  • Windows Server 2012 R2
  • Windows Server 2008 R2 SP1 (not recommended)

Database Server

Supported operating systems:

  • Windows Server 2016 (recommended)
  • Windows Server 2012 R2

SQL Server:

  • SQL Server 2017 (recommended)
  • SQL Server 2016
  • SQL Server 2014 (not recommended)

Workstation

Supported operating systems:

  • Windows 10
  • Windows 8.1
