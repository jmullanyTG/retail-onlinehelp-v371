Profile Set Item Details

The profile set item details will show a graphical representation of the data in five standard data series.

  • Effective Profile Volume – volume per period using the profile curve listed (effective periods only).
  • Sample Profile Volume – Sales volume per period using the profile curve listed (sample periods only).
  • Sales History Quantity – Actual sales history.
  • Profile Volume Override – Sales volume per period using the override adjustments.
  • Profile Volume – Sales volume per period using the profile curve.
