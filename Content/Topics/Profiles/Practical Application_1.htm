Practical Application

Adding and removing profile set items is done once the profile set is open in a tab using the “add” and “remove” buttons.

“Remove” will remove the highlighted item from the set. “Add will launch the “add items” window.

Selecting “choose curve” will bring up the curve manager window. Here, users should select the curve to be used and hit “ok”.

Once the curve is selected, users should enter the date ranges for the profile start, profile sample, profile effective, and value for the profile evaluation period. 

Once completed, users should select “next”.

Next, users will need to enter the group types for the profile item. Depending on how groups have been configured, this will vary from one implementation to another.

The included or excluded groups should be managed for each group type. In the given example, certain site regions are included.

Once completed, hit “ok”, then “next” on the group types window.

Once the group types have been set, users may then add the individual profile set items.

After clicking “finish” the profile set items will be populated.

Each item line can be modified for dates and volumes. The effective dates are the time period during which the profile curve will be applied to the projected demand.
