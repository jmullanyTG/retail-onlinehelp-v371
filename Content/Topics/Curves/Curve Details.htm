Curve Details

The curve details section will provide a graphical representation of the curve series values, as well as a familiar grid view.

Essential Business Concepts

Series values in the curve details will be presented both in a graphical format and in a grid.

  • Bucket – Associated time period (buckets) for the x-axis of the line graph will be displayed in the grid.
  • Calculated – The system generated value for each period within the curve.
  • Effective – If manual adjustments are made to the curve, they will be represented as a value in the “effective” series. This may then be compared to the static “calculated” series.
  • Build – The build value will represent a multiplier for the period value’s relationship to the previous period value. This lets users see how drastically the curve values change. This is not applicable to the first period point in the curve (anchor point).
  • Contribution % - The contribution % will be a percentage for each period of the representative value of the effective value when compared to the whole (e.g. a flat line curve with 10 periods would have a contribution of 10% for each period).
  • Period to Mean – The period to mean y-axis shows a graphical representation of the “distance” to the mean by period.

Details

The details button will toggle the build and period to mean y-axes.

Practical Application

Interaction with curve values in the curve detail section is limited to two actions: manipulation of the effective series and the build series values.

Curve are graphically represented using an interactive line graph. Values may be manipulated by dragging one of the “dots” up or down. The curve values are re-calculated simultaneously.

Grid values will also change dynamically if manual adjustments are made to the curve.
