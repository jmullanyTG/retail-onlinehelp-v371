Inventory Supply Chain Resolution Task

The Inventory Supply Chain Resolution task reprocesses supply chains when added to nightly or weekly jobs.

Because supply chain overrides are filter dependent, be sure to run this task AFTER imports and filter-rebuild tasks and BEFORE replenish/order tasks.

Task parameters

Parameter Description
Name The task name.
Description The task description.
Progress Type

What to do if an error occurs when processing this task:

  • Report Error And Continue
  • Report Error And Halt
Time Out (Seconds) The timeout period.
Retry Count The number of retry attempts to make if a timeout occurs.
Retry Invterval The time to wait in between retry attempts.
Task Enabled

Whether or not this task is active or disabled.

  • When selected, the task is enabled and will be run if manually executed or part of a scheduled job.
  • When not selected, this task is disabled and will not be run even if manually executed or part of a scheduled job.
Work Item Filter Type

The type of supply chains this task will process:

  • All supply chains
  • Only supply chains with exceptions
  • Specified supply chains
Supply Chain(s) Visible only when Specified Supply Chains is selected as the Work Item Filter Type value. Lists all of your supply chains. Those selected will be processed by this task.
