Practical Application

Accessing the widget configuration screens is done through the “configure” tab in the navigation pane of the view module.

Clicking on “maintain widgets” will bring up the maintain widgets window.

Within the maintain widget window, users may choose to create a new widget, or edit an existing one. Users may highlight an existing widget and click “preview” to bring up a working version of the widget. Widgets may be imported and exported via the user interface as well (detailed in technical execution).

Application of widgets will vary greatly depending on the widget type.
