Presentation

The presentation section determines the details and additional features of the chart.

  • Chart – Allows users to affect the appearance of the chart, including background colors, fill styles, borders, and other settings.
  • Diagram – The diagram section will allow users to manipulate the layout, rotation, scrolling, and zoom functions of the chart.
  • Panes – Panes will frame the chart. This feature may not be available for all chart types. 
  • Axes – The axes settings will determine if and how the chart axes are displayed. This feature may not be available for all chart types. 
  • Series Views – Series views will allow users to manipulate how the individual series in the chart are displayed and treated. 
  • Point Labels – The point labels section will affect how the data points are graphically represented within the chat. If it is intended for the values to be presented as part of the chart, users must determine how they would like to view that information. 
  • Chart Titles – Chart titles allows users to add a title to the chart. This may be unnecessary as the widget name is typically displayed at the header level of the container. 
  • Legend – If the chart requires or would benefit from the use of a legend, this may be added using the legend settings. 
  • Annotations – Any required annotations may be added using the annotations settings.
