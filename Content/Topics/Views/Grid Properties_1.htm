Grid Properties

General Options

  • View Mode – The view mode will determine how the data is displayed.
  • Grid – The grid view is a spreadsheet type view consisting of rows and columns.
  • Layout – The layout view may be used to construct the data grid as a form (or card).
  • Toggle – The toggle mode allows users to switch between the grid and layout types.
  • Show Embedded Navigator – The embedded navigator facilitates ease of navigation through the data.

  • Embedded Navigator Format String – The syntax in the embedded navigator may be formatted at the widget level to accommodate special conditions.
