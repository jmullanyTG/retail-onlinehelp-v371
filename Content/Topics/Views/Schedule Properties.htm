Schedule Properties

The schedule properties will define the layout of the schedule and how users are allowed to interact with it.

  • Resource Name Caption – Name given to the schedule.
  • Resource Tree Visibility – If selected, the resource tree will be visible.
  • Auto Node Height – If selected, will automatically generate the height for the node.

Use Today as Go to Date – If selected, today’s date (GETDATE) will be used as the go to date.

  • Go to Date – The date which will be presented when the widget is launched.
  • Show Resource Headers – If selected, the resource headers will be displayed.
  • Resource Headers Height – Height for the resource headers may be hard coded.
  • Resources Per Page – Will limit how many resources are displayed per page. When the widget has been launched, this may be adjusted on the fly using the plus and minus arrow keys.
  • Cells Auto Height – Will automatically assign a height for the cells.
  • Appointment Auto Height – Will automatically assign a height for the appointments.
  • Appointment Height – If not set automatically, appointment height may be specified.
  • Show More Button – In cases where the data does not fit well into the height and width constraints, the “more” button will alert users to the fact that there is data present which is being hidden.
  • Default Time Scale – The default time scale will determine level of granularity which is initially visible within the time dimension. This may be set at:
  • Year-Quarter
  • Quarter-Month
  • Month-Week
  • Week-Days
  • Day-Hours
  • Hour-Time
  • Default View – The view for the schedule.
  • Gantt View – The Gantt view presents the schedule as a series of bars with interdependencies.
  • Month View – The month view looks similar to a regular calendar view.
  • Months Per Page – Sets the number of months displayed when in month view.
  • Week Count – Will show the number of weeks when in month view.
  • Cell Colors – The cell colors section will allow users to change the foreground (text) and background cell colors for certain sections of the schedule.
