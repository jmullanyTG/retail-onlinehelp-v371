Column

  • Column – The column name.

Band Settings

  • Band Type – Determines if the column is of type “pivot”, the default band type will place the column in the center band. If the column is of type “non-pivot”, and does not contain the “JESUM” expression, then the column will be placed in the left band. If the column is of type “non-pivot”, and does contain the “JESUM” expression, then the column will be placed in the right band. Band types may also be manually set to determine the band placement.
