Create/Edit Parameter

Several fields must be supplied in order to create a valid parameter, including some knowledge of the SQL parameter (which will not be available anywhere within the user interface).

  • Parameter Name – The name for the parameter.
  • Parameter Description – An optional description of the parameter.
  • Parameter Caption – The name which will be presented as part of the user prompt to supply a parameter value.
  • Parameter SQL Name – The linked SQL parameter. Name must match the name in the database.
  • Parameter is Visible – If checked, the parameter will be visible to users.
  • Parameter Width – The specified width of the prompt box.
  • Parameter Height – The specified height of the prompt box.
  • Parameter Default Value – The default value supplied without user override.
  • Parameter Type – Defines the how the value or values may be passed to the parameter. Options include single and multi-selects.
  • Parameter Data Type – The associated data type.
  • Allow Null – If selected, will permit a null value to be returned to the parameter.
