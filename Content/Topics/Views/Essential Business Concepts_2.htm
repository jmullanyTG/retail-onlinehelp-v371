Essential Business Concepts

There are two types of parameters which may be managed through the parameter maintenance screen: system parameters and user parameters. System parameters will not be visible to the user; it is possible that system parameters may have values injected into them (by .NET code) such as user ID, and session ID. It is not advisable to alter system parameters.

User parameters operate at the user level and may be built to create consistency throughout the application, or at minimum, across widgets.
