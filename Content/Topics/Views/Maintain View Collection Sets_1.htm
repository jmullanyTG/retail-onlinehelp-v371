Maintain View Collection Set Window

The Maintain View Collection Set window is where you can manage groups, view collection sets, and view collections.

It contains a toolbar and a grid showing the full tree structure of existing groups, view collection sets, and view collecitons.

Getting here

To open the Maintain View Collection Set window:

  1. From the Plan tab, click Views.
  2. On the Views toolbar, click View Collection Sets.

Toolbar

The toolbar contains the following actions.

Action Description
Group
New Adds a new group.
Edit Edits the selected group.
Delete Deletes the selected group.

View Collection Set

New

Creates a new view collection set.

Edit

Edits the selected view collection set.

Delete

Deletes the selected view collection set.

Import New

Imports a new view collection set (.jevcs file).

Import Existing

Imports an existing view collection set (.jevcs file).

Export

Exports the selected view collection set to a .jevcs file.

View Collection

New

Adds a new view collection. View collections may only be created within a view collection set.

Edit

Edits the selected view collection.

Delete

Deletes the selected view collection.

Remove

Removes the selected view collection.

 

Preview

Shows the current layout of the selected view collection set or view collection.

Refresh

Refreshes the data in the grid.

Grid

The grid displays the current hierarchy of groups, view collection sets, and view collections. In the grid, you can "drag-and-drop" objects up and down in the hierarchy.
