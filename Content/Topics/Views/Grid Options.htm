Grid Options

  • Data Field Area – Specifies the location, or lack thereof, of the data field. It is common to leave this as “none”.
  • Horizontal Scrolling – Determines how the grid may be scrolled horizontally.
  • Cells Area – Will enable the scroll bar to allow for horizontal scrolling through the data cells only.
  • Control – Will allow users to scroll horizontally through the entire widget, including the control area.
  • Show Horizontal Lines – Similar to Excel, will toggle the horizontal row lines on and off.
  • Show Vertical Lines – Similar to Excel, will toggle the vertical column lines on and off.
  • Totals Location – Determines the location for totals columns.
  • Near – Will display the totals column to left of the column field values.
  • Far – Will display the totals column to left of the column field values.
  • Allow Compact View – When checked, this value will allow users to create compact layouts for hierarchical pivot table row values thus creating more space horizontally without sacrificing the distinction between the hierarchical row values.
  • Show Filter Drop Area – Will display the drag-and-drop area to add fields to the “filter” area.
  • Show Data Drop Area – Will display the drag-and-drop area to add fields to the “data” area.
  • Show Column Drop Area – Will display the drag-and-drop area to add fields to the “column” area.
  • Show Row Drop Area – Will display the drag-and-drop area to add fields to the “row” area.
  • Show Defer Rolling Actions –Presents button to prevent rolling actions
  • Pivot Level Calculated Fields Use Summary Values For Calculations – If selected, the pivot grid will use the aggregated level values for the calculation. If not selected, calculations will be performed at the lowest level and aggregated using the aggregation type (e.g SUM, AVG).
  • Editor Show Mode – Will specify how an in-place editor is activated for a cell within the grid.
  • Enable Word Wrap – If enabled, the text will wrap within the cell (mostly used for column names).
  • Enable Runtime Expression Edit– If enabled, right clicking on a row within the grid will allow users to view the expression.
