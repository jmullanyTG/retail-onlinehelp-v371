Lookup Settings

  • Lookup Column – Will be checked if the column is a lookup column.

Actions

  • Allow Actions for Column – If selected, will display the action button.
  • Action Editor Style Type – Determines the action button options which will be displayed to end users.
  • Display Editor with Action Buttons – Will provide potential action edit functionality to end users.
  • Display Only Action Buttons – Will display only the action button and not the editor.
