Update Options

  • Confirmation Message on Update – If checked, will pop a confirmation message if the data has been updated by the user.
  • Confirmation Message – The text of the confirmation message. Will default to “Save Changes?”
  • Refresh Data on Update – If checked, the data will update when changed. This option should be checked by default.
  • Refresh Widget Siblings – Must be checked in order for dependent widgets within the view require data updates.
