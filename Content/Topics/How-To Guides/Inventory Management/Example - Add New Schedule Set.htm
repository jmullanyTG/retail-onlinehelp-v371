Example - Add New Schedule Set

In this example, we will add a new schedule set that allows 7 day a week delivery to the New York City store.

Related video

This video shows how to create and apply a "weekday delivery only" schedule set.

Instructions

1. Create a schedule set and add a schedule that allows daily deliveries.
  1. From the Maintain tab, click Schedules.
  2. In the Schedule Set toolbar, click New.
  3. In the Name field, enter "7 Day Delivery".
  4. Click Save to save the schedule set.
  5. In the Schedule toolbar, click Add.
  6. Enter the following schedule details:
    • Name: Daily Deliveries
    • Frequency: Occurs: Daily
    • Start Date: Choose today's date

  7. Click Save to add the Daily Deliveries schedule to the 7 Day Delivery schedule set.

  8. In the Schedule Set list, expand the 7 Day Delivery schedule set to show the Daily Delivery schedule. In the Daily Deliveries schedule row, set the Mode column value to Include. This instructs JustEnough to include deliveries on the days specified in the schedule.

  9. Click Save in the Schedule Set toolbar.
 2. Apply the new schedule set to items at the New York City store.
  1. From the Manage tab, click Item Manager.
  2. If the Items dialog box does not automatically open, click Inventory Manager Document to open it.
  3. In the Items dialog box, select the Receive Schedule: Week Day Delivery Only filter. (This filter limits item/site combinations to items at retail stores).

  4. In the Choose Fields dialog box, choose Receive Schedule.

  5. In the new inventory management document, filter the Site Name Column for New York City. Now, only items for the New York City store are displayed.

  6. To change the receiving schedule for the items:
    1. Select all rows in the document.
    2. Right-click in the Receive Schedule column and choose Multiple Changes.
    3. In the Multi Data Change Form's new value field, choose the new schedule set, 7 Day Delivery.

    4. In the Name field, enter "7 Day Delivery" and click Save.

  7. The document now shows 7 Day Delivery as the receive schedule for items at the New York City store.

  8. Click Save to save the changes.
