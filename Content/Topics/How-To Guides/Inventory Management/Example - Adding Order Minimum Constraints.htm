Example - Adding Order Minimum Constraints

In this video example, we will add a new order constraint that sets a minimum order value for orders purchased from a specific vendor. Once set, the order recommendations that JustEnough generates for this vendor will meet the specified order minimum.

Related video

Instructions

  1. From the Manage tab, click Ordering Constraints.
  2. In the Constraints toolbar, click New.
  3. In the Add/Modify Constraint Set dialog box:
    1. Name the new constraint "$10,000 minimum".
    2. Click Add Constraint.
    1. In the Add/Modify Constraint dialog box:
      1.  Enter the following values:
        • Constraint so that: Default Cost Value
        • is: Greater than

        • the value: 10,000

      1. Click OK.
  4. Back in the Add/Modify Constraint Set dialog box:
    1. Select Groups of items within the selection
    2. Click Select Groups.

    3.  In the Constraint Groups dialog box, include the Supplier Name group and click OK.

    4. Click OK to save the constraint set.
