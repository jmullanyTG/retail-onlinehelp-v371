Planning Demand for a New Store

This process guides you through the process of planning demand for new stores. It spans site, supply chain, and item management to ensure that the sites, SKUs, supply chains, and model stores are set up for JustEnough to accurately forecast demand for the new stores.

Step Task Example More Infomation
1. Identify new stores and confirm opening dates. Example Site Manager Overview
2. Validate SKUs ranged to new stores. Example Item Manager Overview
3. Ensure new stores are in your supply chains. Workflow Supply Chain Overview
4. Create store supersessions. Example Site Manager Overview
5. Review demand forecasts for the new stores. Example Demand Forecasting Overview
6. Manage item supersessions as necessary. Example Supersessions Overview
