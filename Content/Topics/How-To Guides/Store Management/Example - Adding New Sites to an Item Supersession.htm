Example - Adding New Sites to an Item Supersession

In this example, we will add a new site, Calgary, to an item supersession to ensure that demand for the item at the new store is accurately forecasted.

Instructions

  1. From the Manage tab, click Item Manager.
  2. In the Items dialog box, select the Review Resultsfilter.

  3. In the item document, filter the Site Name column for Calgary.

  4. Double-click on an item to display its Details pane.
  5. In the Details pane, open the Forecast Viewer tab to see the demand that JustEnough forecasted for the item at the new Calgary store, based on the store supersession created previously.
