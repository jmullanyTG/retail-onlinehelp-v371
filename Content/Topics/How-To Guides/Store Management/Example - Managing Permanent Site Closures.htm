Example - Managing Permanent Site Closures

In this example, we will manage the process of permanently closing a store. Steps in this process include:

  • Identifying the store(s) planned for closure and setting or confirming the store closure date.
  • Managing the stocking strategy for the closing store.
  • Updating phase out dates.
  • Reviewing impact of changes and making any changes as required.
  • Redistributing any leftover inventory.

Instructions

1. Set or confirm the site closing date.
  1. From the Manage tab, click Site Manager.

  2. In the Items dialog box, select the (All Sites) filter.

  3. In the Choose Fields dialog box, select any additional fields you want to view and click OK. For this example, we will select Site Group and Site Status Temporal fields.

  4. In the site document, find the Cincinnati store and click on its row in the grid to view its details. (If the Details pane is hidden, click Details on the toolbar to show it.)

  5. In the Site Status tab, if there is not already a "Closed" status entry for the site, click New in the Entry toolbar to add a new status.

  6. Enter the following values for the status entry:
    • Site Status = Closed
    • Site Status Attribute = Closed
    • Status Start Date = select a date at least 30 days in the future.

  7. Click Save in the Entry toolbar to save the new status entry.

 2. Set or confirm the stocking strategy for the closing store.
  1. From the Manage tab, click Policy.
  2. In the Policy screen, open the Manage Store Closures policy.
    1. In the Policy tab, click Add Dynamic Policy in the Dynamic Policy area.
    2. Enter or confirm the following values for the dynamic policy and click OK:
      • Start Date = 2 weeks before the store closing date. (This assumes a two week replenishment cycle for the store, meaning this policy is in effect for the store's final replenishment cycle prior to closing.)
      • Policy Type = Replace
      • Set Minimum Safety Stock = 0

      • Safety Stock Days = 0

 3. Set or confirm the phase out dates for items at the closing store.
  1. From the Manage tab, click Item Manager.
  2. In the Items dialog box, select the Manage Store Closures filter, which finds items at the closing Cincinnati store.
  3. In the Choose Fields dialog box, select or create a view that includes the following field:
    • Phase Out Date
  4. If the Phase Out Date shown for the items does not match the store closure date, update the Phase Out Date field to reflect the store closure date.
 4. Identify leftover inventory.
  1. From the Manage tab, click Item Manager.
  2. In the Items dialog box, select the Identify Leftover Product filter, which finds items at all permanently closed stores.
  3. In the Choose Fields dialog box, select or create a view that includes the following fields:
    • Stock On Hand
    • Excess Stock Quantity
    • Excess Stock for Redistribution Quantity
    • Excess for Redistribution
    • Phase In Date
    • Phase Out Date
  4. In the Item Manager document, review the leftover items. This analysis can help you determine the best course of action for redistributing the leftover product. For example, you may conclude you need to refine your redistribution supply chain as a result, in which case you should do that before having JustEnough generate any redistribution orders for the leftover inventory.
 5. Redistribute leftover inventory.
  1. From the Manage tab, click Order Manager.
  2. In the Order toolbar, click Redistribution.
  3. In the Items dialog box, select the Identify Leftover Product filter and choose the Select only some items from the current filter option.

  4. In the Item Selection dialog box, filter for a closing site (such as New Orleans) and select all items returned for that site.

  5. In the Add Redistribution Order dialog box, enter the following parameters and click OK:

    • Supply Chain: [Redistribution Supply Chain]
    • Source Stock:Excess, Excess for Redistribution
    • Future Recommended Orders: Choose Today's Orders to generate redistribution orders based on today's need within the other stores in the New Orleans store's redistribution pool.

  6. Once JustEnough calculates the recommended orders, they appear in the Order screen. Each row represents a different recommended order, and contains basic details of the order.

Results

Once the store closure date, stocking strategy, and phase out dates have been set and confirmed for the closing store, JustEnough automatically recalculates replenishment strategies during its nightly processing to adjust for the closure. It will reduce stock as much as possible while ensuring there is enough stock left at the store to meet demand until the closing date.

You can review stock holding projections for store items in the Item Manager to review the impact of the changes you made during this process.
