Example -Validating SKUs Ranged to New Stores

In this example, we will review the items that have been ranged to new stores.

Instructions

  1. From the Manage tab, click Item Manager.

  2. In the Items dialog box, select the Review New Stores filter.

  3. In the Choose Fields dialog box, select any additional fields you want to view and click OK. For this example, we will not select any additional fields.

  4. In the item document, you can see all of the items that have been ranged for the new Calgary store.
