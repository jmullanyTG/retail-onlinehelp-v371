Essential Business Concepts

The Comp and Non-Comp planning view works exactly the same as the main planning view in terms of core functionality.  The purpose of this tab is to allow for the budgeting of growth and/or decline up and down the product and location hierarchies over time in order to be able to determine the sales activity in these segments of the business.  Comparative business is typically defined as the segment of the business that has been active for a period of more than 12 months.   Non-Comparative business is typically defined as the segment of the business that has only been active for 12 months or less.

The parameters selected in the main planning view are connected to the comp/non-comp planning view.  If a user makes any changes to the parameters defined in the header area, this will update for the main planning view as well as the comp/non-comp planning view.  Additionally, the calculation toggles and the locks that are invoked will be consistent with the selections made in the main planning view.

The user has the ability to change the sales metrics for the comp and non-comp groupings.  In making changes to either of the two groupings, this can have an effect on the main planning view and the total sales related to the plan.  Below is an example of how this works:

Total sales for the 2017_wk30 are $3,626.  If the user changes the Comp Sales metric for that period from $2,626 to $3,000, total sales will update in this tab and the main planning view simultaneously.

Comp Sales for the period 2017_wk30 are now $3000 and have impacted the total sales plan updating that value to $4000 for the same period.  The same behavior applies if the user makes changes to the Non-Comp Sales.
