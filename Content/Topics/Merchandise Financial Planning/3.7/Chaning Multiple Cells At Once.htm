Chaning Multiple Cells At Once

Users can make changes to multiple cells in a single action by selecting the desired cells and right clicking to bring up the context menu:

Click Multiple Changes to launch the Multi Data Change Form dialog:

Here, users can choose a number of different actions by which to manipulate the selected cells:

  • New Value: sets a new value with which to overwrite the values of the selected cells
  • Increase/Decrease: increases/decreases the selected cells by the specified amount or percent
  • Split Value: splits the entered value across the selected cells either as a weighted split based on the current cell values or evenly
  • Copy From: allows the user to choose a number of cells equal to the number currently selected to copy values from; click the Select Fields button, select the desired cells in the view, then click OK, and review the value mappings in the Results Preview section
  • Round new value: selects a rounding method for the new values – up, down, or nearest
  • Note: only applies when new values have decimals
