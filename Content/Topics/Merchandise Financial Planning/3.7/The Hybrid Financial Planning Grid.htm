The Hybrid Financial Planning Grid

The Hybrid Financial Planning grid is where you view and edit a plan's data.

Because it would be impractical to display an entire financial plan on screen at once, you can choose which part of the plan to display in the grid.

The plan data displayed in the grid is determined by:

  • The parameters you execute.
  • The layout you select.
  • The data layout you restore.
  • The fields you add or remove.

Cell Colors

Data cells in the grid are color-coded to indicate what type of metrics they represent and whether their values are locked.

Cell Color Description
Light blue Plannable metrics. These cells are editable. Their values represent the financial plan data.
Light purple

Comparison metrics. These cells are read-only. Their values are read directly from the comparison plan selected in the Parameters pane.
Dark Grey Actualized metrics. These cells are read-only. Their values come from the actual sales data that has been imported into Financial Planning.
Light green Ex-VAT metrics. These cells are read-only.
Light red

This cell value is locked. Light-red coloring is applied only to sales-related metrics. Other types of metrics do not change color when locked.

Aggregate-level values follow the same color-coding scheme, but use a gradient instead of a solid background. Examples:

  • - a locked aggregated sales-related value.

Toolbar

Actions available from the Hybrid Financial Planning Grid are described below.

Button Description
Unlock Removes the lock from the selected cell and its dependents, meaning their values can now change as a result of changes made elsewhere in the plan.. See Lock & Spread for more information.
Lock Prevents the value of the selected cell and its dependents from being changed as a result of changes made elsewhere in the plan. See Lock & Spread for more information.
Remove All Locks Removes all locks on the financial plan, meaning all cell values are subject to change as a result of changes made elsewhere in the plan. See Lock & Spread for more information.
Revert Cell  
Graph Toggles showing/hiding the graph.
Fields Adds/removes/rearranges fields displayed in the grid.
Save Saves changes made to data values in the plan.
Cancel Cancels unsaved data value changes.
