The Financial Planning Screen

The Financial Planning screen contains the following panes:

  • Toolbar
  • Document pane

Toolbar

The Financial Planning toolbar contains the following items:

Item Description
Manage button Click to open the Financial Plan Maintenance window, where you can create new plans, manage existing plans, and select plans to open in the Financial Planning document pane.
[Plan name] Displays the name of the plan currently open in the document pane.

Document pane

This pane contains all of the plans you have opened in your current Demand Management session. Each plan appears as a separate document with the plan name displayed on the tab. Simply click the tab of the plan you want to view to switch to that document.
