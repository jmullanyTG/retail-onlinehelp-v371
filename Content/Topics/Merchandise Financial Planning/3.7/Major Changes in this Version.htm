Major Changes in this Version

Version 3.7 includes the following major changes:

  • Auto Lock on Cell Change option. You can now choose to have sales-related cell values lock automatically after you update them, without having to continually interrupt your flow to click the Lock button. This optFor more information, see Auto Lock on Cell Change.
  • New Save/Restore Data Layout feature. There are now two options for saving and restoring planning view layouts. The existing Save/Restore Layout feature saves and restores the entire layout of the planning grid (row fields, column fields, data fields, and filters). A new feature, Save/Restore Data Layout, lets you save and restore the fields used as metrics in the data area of the planning grid independently of the pivot fields used to aggregate the data into rows and columns. This lets you easily switch the metrics displayed in a layout without having to recreate the product/location/time dimensions and filters in your current layout. For more information, see Customizing Layouts.
