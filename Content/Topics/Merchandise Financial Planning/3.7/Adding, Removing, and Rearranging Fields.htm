Adding, Removing, and Rearranging Fields

To add, remove, or reorder fields in the Hybrid Financial Planning Grid:

  1. Click in the grid toolbar. The Fields Customization dialog box opens. It contains two tabs: Simple and Advanced.

    The Simple tab contains different areas representing where fields appear in the grid.

    The Advanced tab displays all fields in a single list (see step 4 below).

  2. To rearrange fields using the Simple tab, simply drag a field from its current area to the area you want it to appear in on the grid.
    • None Area: these fields are not displayed in the grid.
    • Filter Area: these fields are displayed in the "Drop Filter Fields Here" area above the grid.

    • Row Area: these fields are used to create rows.

    • Column Area: these fields are used to create columns.

    • Data Area: these fields are the metrics whose data values appear in the grid.

  3. To change the order in which a field appears on the grid, drag it to the desired position in its area. The numbers in the column next to the fields update to show the new order. (Instead of dragging fields, you can also manually type over the numbers in the column to enter a new order.)

  4. To rearrange fields using the Advanced tab:

    1. Find the field in the list and click in its Pivot Area cell to select its new area.

    2. To rearrange the order in which fields appear in an area, drag them up or down the list or manually enter the new order in the Index column.

    Note: The Advanced tab is best used for moving multiple fields at once from one area to another.

    1. Select the fields you want to move.

    2. Right-click in one of the selected field's Pivot Area cell and choose Multiple Changes for Pivot Area.

    3. In the Pivot Area dialog box, select the desired new area in the New value option.

    4. Click Apply to save the change. The fields now appear at the top of their new area. You can drag them to their desired position in the list or manually retype their order in the Index column.

  5. When you are done making changes to field placements, click OK to save the changes and apply them to the Hybrid Financial Planning Grid.
