Plan Documents

A financial plan document is a workspace containing the following areas:

  1. Parameters pane. This is where you set the filters that determine which data is displayed in the hybrid financial planning grid.
  2. Hybrid Financial Planning Grid. This is the Excel-style grid displaying your plan data.
  3. Calculation Direction Toggles pane. This pane controls which calculation toggles are in effect in the hybrid financial planning grid.
  4. Locks On Selected Data pane. This pane controls which locks are in effect in the hybrid financial planning grid.

Customizing a document workspace

The "factory default" workspace for a financial plan document contains the hybrid financial plan grid in the left-side pane and the Calculation Direction Toggles and Locks on Selected Data panes stacked on the right, as shown below.

You can customize this workspace by dragging any of the pane headers (except the Parameters pane) to a desired location, including off the screen or to another monitor. Panes are kept in synch no matter where they are positioned, so when you make changes in one pane, the effects are immediately visible in the other panes.

Maximize/Restore and Pin buttons

The Calculation Toggles and Locks on Selected Data panes have the following buttons on their headers:

  • Maximize/Restore: toggles increasing/restoring the size of the pane.
  • Auto-HIde/Dock: toggles auto-hiding and docking the pane. Auto-Hide "hides" the pane until it is clicked on. Dock restores a hidden pane to its fixed position in the workspace.

Saving/Restoring Workspaces

Once you have your workspace configured just right, you can save it (and share it) for later use. Just right-click on the plan document tab (or click the down arrow) to open the Workspaces context menu.

From this menu, you can save and load alternate workspaces. (The current workspace is indicated with a checkmark.)

Menu Option Description
Save New

Saves your current workspace configuration as a new workspace. You have the option of making your saved workspace private (only you can use it) or public (others can use it as well).
Save: [workspace name] Saves changes to the current workspace.
Delete: [workspace name] Deletes the current workspace.
Workspaces > [workspace name] Loads a previously saved workspace.
Workspaces > Restore Default Loads the default workspace.
Workspaces > Save Default Saves your current workspace as the new default workspace for all users.
