The Financial Plan Maintenance Window

The Financial Plan Maintenance window lists all existing financial and location plans you have access to. From this window you can open plan documents as well as perform administrative actions such as creating, branching, merging, and initializing plans.

Getting here

  • Click Manage on the Financial Planning toolbar.

Actions

The following actions are available on the Financial Plan Maintenance toolbar.

Action Description
New Creates a new financial or location plan.
Edit Edits the settings of the plan selected in the plan list.
Initialize Seeds/populates data values in the selected plan according to the measure set selected for the plan. This action is only available to plans with the status, "Created".
Re-Initialize Re-initializes the selected plan.
Close Deletes the selected plan.
Re-Seed Re-seeds future periods within the selected plan.
Branch Creates a branch of the selected master or branch plan.
Merge Merges the selected branch plan into another branch or master plan.
Submit Submits the selected branch plan for approval.
Approve Approves the selected plan.
Reject Rejects the selected plan.
Global Groups Manages the site and product hierarchies used in your financial plans.
User Access Controls user/role access to the selected plan, including product, channel, and measure sets within the plan.
Merge Measure Sets Controls which set of metrics are merged from the selected plan.
Refresh Updates the information displayed in the Financial Plan Maintenance window.
