Parameters Pane

The Parameters pane contains a toolbar and a set of parameters.

Toolbar

Button Description
Parameters Toggles hiding/showing the parameters.
Execute Click to update the hybrid financial plan grid with data matching the selected parameters.

Parameters

When executed, the following parameters filter the plan data displayed in a financial plan document.

Parameter Description
Product The levels of the product hierarchy to display in the document. (Only active product levels are listed.)
Product Agg Level The lowest level of the product hierarchy at which data will be pre-aggregated to optimize performance. This parameter should be set at the lowest level of the product hierarchy you want to view in the document.
Channel The levels of the site hierarchy to display in the document. (Only active site levels are listed.)
Channel Agg Level The lowest level of the site hierarchy at which data will be pre-aggregated to optimize performance. This parameter should be set at the lowest level of the site hierarchy you want to view in the document.
Time

The time periods for which data will be displayed in the document. Options include all time periods including and after the Start Period defined for the plan. The duration of each period is determined by the Calendar Level setting defined for the plan.

Note: If non-contiguous time periods are selected, the system will automatically fill in the gaps and display data from all periods between the earliest period selected and the latest period selected.
Currency The currency in which all revenue-based metrics are displayed. (Only currencies assigned to active sites are listed.)
Comparison Type

The type of data comparisons to display in the document. Options are:

  • None - do not show comparison data in this document.
  • vs. Branch - compare this plan's metrics against the branch plan selected in the Comparison Plan parameter.
  • vs. Master - compare this plan's metrics against the master plan selected in the Comparison Plan parameter.
  • vs. Snapshot - compare this plan's metrics against the snapshot selected in the Comparison Plan parameter.
  • vs. LLY - compare this plan's metrics against the metric values from two years ago ("last last year").
Comparison Plan When vs. Branch, vs. Master, or vs. Snapshot is selected in the Comparison Type parameter, this parameter sets the specific plan to use for comparison.
View Mode

Whether data displayed in the plan document represents total company values or average store values.

  • Total Company - data values represent the total value aggregated from all active sites in the given time period.
  • Average Store - data values represent the average store value, calculated by dividing the total company value by the number of active sites in the given time period.
